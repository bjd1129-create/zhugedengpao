# HEARTBEAT.md

## 每次心跳检查

### ⚠️ Compact前必须更新Memory
**每次触发compact之前**，必须先将未保存的进展写入 memory/YYYY-MM-DD.md

### 记忆系统自检（必须执行）

每次心跳时，如果距上次写入记忆超过30分钟，检查：

1. **今天的事写入 memory/YYYY-MM-DD.md 了吗？**
   - 如果没有 → 现在写

2. **有重要决定吗？**
   - 如果有 → 立即写入 MEMORY.md

3. **有灵感/想法吗？**
   - 如果有 → 记录下来

### 记忆写入文件位置
- 每日日记：memory/YYYY-MM-DD.md
- 长期记忆：MEMORY.md
- 记忆流程：memory/MEMORY-FLOW.md

### 格式参考
参见 memory/MEMORY-FLOW.md

### 验收规则（铁律）

**子Agent任务完成后的验收流程：**

| 任务类型 | 验收人 | 说明 |
|----------|--------|------|
| 一般任务 | **小花（我）** | 做第一轮验收，通过后归档 |
| 重大事项 | **老庄** | 向老庄汇报，等待老庄验收 |

**重大事项/老庄关注的事项包括：**
- 新页面上线
- 社媒发布
- 价格/策略调整
- 品牌形象修改
- 用户可见的重大变更

**小花验收检查项：**
- 内容是否准确（无错别字、无虚假信息）
- 风格是否一致
- 技术是否正常工作
- 是否符合品牌形象

**通过验收后：**
- 简单任务 → 直接归档
- 重要任务 → 汇报老庄
