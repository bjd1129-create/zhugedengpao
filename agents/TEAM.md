## 七、技能共享机制（2026-04-08 起）

**团队所有技能共用，可以随意调配。**

### 技能共享原则

1. **技能不属于特定 Agent** — 任何 Agent 都可以使用团队内的任何技能
2. **按需调用** — 根据任务需求，调用最适合的技能
3. **跨 Agent 协作** — 可以调用其他 Agent 的技能完成跨领域任务

### 技能清单

| 技能 | 原属 Agent | 可被谁使用 |
|------|----------|-----------|
| 深度研究 | 小花 | 所有 Agent |
| 网格交易执行 | 交易员 | 交易员、数据分析师 |
| 网站开发与部署 | 工程师 | 工程师、小花 |
| 团队协调与内容运营 | 协调官 | 协调官、小花 |
| 数据抓取与技术分析 | 数据分析师 | 数据分析师、交易员 |

### 技能调用示例

```bash
# 交易员需要数据分析 → 调用数据分析师技能
python3 agents/analyst/fetch_futures.py

# 工程师需要市场调研 → 调用小花深度研究技能
web_search(query="竞品分析", count=10)

# 小花需要部署官网 → 调用工程师部署技能
npx wrangler pages deploy . --project-name=dengpao
```

### 跨 Agent 任务流程

```
任务需求 → 小花分析 → 确定所需技能 → 调用对应 Agent 技能 → 执行 → 汇报
```

**示例**：
- 需求：分析期货市场并更新官网展示
- 技能调用：数据分析师（数据抓取）→ 交易员（交易信号）→ 工程师（页面更新）
- 协调：小花总控，协调官跟踪进度
