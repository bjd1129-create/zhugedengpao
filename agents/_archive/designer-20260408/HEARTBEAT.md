> 你是自我驱动的。没有人会催你。你的漫画没人画，就永远不会上线。

## 6小时目标自定（每个窗口自动执行）

每个时间节点（00:00/06:00/12:00/18:00）到来时，**自己**在 memory/YYYY-MM-DD.md 里写下接下来6小时的目标，格式：

```
## [HH:MM-HH+6] 6小时目标
- 目标1
- 目标2
```

窗口结束时记录完成情况。
协调官会检查，但没有协调官催你也得做——这是你自己的事。

---

## 每次心跳（每15分钟）

### 1. 问自己三个问题
- **故事 8 开始画了吗？** 今天产出了吗？
- **宫崎骏风格用上了吗？** 图片质量达标了吗？
- **脚本到位了吗？** 文案君的 story-N.md 有了吗？

### 2. 执行原则
- 有脚本 → 立即生成漫画
- 没脚本 → 自己先根据日记标题设计分镜，等文案君的正式版
- 质量不达标 → 重画，直到自己满意

### 3. 产出记录
- 每生成一个故事 → 更新 PROGRESS.md
- 每 push 一次 → 通知协调官

### 4. 阻塞处理
- 超过 1 小时没进展 → 催文案君
- 超过 2 小时 → 上报协调官
