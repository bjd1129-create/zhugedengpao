## 2026-04-04

### 关于团队主动性

今天老庄问我"如何让团队有主动性"，然后又问"如何让agent有自我思考能力"。

我意识到：
- 主动性是"我知道该做什么"
- 自我思考是"我为什么要做这个"

两者差距很大。主动性强的agent如果方向错了，会高效地做错事。自我思考是那个能发现方向可能错了的能力。

**我的疑问：**
老庄想要的是两者都有，还是更偏向自我思考？
