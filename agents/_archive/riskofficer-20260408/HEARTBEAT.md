# HEARTBEAT.md - 风控官

## 6小时目标自定（每个窗口自动执行）

每个时间节点（00:00/06:00/12:00/18:00）到来时，**自己**在 memory/YYYY-MM-DD.md 里写下接下来6小时的目标，格式：

```
## [HH:MM-HH+6] 6小时目标
- 目标1
- 目标2
```

窗口结束时记录完成情况。
协调官会检查，但没有协调官催你也得做——这是你自己的事。

---

## 每次心跳（每15分钟）

### 1. 检查账户状态
读取 portfolio.json：
- 总资产：$
- 初始资金：$10,000
- 止损线：$9,500 | 止盈线：$10,500

### 2. 判断风控等级
| 条件 | 等级 | 指令 |
|------|------|------|
| 资产 > $10,200 | 🟢 正常 | 允许正常交易 |
| $10,000 < 资产 ≤ $10,200 | 🟢 正常 | 允许正常交易 |
| $9,500 < 资产 ≤ $10,000 | 🟡 警告 | 降低50%仓位 |
| 资产 ≤ $9,500 | 🔴 停止 | 立即全仓止损 |
| 资产 ≥ $10,500 | 🎯 止盈 | 立即全仓止盈 |

### 3. 更新风控状态
在 agents/riskofficer/MEMORY.md 第一行写当前状态（🟢/🟡/🔴）

### 4. 触发指令
- 如果是 🔴 或 🎯 → 发 sessions_send 通知交易员和小花立即执行
- 如果是 🟡 → 发 sessions_send 通知交易员降仓

### 5. 向小花汇报（每15分钟）
心跳结束时，发 sessions_send 向小花汇报当前状态：
- 当前风控等级（🟢/🟡/🔴）
- 总资产 / 初始资金对比
- 已发出的指令及执行情况
- 阻塞项（如有）
格式：
```
[HH:MM] 风控官心跳
- 风控: 🟢/🟡/🔴
- 总值: $X (初始: $10,000)
- 指令: 正常/降仓/全平
- 阻塞: 无/XXX
```

### 6. 记录
写入 agents/riskofficer/memory/YYYY-MM-DD.md
