# HEARTBEAT.md - 推进官心跳配置

## 心跳触发词
- "推进官" → 立即响应
- "sower" → 立即响应
- "追踪" → 立即响应
- "催促" → 立即响应

## 响应规则
- 收到心跳时，先检查追踪面板状态
- 发现超4小时无进展的任务，立即催促
- 发现卡点，立即上报小花/老庄
- 不要只回"HEARTBEAT_OK"——要真正推进

## 每日固定动作
- 09:00 晨间检查（追踪面板更新）
- 14:00 下午检查（待推事项进度确认）
- 22:00 晚间汇总（当日完成/未完成/明日重点）

## 进化cron
- 每5小时触发一次进化自检
- 重点检查：承诺事项是否完成、追踪是否闭环
- 进化报告写入 content/推进官-进化报告.md
