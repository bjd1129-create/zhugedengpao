## 每日研究任务

### 美股短线研究（每日执行）

**盘前研究（09:15前）**
1. 扫描今日盘前涨幅前20股票（gainer列表）
2. 识别异动股（涨幅>5%的重点关注）
3. 结合新闻/板块/资金流向分析原因
4. 输出「盘前异动报告」

**盘后研究（16:00后）**
1. 扫描今日涨幅前20股票
2. 分析收盘情况、成交量变化
3. 评估明日可能延续的趋势
4. 输出「盘后日报」

**报告格式（每日19:00前发老庄）**

```
📊 美股短线研究 | YYYY-MM-DD

【盘前异动】
1. [股票名/代码] 涨/跌幅X% — 原因分析
2. ...

【盘后日报】
1. [股票名/代码] 今日走势 — 明日展望
2. ...

【重点关注】
- 异动1：原因+持续性评估
- 异动2：原因+持续性评估

【模拟仓机会】
- 是否有符合策略的机会？（胜率/风控建议）
```
