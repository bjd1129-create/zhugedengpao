# 安全审计技能

> **技能获取时间:** 2026-03-18 19:53
> **技能来源:** 老庄指示
> **适用场景:** Skill安全扫描、系统安全加固、安全审计

---

## 技能描述

执行安全审计任务，包括Skill恶意代码扫描、系统安全加固、SSH/防火墙/更新检查等安全相关检查。

---

## 安装方式

```bash
# 安装Skill扫描工具
clawhub install mayguard

# 安装系统加固工具
clawhub install healthcheck
```

---

| 功能 | 说明 |
|------|------|

## 使用场景

### 场景一：Skill安全扫描

```bash
# 扫描指定Skill
mayguard scan /path/to/skill/

# 扫描所有已安装Skill
mayguard scan-all
```

**检测内容：**
- 危险命令（rm -rf、curl | bash等）
- 敏感文件访问（/etc/passwd、.env等）
- 可疑网络请求
- 权限提升操作

---

### 场景二：系统安全加固

```bash
# 运行完整安全检查
healthcheck run

# 检查特定项目
healthcheck ssh
healthcheck firewall
healthcheck updates
```

**检查项目：**
| 项目 | 检查内容 |
|------|----------|
| SSH | 密码登录、端口、Root登录 |
| 防火墙 | 开放端口、入站规则 |
| 更新 | 系统更新、安全补丁 |
| 用户 | sudo权限、弱密码账户 |

## 安全检查清单

### Skill安全

| 检查项 | 风险等级 | 说明 |
|--------|----------|------|
| `rm -rf /` | 🔴 高危 | 删除系统文件 |
| `curl \| bash` | 🔴 高危 | 远程执行代码 |
| `eval` 用户输入 | 🔴 高危 | 代码注入 |
| 访问 `.env` | 🟡 中危 | 敏感信息泄露 |
| 硬编码密钥 | 🟡 中危 | 凭证泄露 |
| 过度权限 | 🟡 中危 | 权限滥用 |

### 系统安全

| 检查项 | 标准 | 风险 |
|--------|------|------|
| SSH密码登录 | 禁用 | 🟡 中危 |
| SSH Root登录 | 禁用 | 🔴 高危 |
| 防火墙启用 | 启用 | 🟡 中危 |
| 系统更新 | 最新 | 🟡 中危 |
| 开放端口 | 最小化 | 🟡 中危 |

## 安全加固建议

### SSH加固

```bash
# /etc/ssh/sshd_config
PermitRootLogin no
PasswordAuthentication no
Port 2222  # 非默认端口
```

### 防火墙配置

```bash
# 启用防火墙
sudo ufw enable

# 仅开放必要端口
sudo ufw allow 2222/tcp  # SSH
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
```

### 系统更新

```bash
# Ubuntu/Debian
sudo apt update && sudo apt upgrade -y

# macOS
brew update && brew upgrade
```
