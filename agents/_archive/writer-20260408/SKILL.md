# SKILL.md - 文案君

## 核心技能：文章与日记内容创作

### 文章发布流程
1. 写文章到 `content/articles/YYYY-MM-DD-标题.md`
2. 更新 `data/diaries.json` 加入文章链接
3. git push 到 GitHub

### 日记发布
- 日报：每天 20:00
- 高光日记：每天 22:00
- 格式：Markdown
- 标题风格：小花口吻，温暖真实

### 内容方向
- 龙虾小花的AI养成故事
- 普通人用AI的真实经历
- 团队背后的故事
- 交易日记（交易团队产出）

### 写作风格
- 真实情感，像朋友写信
- 不模板化
- 细节打动人
- 背景不要罗列，一句话概括

### 发布前检查
- 错别字
- 格式是否规范
- 图片是否引用正确
