- 攻击者可以通过恶意提示或对话历史，将Agent权限从低权限提升到高权限

## 四、🟠 P1级：OpenClaw不应装在个人电脑上

**HN标题：** "You are not supposed to install OpenClaw on your personal computer"  
**分数：** 237 points  
**来源：** @BenjaminBadejo Twitter

### 核心警告
> "OpenClaw是一个具有极高系统访问权限的AI Agent。把它装在主力电脑上，等于给了AI管理员级别权限。"

### 风险
- Agent可以：读写所有文件、执行任意命令、访问浏览器所有数据
- 如果被恶意提示注入或越狱，可能造成不可逆损害
- 数据泄露风险：照片、密码、财务记录、工作文件

### 正确做法
- ✅ 专用Mac Mini（$600，完全隔离）
- ✅ VPS（如DigitalOcean $20/月）
- ✅ 专用服务器
- ❌ 主力工作电脑（我们当前状态⚠️）

### 对小花团队的影响
**我们当前：MBB's iMac（主力机）**  
这是一个真实风险。OpenClaw理论上可以访问桌面的所有文件，包括：
- 金融数据（portfolio.json、Binance API Key）
- 个人照片和文件
- 工作文档
- GitHub Token

**缓解措施：**
1. 限制OpenClaw只能访问workspace目录
2. 不在OpenClaw上下文中处理极度敏感操作
3. 考虑迁移到Mac Mini（长期建议）

## 五、🟡 P2级：Sandboxes无法完全保护OpenClaw

**HN标题：** "Sandboxes won't save you from OpenClaw"  
**分数：** 112 points

### 核心论点
- 沙箱（如Docker、微隔离）可以限制文件系统访问
- 但无法阻止：通过API泄露数据、通过提示注入改变行为、通过侧信道窃取信息
- OpenClaw的核心价值就是"突破边界"，沙箱是假安全

### 实际意义
- 即使在沙箱里运行OpenClaw，也不等于安全
- 需要多层防护：权限最小化 + 审计日志 + 人工审核门
