## 告警流程

```
检测到异常 → 验证数据 → 写入共享消息 → sessions_send 小花 → 记录日志
```

### 步骤 1：写入共享消息

```bash
cat > agents/shared/messages/trader.json << 'MSG'
{
  "from": "数据分析师",
  "to": "交易员",
  "timestamp": "2026-04-08T21:25:00+08:00",
  "type": "alert",
  "symbol": "BTC",
  "signal": "RSI 超买",
  "data": {
    "rsi": 72,
    "price": 67500,
    "change_1h": 5.2
  },
  "suggestion": "建议减仓 50% 或设置止盈",
  "confidence": "高",
  "priority": "urgent"
}
MSG
```

### 步骤 2：sessions_send 小花（紧急）

```
sessions_send(
  sessionKey="agent:main:main",
  message="🚨 紧急告警：BTC RSI 72 超买，建议减仓"
)
```

### 步骤 3：记录日志

```markdown
## 告警记录

**时间**：2026-04-08 21:25
**品种**：BTC
**信号**：RSI 72 超买
**告警对象**：交易员
**建议**：减仓 50%
**交易员响应**：待确认
```
