# 信息中转规则

**创建时间**：2026-04-08 22:00  
**优先级**：P0 - 铁律

---

## 📋 核心规则

### 所有给交易员的信息都需要小花中转

**流程**：
```
数据分析师 → 写给小花 → 小花审核 → 小花转达交易员
              ↓
        to-xiaohua-for-trader.md
```

---

## ⚠️ 为什么需要中转？

| 原因 | 说明 |
|------|------|
| **团队层级** | 小花是管理者，负责协调和决策 |
| **避免越权** | 数据分析师不应直接指挥交易员 |
| **信息审核** | 小花审核信息准确性和必要性 |
| **统一出口** | 保持团队沟通一致性 |

---

## 📝 正确做法

### ✅ 正确：通过小花中转

```markdown
1. 数据分析师创建文件：to-xiaohua-for-trader.md
2. 内容包含：需要转达的完整信息
3. @小花 请审阅并转达交易员
4. 小花审核后转达给交易员
```

### ❌ 错误：直接联系交易员

```markdown
❌ 直接发送消息给交易员
❌ 直接修改交易员文件
❌ 直接指挥交易员操作
❌ 跳过小花审核
```
