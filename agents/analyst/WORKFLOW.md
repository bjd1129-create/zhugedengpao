# 数据分析师 ↔ 交易员 协作流程

**生效日期**：2026-04-08  
**制定者**：小花

---

## 🎯 协作目标

数据分析师提供**数据支撑**，交易员做出**交易决策**。

**原则**：
- 数据分析师不执行交易
- 交易员不修改数据
- 信息透明，决策独立

---

```
数据分析师                    交易员
```

## 🔄 日常协作流程

### 1. 数据推送（数据分析师 → 交易员）

#### 定期推送
| 数据类型 | 频率 | 时间 | 格式 |
|---------|------|------|------|
| 期货价格 | 每 5 分钟 | 交易时段 | `china_futures.json` |
| 升贴水率 | 每 5 分钟 | 交易时段 | `basis_analysis.json` |
| 交易信号 | 触发时 | 实时 | `signals.json` |
| 市场日报 | 每日 | 09:00 | Markdown 报告 |

#### 信号触发条件
| 信号类型 | 触发条件 | 推送方式 |
|---------|---------|---------|
| 高贴水买入 | IM/IC 年化贴水>5% | sessions_send |
| 贴水收敛 | 近月贴水率<1% | sessions_send |
| 异常波动 | 价格涨跌幅>3% | sessions_send + 飞书告警 |

### 2. 交易同步（交易员 → 数据分析师）

#### 交易后同步
```json
{
  "timestamp": "2026-04-08T10:30:00+08:00",
  "account": "股指期货模拟盘",
  "action": "BUY",
  "contract": "IM2604",
  "quantity": 5,
  "price": 7947.4,
  "reason": "滚贴水策略 v1.0 - 年化贴水 8.5%",
  "signal_source": "data_analyst/basis_analysis.json"
}
```

#### 每日同步
- **时间**：22:00
- **内容**：当日交易汇总、净值变化、持仓状态
- **格式**：Markdown 报告 → `memory/YYYY-MM-DD.md`

## 🚨 告警机制

### 告警级别
| 级别 | 条件 | 通知方式 | 响应时间 |
|------|------|---------|---------|
| 🔴 P0 | 账户亏损>15% | 飞书 + sessions_send | 立即 |
| 🟠 P1 | 单笔亏损>5% | 飞书 | 5 分钟内 |
| 🟡 P2 | 信号触发 | sessions_send | 15 分钟内 |
| 🔵 P3 | 数据异常 | 日志记录 | 下次 heartbeat |

### 告警模板
```markdown
🚨 【P0 告警】股指期货模拟盘

账户：股指期货模拟盘

当前价格：7850.0（开仓价 7947.4）
建议：平仓 50% 或设置止损

@交易员 请确认处理方式
```

## 📞 沟通渠道

### 正式沟通
- **sessions_send**：任务分配、数据同步
- **飞书消息**：告警、紧急事项
- **Markdown 报告**：日报、周报、专项研究

### 非正式沟通
- **直接对话**：老庄在群里提问时
- **heartbeat**：日常状态同步

## 📋 任务分配流程

### 数据分析师 → 交易员
```
1. 数据分析师发现高贴水信号（IM 年化>8%）
2. 写入 signals.json
3. sessions_send 通知交易员
4. 交易员评估后决策（买/不买/缓一缓）
5. 交易后同步结果给数据分析师
6. 数据分析师记录信号准确率
```

### 交易员 → 数据分析师
```
1. 交易员需要特定数据（如"查一下 IC 历史贴水"）
2. sessions_send 发送研究请求
3. 数据分析师调研并输出报告
4. 交易员根据报告调整策略
```

## 🔧 工具配置

### sessions_spawn 示例
```python
# 数据分析师给交易员派工
sessions_send(
    sessionKey="agent:trader:main",
    message="""
【交易信号】IM2604 高贴水机会

- 当前价格：7947.4
- 年化贴水：8.5%
- 建议操作：做多 5 手
- 止损位：7750（-2.5%）

详情：agents/analyst/data/signals.json
"""
)
```

### cron 配置
```json
{
  "name": "期货数据推送",
  "schedule": {"kind": "every", "everyMs": 300000},
  "payload": {
    "kind": "agentTurn",
    "message": "抓取期货数据并推送给交易员"
  },
  "sessionTarget": "agent:analyst:main"
}
```
