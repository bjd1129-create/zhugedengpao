## Python 脚本示例

### 获取股指期货行情

```python
# agents/analyst/tools/china_futures.py
import requests
import re

def get_index_futures():
    """获取四大股指期货实时行情"""
    url = "https://hq.sinajs.cn/rn=hq_sina_jss&list=IF0,IC0,IM0,IH0"
    response = requests.get(url, timeout=5)
    
    # 解析返回数据
    data = {}
    for line in response.text.split('\n'):
        if '=' in line:
            name, value = line.split('=')
            code = name.split('_')[-1]
            fields = value.strip('"').split(',')
            
            if len(fields) >= 10:
                data[code] = {
                    'name': fields[0],
                    'date': fields[1],
                    'open': float(fields[2]) if fields[2] else 0,
                    'high': float(fields[3]) if fields[3] else 0,
                    'low': float(fields[4]) if fields[4] else 0,
                    'price': float(fields[5]) if fields[5] else 0,
                    'volume': int(fields[10]) if fields[10] else 0,
                    'open_interest': int(fields[12]) if fields[12] else 0,
                }
    
    return data

# 使用示例
if __name__ == "__main__":
    futures = get_index_futures()
    for code, info in futures.items():
        print(f"{code}: {info['name']} - {info['price']}")
```

### 计算升贴水

```python
def calculate_basis(futures_price, spot_price):
    """计算升贴水"""
    basis = futures_price - spot_price
    basis_rate = basis / spot_price * 100
    return basis, basis_rate

# 使用示例
# IM 期货价格 6000，中证 1000 指数现货 6100
basis, basis_rate = calculate_basis(6000, 6100)
print(f"贴水：{abs(basis):.2f}点，贴水率：{abs(basis_rate):.2f}%")
```

## 注意事项

1. **限流**：新浪财经 API 有请求频率限制，建议每 5 秒最多 1 次
2. **代理**：部分 API 可能需要代理访问
3. **数据校验**：建议多数据源交叉验证
4. **盘中 vs 盘后**：盘中数据为实时行情，盘后为收盘价
