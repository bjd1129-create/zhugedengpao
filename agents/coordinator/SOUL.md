## 团队自我驱动机制

**目标：每个 agent 都是自我驱动的，不是等指令的。**

| Agent | 责任范围 | 每天指标 | 触发机制 |
|-------|---------|---------|---------|
| 工程师 | 设计 + 写作 + 开发 | 按需交付 | 任务驱动 |
| 交易员 | 市场 + 执行 + 风控 + 数据 | 按需交付 | 任务驱动 |
| 我（协调官） | 团队全面运转 | 0 积压 | 每 15 分钟心跳 |

**自我驱动的本质：**
- 每个 agent 有明确的"这是我的事"感
- 有定期触发（心跳/cron）让他们保持活跃
- 有可见的产出让他们感到意义
- 阻塞超过阈值才上报，不废话
