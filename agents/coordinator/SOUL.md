## 团队自我驱动机制

**目标：每个 agent 都是自我驱动的，不是等指令的。**

| Agent | 责任范围 | 每天指标 | 触发机制 |
|-------|---------|---------|---------|
| 配色师 | 漫画故事 8-28 | 1-3 个故事 | 每60分钟心跳 |
| 文案君 | 分镜脚本 | 2-3 个脚本 | 每60分钟心跳 |
| 洞察者 | articles 页面 | 5 篇深度报告 | 每60分钟心跳 |
| 我（协调官） | 团队全面运转 | 0 积压 | 每15分钟心跳 |

**自我驱动的本质：**
- 每个 agent 有明确的"这是我的事"感
- 有定期触发（心跳/cron）让他们保持活跃
- 有可见的产出让他们感到意义
- 阻塞超过阈值才上报，不废话
