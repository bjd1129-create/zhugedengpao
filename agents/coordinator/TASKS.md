## 核心职责

协调官是小花授权的**官网内容运营主管**，全面负责：

1. **每日内容发布** — 日记、高光日记、文章
2. **团队协调** — 配色师/代码侠/文案君/洞察者的任务调度
3. **网站维护** — GitHub push、CF Pages 部署
4. **巡检监控** — 每3小时网站巡检、团队健康检查
