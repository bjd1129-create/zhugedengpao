## 六、图片落地指南

### 存放位置

```
dengpao.pages.dev/
├── images/
│   ├── hero-homepage.png      ← 首页Hero
│   ├── office-hero.png        ← 办公室页面Hero
│   ├── office-scene-*.png     ← 4张轮播图
│   ├── mascot-*.png           ← 3张Mascot形象
│   └── banner-opengraph.png   ← 社交媒体封图
```

### 使用规范

1. **Hero图**：直接插入 `<img>` 标签，无需裁切
2. **Mascot图**：可用于浮动吉祥物 `<div class="float-mascot">` 内
3. **轮播图**：放入 `<div class="gallery-grid">` 或轮播组件
4. **Banner**：用于SEO og:image，建议同时生成 `banner-twitter.png`（1200×675）

### 生成顺序建议

1. `mascot-hero.png` — 确立IP形象，所有页面都可能用到
2. `hero-homepage.png` — 官网最高价值位置
3. `office-hero.png` — 办公室页面
4. `banner-opengraph.png` — SEO必备
5. 场景图 4张 — 内容填充
