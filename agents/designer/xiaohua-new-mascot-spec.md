## 一、形象设计

### 1.1 核心形象——"脑洞精灵"

**外形特征：**
| 特征 | 设计 |
|------|------|
| 物种 | 米色小熊/仓鼠（原创，非猫） |
| 主色 | 米白色 #F5E6D3 |
| 眼睛 | 圆润黑豆眼，明亮有神 |
| 服装 | 珊瑚红 #E07A5A 毛衣 |
| 标志物 | 头顶小灯泡（代表灵感/脑洞） |
| 道具 | 手持小笔记本 |
| 气质 | 温暖、有爱、好奇、创意感 |

### 1.2 与加菲猫的差异

| 维度 | 加菲猫 | 小花新形象 |
|------|--------|-----------|
| 物种 | 橘猫 | 米色小熊/仓鼠 |
| 颜色 | 橘色+条纹 | 米白纯色 |
| 眼睛 | 慵懒下垂 | 圆润明亮 |
| 气质 | 懒散躺平 | 好奇打工 |
| 风格 | 美式幽默 | 日系治愈 |

**相似度：<50%** ✅

## 五、漫画风格规范

**风格：** 日系治愈Kawaii  
**线条：** 柔和棕褐色细线  
**配色：** 低饱和暖色调  
**文字：** 手写体风格  
**背景：** 纯白或淡暖色  

**四格漫画标准尺寸：** 800x600px（横版）或 600x800px（竖版）  
**单格文件大小：** <200KB（WebP格式）
