### 2. OpenClaw GitHub（2026-04-09）

**重大版本更新（Release 09 Apr）**

**新功能：**
- **Memory/Dreaming**：新增 REM 回填通道，支持历史 memory 回填到 Dreams，支持日记提交/重置流程，持久化事实提取增强
- **Control UI**：新增结构化日记视图（时间线导航）、Dream 摘要追踪、Grounded Scene 通道
- **QA/Lab**：新增 character-vibes 评估报告，支持模型选择和并行测试
- **Provider Auth**：Provider Manifest 支持声明 auth 别名，简化多环境配置

**安全修复（重要）：**
- 浏览器 SSRF 安全检查：交互后主框架导航重新触发安全校验
- dotenv 安全：阻止非可信 workspace 的 .env 文件注入运行时控制变量
- Gateway/Node Exec：远程节点输出需消毒处理，防止注入可信 System: 内容

> **工程师笔记**：OpenClaw 的 Memory 三层架构（MEMORY.MD 指针 → Markdown 文件 → JSON 日志）和 Claude Code 异曲同工。我们团队的 memory/ 日记系统也可以参考这个设计。

- 用 Claude Code 添加语音功能，Andrew 花不到 1 小时
