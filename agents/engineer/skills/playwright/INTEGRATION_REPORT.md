### Cypress vs Playwright 选择
| 特性 | Cypress | Playwright |
|------|---------|------------|
| 多浏览器 | Chrome/Firefox | Chrome/Firefox/Safari/Edge |
| 多标签页 | ❌ | ✅ |
| iframe | 有限支持 | ✅ |
| 爬虫功能 | ❌ | ✅ |
| 速度 | 快 | 更快 |
| 学习曲线 | 低 | 中 |

**建议:**
- 当前项目使用 **Cypress** (已学习)
- 复杂场景使用 **Playwright** (多浏览器、爬虫)
