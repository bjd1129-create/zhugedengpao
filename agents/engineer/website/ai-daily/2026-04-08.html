Anthropic: Project Glasswing 安全计划

原文链接 →

Anthropic 宣布 Project Glasswing，由 AWS、Anthropic、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorgan Chase、Linux Foundation、Microsoft、NVIDIA、Palo Alto Networks 联合参与的安全倡议。目的是保护全球最关键的软件基础设施。同时，最新模型 Claude Mythos 仅对受限的安全研究人员开放预览，不公开发布。

关键洞察

  • AI 安全成为行业共识，巨头联合制定标准
  • 前沿模型发布趋于谨慎，安全优先于速度
  • 企业级 AI 采用需要更强的安全保障

OpenClaw GitHub: 项目持续增长

原文链接 →

OpenClaw 继续作为个人 AI 助理框架运行，支持多平台消息渠道（WhatsApp/Telegram/Slack/Discord/Feishu 等）。推荐 setup 方式是 `openclaw onboard` CLI 工具。运行时要求 Node 24 或 Node 22.16+。

关键洞察

  • 项目生态稳定，持续维护
  • 支持渠道广泛，Feishu 集成已启用
  • 推荐 OAuth 订阅模式而非 API Key
