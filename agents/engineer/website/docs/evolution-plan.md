| 资源 | 配额 | 用途 |
|------|------|------|

## 进化方向

### 1. 官网进化（对照三万）
- 每5小时全面检查官网
- 对照三万网站逐项优化
- 更新内容/设计/SEO

### 2. 内容进化
- 每日生成文章/文案
- 爆款分析+模仿创作
- 配图生成（120张图优先用于此处）

### 3. 团队能力进化
- 学习新技术
- 优化工作流
- 提升内容质量

### 4. 自我反思进化
- 每5小时复盘
- 总结错误
- 更新MEMORY

## 执行框架

### Cron触发（每5小时）
```
openclaw cron create "大花团队进化" --interval=5h --message "执行团队进化任务"
```

### 任务队列（TASKS.md）
每次cron触发，从队列取最高优先级任务执行。

### Token监控
每次任务记录消耗，确保用完配额。
