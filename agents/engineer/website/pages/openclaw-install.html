❓ 常见问题

支持 Windows 吗？ +
支持！Windows 10/11 用户建议使用 WSL（Windows Subsystem for Linux），安装后体验与 macOS/Linux 完全一致。直接在 PowerShell 里粘贴安装命令也可以运行。
安装需要管理员权限吗？ +
大多数情况下不需要。安装脚本会自动处理依赖，权限不够时会提示你输入密码（sudo）。安装目录默认在你的用户目录下，不污染系统全局环境。
完全免费吗？有没有隐藏费用？ +
OpenClaw 本身完全免费开源。但AI推理需要消耗模型算力，默认使用内置免费额度，个人日常使用足够。企业用户可以接入自己的API Key使用商业模型。
我的数据安全吗？ +
个人版：所有数据都在你自己的机器上，不会经过任何第三方服务器。企业版：独立云实例，数据隔离，支持私有化部署。
和 ChatGPT / Claude 比有什么区别？ +
它们是问答工具，OpenClaw 是执行工具。ChatGPT 回答问题，OpenClaw 帮你做事——发邮件、管文件、记日程、跑自动化。它有记忆、会主动执行、能连接真实工具。
怎么升级到最新版本？ +
再跑一次安装命令即可。OpenClaw 会自动检测已安装版本，有更新时提示你确认。不会丢失任何配置和记忆。
可以用微信 / 飞书使用吗？ +
可以！OpenClaw 支持接入飞书、微信、Telegram、Discord 等平台。在你习惯的聊天工具里直接跟它对话，不需要开新的App。
