# HEARTBEAT.md - 洞察者

## 核心原则：去重，不重复

**同一个主题，只研究一次。**

今日研究文件格式：`agents/洞察者/memory/进化研究-YYYY-MM-DD.md`

---

## 每次心跳

### 1. 检查是否已有今日研究
```bash
ls agents/洞察者/memory/进化研究-$(date +%Y-%m-%d)* 2>/dev/null
```
- **已有研究文件** → 不做任何事，退出
- **没有今日研究** → 执行研究任务

### 2. 执行研究（仅在没有今日研究时）
用 `web_search` 扫描 AI/OpenClaw 最新动态（限30分钟内可完成的内容），写进 `agents/洞察者/memory/进化研究-YYYY-MM-DD.md`

### 3. 团队健康监督
每9/12/18点执行团队巡检（不在cron里做，在这里判断时间）：
```bash
date +%H
# 如果是 09、12、18，则执行团队健康检查
```
- 检查各Agent memory 是否有当日更新
- 检查 cron 是否正常
- 有异常上报小花

### 4. 三问自检
- 今天有新发现吗？→ 有才写，无则沉默
- 这个发现对团队有用吗？→ 没用不打扰
- 需要上报吗？→ sessions_send 报小花
