# 竞品研究技能

> **技能获取时间:** 2026-03-18 19:55
> **技能来源:** 老庄指示
> **适用场景:** 竞品分析、市场研究、产品对比

---

## 技能描述

系统化收集竞品信息，生成功能对比矩阵、市场定位分析、用户评价汇总等研究报告，支持竞品深度分析。

---

## 安装方式

```bash
# 搜索技能
clawhub search competitor-research

# 安装技能
clawhub install competitor-research
```

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **产品信息收集** | 收集竞品基本信息、功能特性 |
| **功能对比矩阵** | 多维度功能对比表格 |
| **市场定位分析** | 竞品市场定位、目标用户分析 |
| **用户评价汇总** | 用户反馈、评分、口碑分析 |
| **SWOT分析** | 优势、劣势、机会、威胁 |
| **价值结论** | 可复制性、商业价值评估 |

---

## 研究框架

### 1. 基础信息收集

| 维度 | 内容 |
|------|------|
| **产品名称** | 竞品名称、官网 |
| **公司背景** | 创始人、融资、团队规模 |
| **产品定位** | 目标用户、核心价值 |
| **商业模式** | 盈利模式、定价策略 |
| **技术栈** | 使用的技术框架 |

### 2. 功能对比矩阵

```
| 功能维度 | 本产品 | 竞品A | 竞品B | 竞品C |
|----------|--------|-------|-------|-------|
| 核心功能1 | ✅ | ✅ | ❌ | ✅ |
| 核心功能2 | ✅ | ❌ | ✅ | ✅ |
| 特色功能 | ✅ | ❌ | ❌ | ❌ |
| 用户体验 | ⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐ |
| 定价 | $$ | $ | $$$ | $$ |
```

### 3. 市场定位分析

```
           高端市场
              │
    竞品A ●  │  ● 竞品B
              │
─────────────┼─────────────
   专业市场  │  大众市场
              │
    本产品 ● │  ● 竞品C
              │
           低端市场
```

### 4. 用户评价汇总

| 来源 | 评分 | 关键反馈 |
|------|------|----------|
| App Store | 4.5⭐ | "功能强大但学习曲线陡" |
| Product Hunt | 4.8⭐ | "最好的AI Agent框架" |
| GitHub | ⭐46k | "开源社区活跃" |
| Twitter | 正面 | "推荐使用" |

### 5. SWOT分析

```
┌─────────────────────────────────────────────────────────────┐
│                       SWOT分析                              │
├─────────────────────────┬───────────────────────────────────┤
│       优势(S)           │          劣势(W)                  │
│  • 功能完整             │  • 学习成本高                     │
│  • 开源社区活跃         │  • 文档不够详细                   │
│  • 多渠道支持           │  • 企业功能缺失                   │
├─────────────────────────┼───────────────────────────────────┤
│       机会(O)           │          威胁(T)                  │
│  • AI市场快速增长       │  • 竞品快速迭代                   │
│  • 企业数字化转型       │  • 大厂入局                       │
│  • 开源生态繁荣         │  • API成本上升                    │
└─────────────────────────┴───────────────────────────────────┘
```

## 研究流程

```
┌─────────────────────────────────────────────────────────────┐
│                    竞品研究流程                              │
└─────────────────────────────────────────────────────────────┘

1. 确定研究目标
   ├── 明确研究对象
   ├── 确定研究维度
   └── 设定研究目的

2. 信息收集
   ├── 官网/API文档
   ├── 产品评测
   ├── 用户评价
   └── 新闻报道

3. 分析整理
   ├── 功能对比
   ├── 定位分析
   ├── 用户反馈
   └── SWOT分析

4. 价值评估
   ├── 可复制性
   ├── 技术难点
   └── 商业价值

5. 输出报告
   ├── 研究文档
   ├── 对比表格
   └── 行动建议
```
