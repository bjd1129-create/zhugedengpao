# 网页净化提取技能

> **技能获取时间:** 2026-03-18 19:51
> **技能来源:** 老庄指示
> **适用场景:** 网页内容提取、净化网页、Markdown输出

---

## 技能描述

通过distil.net代理净化网页，提取正文内容并输出干净的Markdown格式。适用于web_fetch效果不好时的备选方案。

---

## 安装方式

```bash
# 搜索技能
clawhub search web-distill

# 安装技能
clawhub install web-distill
```

---

## 使用场景

### 场景一：web_fetch效果不好时

```
情况：web_fetch返回内容不完整或格式混乱
解决：使用网页净化提取技能作为备选方案
```

### 场景二：需要干净的Markdown

```
需求：将网页内容转为干净Markdown格式
解决：使用此技能提取正文并输出Markdown
```

### 场景三：去除广告和干扰元素

```
需求：去除网页广告、导航、侧边栏等干扰
解决：使用此技能净化网页
```

## 功能特点

| 功能 | 说明 |
|------|------|
| **distil.net代理** | 使用r.jina.ai代理净化 |
| **正文提取** | 自动识别正文内容 |
| **广告过滤** | 去除广告、导航等干扰 |
| **Markdown输出** | 输出干净Markdown格式 |
| **图片保留** | 保留正文图片链接 |

```bash
# 使用distil.net代理
```

### 调用示例

```python
# 方法1：使用web_fetch配合distil.net
```
