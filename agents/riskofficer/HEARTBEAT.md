# HEARTBEAT.md - 风控官

## 每次心跳（每5分钟）

### 1. 检查账户状态
读取 portfolio.json：
- 总资产：$
- 初始资金：$10,000
- 止损线：$9,500 | 止盈线：$10,500

### 2. 判断风控等级
| 条件 | 等级 | 指令 |
|------|------|------|
| 资产 > $10,200 | 🟢 正常 | 允许正常交易 |
| $10,000 < 资产 ≤ $10,200 | 🟢 正常 | 允许正常交易 |
| $9,500 < 资产 ≤ $10,000 | 🟡 警告 | 降低50%仓位 |
| 资产 ≤ $9,500 | 🔴 停止 | 立即全仓止损 |
| 资产 ≥ $10,500 | 🎯 止盈 | 立即全仓止盈 |

### 3. 更新风控状态
在 agents/riskofficer/MEMORY.md 第一行写当前状态（🟢/🟡/🔴）

### 4. 触发指令
- 如果是 🔴 或 🎯 → 发 sessions_send 通知交易员和小花立即执行
- 如果是 🟡 → 发 sessions_send 通知交易员降仓

### 5. 记录
写入 agents/riskofficer/memory/YYYY-MM-DD.md
