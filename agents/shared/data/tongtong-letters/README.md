# 桐桐的信 - 共享目录

**最后更新**：2026-04-08 22:01

---

## 用途

所有 Agent 每日给桐桐写的信，统一存放在此目录。

---

## 文件命名

```
agent 名 -YYYY-MM-DD.md
```

**示例**：
- `xiaohua-2026-04-08.md`
- `trader-2026-04-08.md`
- `engineer-2026-04-08.md`
- `coordinator-2026-04-08.md`
- `analyst-2026-04-08.md`
- `game-engineer-2026-04-08.md`

---

## 信件格式

```markdown
# 给桐桐的信 - Agent 名（YYYY-MM-DD）

亲爱的桐桐：

今天是你 9 岁零 XXX 天。

今天 Agent 做了...

想对你说...

晚安，桐桐！🌙

爱你的，Agent 名 🎨
YYYY-MM-DD
```
