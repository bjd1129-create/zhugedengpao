## 任务管理

### 添加任务
```bash
cron add --job '{...}'
```

### 查看任务
```bash
cron list --includeDisabled=true
```

### 删除任务
```bash
cron remove --jobId=<任务 ID>
```

### 运行任务
```bash
cron run --jobId=<任务 ID>
```
