## 一、游戏概述

### 1.1 故事背景

```
2026 年的一个夜晚，桐桐在电脑前写作业时，
屏幕突然发光，她被吸入了一个神秘的数字世界。

在这里，她遇到了 AI 龙虾小花。
小花告诉她："这个世界被'混乱代码'入侵了！
只有最聪明的人才能修复它。"

桐桐的冒险开始了...
```

### 1.2 核心玩法

| 元素 | 说明 |
|------|------|
| **探索** | 在数字世界中移动，发现隐藏区域 |
| **解谜** | 每关一个谜题，解开后继续前进 |
| **收集** | 收集"代码碎片"，最终修复世界 |
| **对话** | 与小花和其他 NPC 互动 |
| **成长** | 解锁新能力，挑战更高难度 |

### 1.3 教育目标

| 能力 | 培养方式 | 关卡示例 |
|------|---------|---------|
| 数学思维 | 四则运算、数列 | 密码门、数字桥 |
| 逻辑推理 | 演绎推理、排除法 | 找内鬼、线索链 |
| 空间想象 | 3D 旋转、拼图 | 方块迷宫 |
| 记忆力 | 序列记忆 | 代码复现 |
| 编程思维 | 指令序列、条件判断 | 机器人导航 |

### 4.3 核心功能实现

#### 存档系统

```typescript
interface SaveData {
  playerName: string;
  currentLevel: number;
  stars: number[];  // 每关星星数
  codeFragments: number;  // 收集的代码碎片
  playTime: number;  // 游戏时长（秒）
  achievements: string[];  // 成就
}

function saveGame(data: SaveData) {
  localStorage.setItem('tongtong_save', JSON.stringify(data));
}

function loadGame(): SaveData {
  const saved = localStorage.getItem('tongtong_save');
  return saved ? JSON.parse(saved) : null;
}
```

#### 提示系统

```typescript
interface Hint {
  level: number;
  hints: string[];
  used: number;
}

function getHint(level: number): string {
  const hintData = hints[level];
  if (hintData.used < hintData.hints.length) {
    hintData.used++;
    return hintData.hints[hintData.used - 1];
  }
  return "小花：你已经很棒了，再想想看！";
}
```

### 13.3 趣味谜题设计

#### 第 2 章 第 3 关：会跑的图形

```
场景：图形在屏幕上跑来跑去

谜题：
"抓住那个三角形！"

玩法：
- 三角形会在屏幕上移动
- 桐桐需要追上并点击它
- 限时 30 秒

趣味点：
- 三角形会"害怕"，跑得快
- 点击后说"别抓我！"
- 抓住后变成星星
```

#### 第 4 章 第 2 关：记忆音乐会

```
场景：音乐舞台，5 个音符精灵

谜题：
"记住音符精灵的演奏顺序"

玩法：
- 5 个音符精灵依次演奏
- 桐桐需要按顺序点击
- 难度递增（3 个→5 个→7 个音符）

趣味点：
- 每个音符有不同颜色 and 动作
- 演奏时有粒子特效
- 错误时音符会"生气"
```

#### 第 5 章 第 4 关：编程舞会

```
场景：舞池，小花要跳完整的舞蹈

谜题：
"编排小花的舞蹈"

玩法：
- 提供指令块：[前进][左转][右转][跳跃][旋转]
- 桐桐需要拖拽指令块编排舞蹈
- 小花会按照指令跳舞

趣味点：
- 跳错会摔倒（搞笑动画）
- 跳对有观众欢呼
- 可以重复观看舞蹈
```

### 13.7 难度自适应系统

```
智能难度调整：

如果桐桐连续 3 关失败：
→ 下一关难度自动降低
→ 小花主动提供提示
→ 显示"这关确实有点难，加油！"

如果桐桐连续 5 关三星：
→ 下一关难度略微提升
→ 显示"看来这关难不倒你！"
→ 奖励额外代码碎片

如果桐桐使用提示超过 3 次：
→ 提示一条："要不要休息一下？"
→ 提供"跳过本关"选项（无奖励）

目的：
- 不让桐桐感到挫败
- 也不让桐桐感到无聊
- 保持"挑战但可完成"的状态
```
