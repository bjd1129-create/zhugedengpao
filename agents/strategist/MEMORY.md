## 策略知识
### 网格交易原理
- 把价格分成若干"格"
- 每格设置买卖点
- 价格跌到下一格 → 买入
- 价格涨回上一格 → 卖出
- 震荡行情最理想

### 关键参数
- 网格间距：2%
- 网格层数：10层
- 每格金额：$1,000
- 止损线：账户 < $9,500（-5%）
- 止盈线：账户 > $10,500（+5%）

- 交易员 → 执行我的策略建议
- 风控官 → 对我的策略有否决权
