## 工程师更新流程

### 接收交易数据

```bash
# 交易员写入共享数据
cat > agents/shared/data/trades_today.json << 'DATA'
{
  "date": "2026-04-09",
  "trades": [...]
}
DATA
```

### 更新网页

```bash
# 1. 读取交易数据
cat agents/shared/data/trades_today.json

# 2. 追加到网页数据文件
python3 scripts/update_trading_page.py

# 3. 提交部署
git add website/data/trading/*.json
git commit -m "update: 交易记录 2026-04-09"
git push
```
