## 🎯 二、策略选择：防守型分批建仓策略（DCA + 网格）

### 为什么推荐这个策略？

| 维度 | 说明 |
|------|------|
| ✅ **适合新手** | 不需要盯盘，设置好买卖规则即可自动运行 |
| ✅ **抗高波动** | 分批买入平滑成本，不抄底猜顶 |
| ✅ **抗通胀** | 持有分红股/ETF，通胀期也有现金流 |
| ✅ **容错率高** | 单次判断失误影响小，长期胜率提升 |
| ❌ 不适合 | 想快速暴富、天天盯盘、All-in 单只股票 |

### 策略核心原则

1. **先防守，后进攻**：先建立安全垫（ETF + 高股息），再考虑少量成长股
2. **分批买入，绝不一次梭哈**：至少分成 3～5 批
3. **止损严格**：单只股票亏损 ≤ 10% 强制止损
4. **持仓分散**：单个标的仓位 ≤ 20%，行业仓位 ≤ 30%

### 每批买入数量计算

```
示例：SCHD 建仓 $200,000
  第一批（40%）：$80,000  → 约 1,200 股 @ ~$67
  第二批（30%）：$60,000  → 约 900 股 @ ~$67（若下跌则买更多）
  第三批（30%）：$60,000  → 约 900 股 @ ~$67（若继续跌反而成本更低）
```

**核心逻辑：** 价格低时买更多，高时买更少，自动形成"低买高卖"的成本优势。

## 🛡️ 五、风险控制

### 止损规则

| 层级 | 触发条件 | 动作 |
|------|---------|------|
| **单只止损** | 亏损达到 10% | 强制卖出，不犹豫 |
| **行业止损** | 单一行业亏损 20% | 减仓，等企稳 |
| **总账户止损** | 总资产跌至 $850,000（-15%） | 全面减仓至 50%，停止新开仓 |
| **止盈** | 单只盈利 +30% | 减半仓，锁定利润 |

### 仓位管理

```
最大单只仓位：     ≤ $200,000（20%）
最大单一行业仓位： ≤ $300,000（30%）
最大高波动股仓位： ≤ $150,000（15%）
最低现金保留：     $100,000（10%）
```

### 分散原则

| 类型 | 占比 | 说明 |
|------|------|------|
| 宽基防守ETF | 50% | SCHD + VYM + SPLV |
| 稳健成长股 | 30% | AAPL + MSFT |
| 现金/备用 | 10% | 等待机会 |
