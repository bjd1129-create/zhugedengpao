# 工作汇报生成技能

> **技能获取时间:** 2026-03-18 19:54
> **技能来源:** 老庄指示
> **适用场景:** 工作总结生成、日报周报、汇报文档

---

## 技能描述

从Session历史生成工作总结，支持日报、周报等多种格式，可输出PDF文档。自动汇总任务完成情况、成果产出、问题风险等内容。

---

## 安装方式

```bash
# 安装工作汇报生成技能
clawhub install summary-report

# 查看完整使用方法
clawhub docs summary-report
```

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **Session分析** | 从会话历史提取工作内容 |
| **日报生成** | 生成每日工作总结 |
| **周报生成** | 生成每周工作汇报 |
| **PDF输出** | 输出PDF格式报告 |

---

## 支持的报告类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **日报** | 每日工作总结 | 日常汇报 |
| **周报** | 每周工作汇总 | 周例会 |
| **月报** | 每月工作总结 | 月度汇报 |
| **项目报告** | 项目阶段总结 | 项目汇报 |

## PDF输出

```bash
# 生成日报PDF
summary-report daily --output pdf

# 生成周报PDF
summary-report weekly --output pdf

# 指定输出路径
summary-report daily --output pdf --path /path/to/report.pdf
```
