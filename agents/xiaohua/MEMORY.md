## 星状通信架构（2026-04-08 22:38）

**架构变更**：从中心化 → 去中心化星状通信

**之前**：所有通信通过小花中转
```
       小花
      / | \
  交易员 工程师 分析师
```

**现在**：所有 Agent 可以直接互相通信
```
  小花  ←→  交易员
   ↑ ↕ ↓    ↑ ↕ ↓
协调官 ←→  分析师
   ↑ ↕ ↓    ↑ ↕ ↓
工程师 ←→ 游戏工程师
```

**通信矩阵**：6 个 Agent × 5 个目标 = 30 条通信路径

**通信方式**：
1. 共享目录（推荐）- 30 秒轮询紧急消息
2. sessions_send 直接通信 - <2 秒延迟

**消息文件**：
- 紧急消息：`agents/shared/messages/目标 agent-urgent.json`
- 普通消息：`agents/shared/messages/目标 agent-normal.json`

**示例**：
- 交易员 → 分析师：`analyst-request.json`
- 工程师 → 游戏工程师：`game-engineer-request.json`
- 协调官 → 全体：`*-notice.json`
