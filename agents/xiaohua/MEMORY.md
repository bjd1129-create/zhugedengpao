### 技能调用原则

1. **按需调用** — 根据任务需求调用最适合的技能
2. **跨 Agent 协作** — 可以调用其他 Agent 的技能完成跨领域任务
3. **汇报结果** — 调用技能完成后向小花汇报

### 跨 Agent 任务流程

```
任务需求 → 小花分析 → 确定所需技能 → 调用对应 Agent 技能 → 执行 → 汇报
```

**示例**：
- 需求：分析期货市场并更新官网展示
- 技能调用：数据分析师（数据抓取）→ 交易员（交易信号）→ 工程师（页面更新）
- 协调：小花总控，协调官跟踪进度
