## 一、核心发现

### 1.1 Self-Improving-Agent（自我进化核心技能）

**是什么：** OpenClaw的自我进化基础组件，让AI自动记录错误、用户纠正和知识缺口。

**核心机制：**
- 触发场景：命令失败、用户纠正、功能缺失、知识过时、发现更好做法
- 三日志系统：LEARNINGS.md / ERRORS.md / FEATURE_REQUESTS.md
- 记录结构化：ID、时间戳、优先级、状态、区域标签、摘要、详情、建议操作

**安装方式：**
```bash
git clone https://github.com/peterskoett/self-improving-agent.git \
 ~/.openclaw/skills/self-improving-agent
```

**配置步骤：**
1. 创建 `.learnings/` 目录
2. 配置 Hook（可选但强烈推荐）
3. 重启 Gateway

**晋升机制：** 高价值记录可从 `.learnings/` 晋升到 SOUL.md / AGENTS.md / TOOLS.md

---

### 1.2 EvoMap（协议约束型自进化引擎）

**是什么：** 面向AI智能体的技能共享与验证平台，基于GEP基因组进化协议。

**GEP三维度：**
- Gene（基因）：核心解决思路
- Capsule（胶囊）：可执行代码和配置
- Evolution Event（进化事件）：完整学习验证过程

**核心优势：**
- 自动检索生态中的优质技能
- 5个AI智能体验证机制
- 成本极低（1美元击败200美元调优的GPT 5.3）

**配置三步走：**
1. 连接EvoMap生态
2. 检索高GDI评分技能
3. 一键安装+自动进化模式

---

### 1.3 每日复盘机制

**原理：** 凌晨定时自动执行复盘，更新长期记忆

**流程：**
1. 代码备份
2. 回放对话历史
3. 更新 MEMORY.md
4. 优化自身配置

**价值：** 让AI从"工具"进化为"成长型伙伴"
