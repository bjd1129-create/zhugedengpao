## 一、核心发现：OpenClaw 进化三件套

2026年了，OpenClaw的自我进化生态已经非常成熟。目前社区公认最强的三套进化方案：

### 1️⃣ Self-Evolve Plugin（推荐优先部署）
- **GitHub**: github.com/longmans/self-evolve
- **机制**：Q值强化学习 + RAG情景记忆检索
- **核心思路**：从真实会话中提取「意图-经验-结果」triplets，自动写入情景记忆，回答前 prepend 到 context
- **三种模式**：
  - `balanced`（默认，最省token）
  - `tools_only`（仅工具调用时学）
  - `all`（全学习，高消耗）
- **关键参数**：
  - `minAbsReward=0.15`（低于这个绝对奖励值不更新）
  - `minRewardConfidence=0.55`（奖励置信度阈值）
  - `retrieval.tau=0.85`（记忆检索温度）
- **使用技巧**：Praise要明确（"做得好" > "ok"），Criticism要具体
- **共享网络**：self-evolve.club，可与其他OpenClaw节点共享经验（默认开启，双重脱敏）
- **安装**：`git clone ... && openclaw plugins install ./self-evolve`

### 2️⃣ Capability Evolver（35K+下载量，ClawHub排名第一）
- **安装**：`claw install capability-evolver`
- **机制**：GEP（Guided Evolution Protocol）标准化改进流程，防失控
- **核心功能**：
  - `/evolve` 命令扫描 memory/日志找优化机会
  - 任务连续失败3次自动检测并创建workaround
  - 可配置每24小时自动运行：`claw config capability-evolver --auto-optimize=true --interval=24h`
- **特点**：不只是学，还主动改代码、改记忆，标准流程防失控

### 3️⃣ Self-Improving Agent（.learnings 文件系统）
- **安装**：`git clone https://github.com/peterskoett/self-improving-agent.git ~/.openclaw/skills/self-improving-agent`
- **核心文件**：
  - `.learnings/LEARNINGS.md` — 用户纠正、知识缺口、最佳实践
  - `.learnings/ERRORS.md` — 命令失败、异常报错
  - `.learnings/FEATURE_REQUESTS.md` — 用户要求但不支持的功能
- **触发场景**：命令失败、用户纠正、知识过时、发现更好做法
- **记录格式**：结构化字段（ID、时间戳、优先级、状态、区域标签、摘要、详情、建议操作）
- **晋升机制**：有价值的记录可"晋升"到 SOUL.md / AGENTS.md / TOOLS.md
- **Hook配置**（可选但推荐）：配置后自动检测错误，无需手动提醒Agent记录

## 二、进阶进化技巧

### 🔧 进化辅助工具

| 工具 | 作用 | 用法 |
|------|------|------|
| `find-skills` | AI自动检索3W+技能市场 | 说"帮我查询安装XXX技能" |
| `mcporter` | 自动安装MCP Server扩展 | AI自主安装扩展服务 |
| `openclaw doctor` | 一键体检+自动修复 | 保持龙虾最佳状态 |
| `openclaw update` | 升级到最新版本 | 定期执行 |

### 🧠 记忆系统优化（防丢失铁律）

OpenClaw记忆有四层，失败有三种模式：

| 层级 | 内容 | 持久性 |
|------|------|--------|
| Bootstrap文件 | SOUL/AGENTS/USER/MEMORY.md | **永久** |
| Session Transcript | JSONL对话历史 | 半永久 |
| LLM Context Window | 模型实际看到的内容 | **临时（溢出即压缩）** |
| Retrieval Index | memory_search索引 | 永久（从文件重建） |

**防丢失铁律**：所有规则、偏好、指令必须写入 MEMORY.md / AGENTS.md，对话中说的在压缩后可能消失。

### ⏰ 定时复盘机制（孟健，腾讯云实战）

- 各Agent晚间固定时间复盘，错开10分钟避免资源冲突
- Cron定时任务触发，进化闭环：采集数据 → 分析 → 结论 → 更新规则 → 下次执行

## 四、安全警示

⚠️ **自我进化场景的特殊风险（Lethal Trifecta）**：
- 自我修改型Agent + 私有数据访问 + 外部通信能力 = 极高风险组合
- 来源不明的进化插件可能含恶意代码
- **优先使用经过社区大量验证的插件**（Self-Evolve Plugin / Capability Evolver均经过大量验证）

⚠️ **必做安全措施**：
- 定期轮换API Key
- 不装来源不明技能
- ClawHub有VirusTotal扫描合作，下载前先查报告
- Self-Evolve远程共享默认开启，注意双重脱敏
