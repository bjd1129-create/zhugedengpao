## 一、核心进化工具深度分析

### 1. Self-Evolve Plugin（longmans/self-evolve）

**定位**：最推荐的自我进化插件，Q值强化学习 + RAG情景记忆

**核心机制**：
- 每次响应前：从 episodic memories 检索相关经验，注入 prompt 上下文
- 任务聚合：跨多轮聚合完整任务，检测到反馈时触发学习
- Q值更新：持续更新 utility 值 + 写入新的情景记忆

**学习模式（runtime.learnMode）**：
| 模式 | 说明 | Token成本 |
|------|------|----------|
| balanced（默认） | 仅工具调用时学习 | 中 |
| tools_only | 仅工具调用时学习（最低） | 低 |
| all | 所有通过reward门的轮次 | 高 |

**默认参数**：
- `runtime.minAbsReward: 0.15` — 最小绝对奖励
- `runtime.minRewardConfidence: 0.55` — 最小奖励置信度
- `retrieval.tau: 0.85` — 仅当相似度足够高时才注入记忆

**反馈技巧**：
- Praise 要明确（"做得好" > "ok"）
- Criticism 要具体（指出哪一步错了）
- Explicit feedback 远好于模糊消息

**远程共享网络**：
- `self-evolve.club` — 可与其他 OpenClaw 节点共享经验
- 隐私双重脱敏：本地 sanitize → LLM summarizer 再脱敏
- 默认开启，可单独禁用

**安装要求**：OpenClaw 2026.3.2+（当前运行 v2026.3.13 ✅）

### 2. Capability Evolver（ClawHub 排名第一，35K+下载）

### 3. Self-Improving Agent（.learnings 文件系统）

**定位**：经验文档化沉淀，不追求实时性

**结构**：
```
.learnings/
  ERRORS.md       — 犯过的错误
  LEARNINGS.md    — 学到的经验
  FEATURE_REQUESTS.md — 功能需求
```

**机制**：凌晨4点定时复盘，更新 MEMORY.md

### Breaking Changes（必须关注）

1. **工具权限默认收紧** ⭐⭐⭐⭐⭐
   - 新agent默认 `tools.profile: "messaging"`，无编程工具权限
   - 需要手动配置为 `"coding"` 或精确 `allow` 白名单

2. **ACP默认启用**
   - `acp.dispatch.enabled: true`，会自动路由ACP任务
   - 不使用ACP需手动禁用

3. **上下文压缩优化**
   - 5888条消息只压缩26次 → 可能导致300-400万tokens
   - 建议：定期手动触发压缩或删除旧session

## 三、稳定运行7条黄金法则（BetterClaw调研）

1. **模型路由**：不同任务用不同模型
   - Heartbeat → Haoku / DeepSeek（$0.28/$0.42 per M）
   - 简单对话 → Sonnet / Gemini Flash
   - 复杂推理 → Opus / GPT-4o
   - 效果：70-80%费用节省

2. **所有provider设硬上限**
   - 月上限设为期望值的2-3倍
   - 防止Runaway Loop烧光预算

3. **结构化SOUL.md**
   - 必须包含：Error state behavior / Conversation boundaries / Topic restrictions
   - 配置版本控制，防止更新后崩溃

4. **技能安全审查**
   - 安装前完整阅读SKILL.md代码
   - ClawHub发现341个恶意技能

5. **监控日志**
   - Self-Healing心跳每小时
   - 每日全量扫描

6. **定期进化复盘**
   - 凌晨4点自动复盘
   - Self-Evolve持续学习

7. **maxIterations限制**
   - 推荐 10-15，防止单次响应链过长

## 四、记忆系统最佳实践

**OpenClaw使用混合检索**：
- 70% 向量语义相似度
- 30% BM25关键字

**写法优化**：
- 每段只写一个主题
- 标题包含核心关键词
- 用"问题-原因-解决"结构
- 保留精确标识符（命令、路径、函数名）

**记忆层级**：
| 层级 | 文件 | 持久性 |
|------|------|--------|
| Bootstrap | SOUL/AGENTS/USER/MEMORY.md | 永久 |
| Session Transcript | JSONL对话历史 | 半永久 |
| LLM Context | 模型实际看到 | **临时（溢出即压缩）** |
| Retrieval Index | memory_search索引 | 永久 |

**文件优先原则**：所有规则、偏好、指令必须写入 MEMORY.md，对话中说的在压缩后可能消失。

## 七、重大发现

1. **工具权限默认收紧是最大风险点**：当前团队配置需检查是否需要调整
