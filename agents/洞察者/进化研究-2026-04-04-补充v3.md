### 3.2 构建结构化日志系统

```bash
mkdir -p ~/.openclaw/workspace/.learnings

# LEARNINGS.md
# - correction: 用户纠正的错误模式
# - insight: 发现的有效新方法
# - knowledge_gap: 过时/缺失知识
# - best_practice: 更优方案

# ERRORS.md
# - 命令失败
# - 集成错误

# FEATURE_REQUESTS.md
# - 用户请求的缺失功能
```

### 3.3 模型路由分层策略

```javascript
// Router Agent决策逻辑
if (task === "classification" || task === "routing") {
  model = "haiku"; // 小模型
} else if (task === "extraction" || task === "summarization") {
  model = "gpt-4o"; // 中模型
} else if (task === "complex_reasoning" || task === "code") {
  model = "gpt-4-turbo"; // 大模型
}
```

- `ℹ️ Info` - 定期检查
