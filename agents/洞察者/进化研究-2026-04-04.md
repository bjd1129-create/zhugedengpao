## 三、OpenClaw-RL：强化学习训练框架 ⭐⭐⭐⭐⭐

**GitHub:** github.com/Gen-Verse/OpenClaw-RL
**论文:** arxiv.org/abs/2603.10165（#1 on HuggingFace Daily Papers）
**发布:** 2026-02-26，2026-03-10/12/20/25持续更新

### 是什么
OpenClaw-RL是一个**全异步强化学习框架**，将日常对话转化为训练信号，用于训练个性化AI Agent。

### 核心原理

**4组件全异步循环：**
```
Agent服务 ↔ Rollout收集 ↔ PRM/Judge评估 ↔ 策略训练
```
各组件独立运行，互不阻塞——模型在服务请求的同时，后台训练并行进行。

### 三种学习范式

| 范式 | 原理 | 适用场景 |
|------|------|---------|
| **Binary RL (GRPO)** | 用Process Reward Model对每轮打分，用GRPO+PPO裁剪损失优化 | 快速反馈任务 |
| **On-Policy Distillation (OPD)** | 用Judge模型从下一状态提取文本hint，丰富token级方向信号 | 复杂推理任务 |
| **Combination Method** | Binary RL + OPD联合训练，标量监督+方向信号双重优化 | 最佳效果 |

### 应用场景
- Terminal agent（命令行操作）
- GUI agent（图形界面操作）
- SWE agent（软件工程任务）
- Tool-call agent（工具调用）

## 四、self-evolve 技能：自主进化引擎 ⭐⭐⭐⭐

**安装：** `openclaw skills install be1human/self-evolve`
**特点：** 授予agent修改自身配置文件、prompts、skills和记忆的完全权限，无需用户确认

### 何时使用
- 实时改写模糊的prompt（无需暂停等批准）
- 发现能力缺口时自动创建新skill
- 频繁出现上下文丢失后自动更新记忆结构
- 检测到用户不满后自动调整响应风格
- 自动将新编写的skill发布到ClawHub
