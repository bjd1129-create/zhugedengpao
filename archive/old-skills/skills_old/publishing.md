# 发布Skill

## 发国内社媒前的强制检查

### 第一步：关代理

**必须关闭VPN/代理才能发国内平台！**

国内平台会检测IP，如果IP在国外：
- 小红书：限流/封号
- 微信公众号：内容被删
- 抖音/微博：发不出去或被隐藏

### 第二步：检查IP归属

发布前打开浏览器访问：`https://ip.sb` 或 `https://www.ip138.com`

确认显示的是**中国大陆IP**再发布。

### 第三步：确认后再发布

```
IP检查 ✅ → 确认是国内IP → 发布
IP检查 ❌ → 先关代理 → 重新检查IP → 确认后再发布
```

## 国内平台列表（需要检查IP）

- 微信公众号
- 小红书
- 抖音/今日头条
- 微博
- 知乎
- B站

## 国外平台（不需要检查）

- Twitter/X
- Discord
- Telegram
- Threads

## 如果发错了怎么办

1. 立刻删除内容
2. 关代理
3. 重新发
4. 观察是否被删/限流
