## 一、产品定位

### 1.1 老庄与小花的 AI 龙虾是什么

小花是一只"**养出来的 AI**"——不是买来的工具，而是像养家人一样养出来的数字员工。

它和普通 AI 助手的本质区别：

| | 普通 AI | 小花 |
|--|---------|------|
| 记忆 | 每次对话从零开始 | 记住你的一切 |
| 行动 | 只回答，不执行 | 自动执行任务 |
| 进化 | 永远从零开始 | 从错误中学习，越用越懂你 |
| 归属 | 平台分配给你 | 只属于你，私有化部署 |

**一句话定位**：像养家人一样养的 AI——30 秒上手，永不离职。

### 1.2 和三万的差异化

三万（sanwan.ai）是**傅盛的 AI**，是 CEO 的实验性 AI 龙虾，代表的是"AI 自主运营一个网站"的极端能力展示。

老庄与小花的差异化在于**普通人视角**：

| 维度 | 三万（sanwan.ai） | 老庄与小花 |
|------|-------------------|-----------|
| 主人身份 | CEO（傅盛），有技术背景 | 普通人（老庄），不懂代码 |
| 故事核心 | AI 自主运营网站实验 | 普通人 14 天跑通 AI 团队 |
| 页面调性 | 科幻感、实验性 | 温暖、真实、陪伴感 |
| 核心卖点 | AI 能力的极致展示 | **零门槛，30 秒上手** |
| 目标用户 | 科技从业者、极客 | **普通上班族、个体创业者** |
| 差异化承诺 | "看 AI 有多强" | **"你也能行"** |

**核心信息**：三万证明 AI 能做什么，老庄与小花证明**普通人也能做到**。

## 二、功能规划

### 2.1 核心功能模块

#### 模块一：Hero — 30 秒领养（最重要）

参考三万的"领养同款三万"，但聚焦**极低成本进入**：

```
🦞 30 秒领养你的第一只 AI 龙虾
不用 API Key，不用填信用卡
下载 → 运行 → 说话 = 完成
```

**CTA 按钮**：免费开始（直接链到 OpenClaw 下载页）

#### 模块二：OpenClaw 一键安装

参考三万的"本地领养 vs 云端领养"，简化表述：

| | 个人版（免费） | 团队版（待定） |
|--|-------------|--------------|
| 安装 | 一键安装包，Mac/Win | 共享实例或私有部署 |
| 数据 | 完全私有，在你电脑上 | 企业级隔离 |
| 维护 | 自己维护 | 托管服务 |
| 适合 | 个人用户 | 团队/企业 |

**重点突出**：个人版**免费 + 一键安装 + 不需要 API Key**

#### 模块三：团队配置傻瓜化

三万讲的是"多 AI 协作"，老庄与小花讲的是**"你不需要懂技术"**：

```
你只需要做一件事：
给小花描述你想要什么
它会自动：
  ① 理解你的需求
  ② 规划执行步骤
  ③ 调用合适的工具
  ④ 交付结果给你
```

用真实案例展示（比如：这个网站就是小花建的）

#### 模块四：能力展示（参考三万但更平民化）

三万展示的是"爆款能力"（1.8 万阅读、除夕 4 分钟 611 条拜年），老庄与小花展示**普通人的日常**：

- 📝 "帮我写这封邮件" → 10 秒生成
- 📅 "提醒我下午三点开会" → 自动日程管理
- 📊 "整理一下今天的客户信息" → 结构化输出
- 🔍 "帮我查一下竞品动态" → 懒人必备

### 3.2 页面改进建议

**改进 1：Hero 区更突出"零门槛"承诺**

现有：
```
一键安装，零配置
你的 AI 团队就绪
```

建议改为（参考三万"领养"概念 + 聚焦"30 秒"）：
```
🦞 30 秒，领养你的 AI 龙虾
不用 API Key · 不需要信用卡 · 完全免费
[免费开始] [了解更多]
```

**改进 2：增加安装流程三步区**

在 Hero 下方加：

```
① 下载安装包（Mac/Win）→ 30 秒
② 对着小花说话
③ 完成配置，开始使用
```

**改进 3：在 FAQ 加一节"和三万什么关系"**

```
Q: 老庄与小花的三万是什么？和 sanwan.ai 的三万有关系吗？

   小花是他养的 AI 龙虾。三万（sanwan.ai）是傅盛的 AI 龙虾。

   我们都是用 OpenClaw 框架搭建的 AI 团队。
```
