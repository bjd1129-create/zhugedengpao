## 一、平台优先级建议

### 🚀 优先级排序

| 优先级 | 平台 | 理由 |
|--------|------|------|
| **1st** | 知乎 | 长文沉淀、专业背书、SEO长期价值、算法对优质内容友好 |
| **2nd** | 小红书 | 流量大、图文门槛低、可快速验证选题热度、适合冷启动 |
| **3rd** | 微信公众号 | 私域沉淀、变现路径清晰、但依赖外部引流 |
| **4th** | B站 | 视频制作成本高、中长期布局、先储备内容再发力 |

### 各平台定位与内容策略

#### 知乎
- **定位**：专业背书 + 深度分析 + SEO长尾流量
- **内容类型**：3000-8000字深度长文
- **策略**：
  - 聚焦AI工具评测、OpenClaw教程（竞品少、搜索友好）
  - 行业趋势分析文（引用数据、建立权威感）
  - 回答热点问题（蹭流量 + 引流）
- **变现路径**：知乎咨询、品牌合作文、专栏付费

#### 小红书
- **定位**：轻量化科普 + 视觉吸引 + 流量收割
- **内容类型**：图文笔记（500-1500字）+ 短视频（1-3分钟）
- **策略**：
  - AI工具速评（截图+亮点提炼，30分钟可出一篇）
  - OpenClaw使用技巧合集（"5个你不知道的OpenClaw技巧"）
  - 信息图风格的内容（如"2026年AI Agent生态图谱"）
- **变现路径**：品牌植入、带货分佣

#### 微信公众号
- **定位**：私域沉淀 + 深度读者 + 变现主阵地
- **内容类型**：长文（4000字+）+ 系列化专题
- **策略**：
  - 知乎/小红书内容二次加工（加深度、加独家观点）
  - 系列专题（如"OpenClaw实战30天"）
  -  newsletter化运营（每周固定推送）
- **变现路径**：付费订阅、广告、课程

#### B站
- **定位**：视频IP化 + Z世代覆盖 + 中长期布局
- **内容类型**：10-20分钟深度视频 / 3-5分钟速评
- **策略**：
  - 储备阶段：先出图文/短视频，积累素材和手感
  - 爆发阶段：系列视频（如"手把手带你搭建OpenClaw工作流"）
  - 内容可复用：知乎长文 → B站脚本 → 小红书图文
- **变现路径**：创作激励、品牌合作、课程

## 三、团队分工细化

### 角色矩阵

| 角色 | 负责人 | 核心职责 | 每日任务 |
|------|--------|----------|----------|
| **👁️ 洞察者** | 待指定 | 选题挖掘 + 热点追踪 + 数据分析 | 每天扫描各平台热榜，输出选题建议 |
| **✍️ 文案君** | 待指定 | 内容创作 + 文案打磨 + 多平台适配 | 根据选题完成初稿，适配不同平台风格 |
| **💻 代码侠** | 老庄 | 技术验证 + 工具实测 + 截图/录屏 | 验证工具操作步骤，提供技术素材 |
| **🎨 配色师** | 待指定 | 配图设计 + 封面制作 + 视觉风格统一 | 输出封面图、信息图、小红书配图 |
| **🚀 播种者** | 待指定 | 发布操作 + 互动回复 + 数据追踪 | 执行发布，回复评论，整理数据 |

### 各角色详细职责

#### 👁️ 洞察者
- **每天任务（30分钟）**：
  - 刷知乎热榜、AI相关话题
  - 刷小红书"AI"标签下热门笔记
  - 搜索微博/即刻/Podcast近期AI热点
  - 输出《每日选题参考》（3个选题备选 + 理由）
- **工具推荐**：新榜（数据）、即刻（热点）、知乎热榜

#### ✍️ 文案君
- **能力要求**：能写专业长文（知乎风格） + 也能写轻松种草文（小红书风格）
- **工作流程**：
  1. 接收洞察者的选题
  2. 与代码侠确认技术细节
  3. 完成初稿 → 配色师配图 → 播种者发布
- **平台风格切换**：
  - 知乎：专业、引用数据、有观点有态度
  - 小红书：轻松、有代入感、"我"的视角
  - 公众号：深度、系列感、有温度

#### 💻 代码侠（老庄本人）
- **核心任务**：
  - 每篇内容发布前，亲测工具/功能，确认真实可用
  - 提供截图、录屏GIF、演示步骤
  - 把关技术准确性（避免"实测发现根本不是这样"的尴尬）
- **素材标准**：截图清晰、标注关键步骤、录屏流畅

#### 🎨 配色师
- **设计规范**：
  - 建立《视觉风格手册》（字体/颜色/版式）
  - 输出标准化封面模板（知乎/小红书/公众号/B站各一套）
- **内容素材**：
  - 信息图（如AI工具对比图、生态图谱）
  - 小红书配图（每篇8-10张）
  - 公众号头图
- **工具推荐**：Canva（快速出图）、Figma（精细设计）

#### 🚀 播种者
- **发布规范**：
  - 建立各平台账号档案（账号/密码/绑定邮箱记录）
  - 制定《发布检查清单》（标题/标签/话题/封面/摘要）
- **互动职责**：
  - 每天花30分钟回复评论（语气要像真人，不要机器人感）
  - 主动去同类热门内容下评论引流
- **数据追踪**：
  - 记录每篇内容的阅读/点赞/收藏/评论/转发
  - 每周汇总到《内容数据追踪表》
