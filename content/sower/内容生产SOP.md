# 内容生产SOP V1.0

## 标准流程

### 1. 需求确认
```
老庄提出需求
    ↓
小花理解并分解任务
    ↓
分配给对应Agent
```

### 2. 执行阶段
```
Agent接收任务
    ↓
执行并产出初稿
    ↓
小花检查质量
    ↓
通过或打回重做
```

### 3. 审查阶段
```
安全官扫描敏感词
    ↓
检查隐私信息
    ↓
合规性验证
```

### 4. 发布阶段
```
格式适配（平台要求）
    ↓
配图制作
    ↓
定时发布
    ↓
数据追踪
```

---

## 各环节标准

### 文案标准
- 字数：800-1500字
- 原创度：>90%
- 错别字：<1/千字
- 品牌调性：符合

### 图片标准
- 尺寸：1200×630（OG图）
- 格式：PNG/JPG
- 大小：<500KB
- 内容：与文章相关

### 发布标准
- 知乎：首发
- 小红书：48小时后
- 公众号：沉淀
- 都要：关代理
