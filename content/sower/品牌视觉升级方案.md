## 色彩系统

### 主色：橙色
```
#FF6B35 - 活力橙（主色调）
#FF8B5C - 浅橙（辅助色）
#E55A2B - 深橙（强调色）
```

### 辅色：蓝色
```
#4A90D9 - 科技蓝（信任感）
#6BA3E0 - 浅蓝（辅助色）
#3A7BC8 - 深蓝（强调色）
```

### 中性色
```
#333333 - 深灰（正文）
#666666 - 中灰（次要文字）
#999999 - 浅灰（辅助文字）
#FFFFFF - 白色（背景）
#F5F5F5 - 浅灰（卡片背景）
```

## 应用场景

### 1. 社交媒体头像
- 尺寸：400×400
- 格式：PNG透明背景
- 场景：小花卖萌/工作/思考

### 2. Banner图
- 尺寸：1920×600
- 格式：JPG
- 内容：小花+文字

### 3. 文章配图
- 比例：16:9
- 格式：PNG/JPG
- 风格：插画风

### 4. OG图片
- 尺寸：1200×630
- 格式：PNG
- 必须包含logo
