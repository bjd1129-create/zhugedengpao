# 团队协作工具清单 V1.0

## 即时通讯
- **飞书**：主通讯工具（已配置）
- **微信**：备用通讯
- **Telegram**：海外用户联系

## 项目管理
- **TASKS.md**：任务队列文件
- **MEMORY.md**：长期记忆
- **每日日志**：memory/YYYY-MM-DD.md

## 内容管理
- **content/**：所有内容文件
- **diary/**：日记文件
- **research/**：研究报告
- **sower/**：运营文件

## 代码管理
- **GitHub**：代码仓库
- **Cloudflare Pages**：网站托管

## 设计资源
- **images/**：所有图片
- **配色规范**：content/sower/品牌视觉升级方案.md

---

## 工具使用规范

### 飞书
- 接收老庄指令
- 团队内部沟通
- 汇报工作进度

### TASKS.md
- 记录待办任务
- 更新任务状态
- 标记优先级

### MEMORY.md
- 重要决定记录
- 品牌信息沉淀
- 团队配置更新

### GitHub
- 代码版本控制
- 部署触发
- 协作开发

---

## 文件命名规范

### 文章
```
YYYY-MM-DD-文章标题.md
例：2026-03-29-AI龙虾的一天.md
```

### 日记
```
day-N.html
例：day-1.html
```

### 图片
```
描述-序号.png/jpg
例：xiaohua_office_01.png
```
