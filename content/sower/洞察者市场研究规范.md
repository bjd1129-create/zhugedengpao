# 洞察者市场研究规范 V1.0

## 研究对象

### 主要竞品
- 三万（sanwan.ai）- 对标对象
- 其他AI助手品牌

### 研究维度

1. **产品功能**
   - 核心功能
   - 差异化功能
   - 用户评价

2. **内容策略**
   - 文章类型
   - 更新频率
   - SEO关键词

3. **商业模式**
   - 定价策略
   - 变现方式
   - 目标用户

4. **技术架构**
   - 使用技术栈
   - 第三方服务
   - AI模型

## 研究方法

1. **手动浏览**：亲身体验产品
2. **网络搜索**：查找相关文章
3. **数据分析**：流量、排名等
4. **用户调研**：了解用户需求

## 输出格式

每份研究报告包含：
- 概述
- 核心发现（5-10条）
- 优劣势分析
- 可借鉴点
- 行动计划

## 更新频率

- 竞品动态：每周一次
- 行业趋势：每月一次
- 用户调研：按需
