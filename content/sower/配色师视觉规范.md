# 配色师视觉规范 V1.0

## 品牌视觉

### 小花形象
- 穿龙虾衣服的橙色加菲猫
- 温暖、可爱、有个性
- 用于全站mascot

### 色彩系统
```
主色：橙色 #FF6B35（代表温暖、活力）
辅色：蓝色 #4A90D9（代表科技、专业）
强调色：红色 #E53935（用于按钮、重点）
背景色：白色 #FFFFFF
文字色：深灰 #333333
```

## 图片类型

### 1. 配图（文章用）
- 比例：16:9 或 4:3
- 风格：插画风或摄影风
- 尺寸：1200×675 或 800×600

### 2. Banner（首页用）
- 比例：16:9
- 尺寸：1920×1080
- 包含文字标题

### 3. OG图片（社交分享）
- 尺寸：1200×630
- 包含logo
- 文字清晰

## 生成提示词模板

```
A cute orange Garfield cat wearing a red lobster costume, [场景描述], [风格描述], high quality illustration
```

## 输出格式
- 保存为PNG或JPG
- 文件名：描述性命名
- 存储路径：images/
