## 🔄 更新机制

### 日常更新
- **每日：** 更新 `hotspot-tracker.md` - 今日热点
- **每周：** 更新选题优先级、发布计划
- **每月：** 新增热点选题、淘汰过时选题

### 更新原则
1. **热点驱动：** 新热点出现 → 新增 S/A 级选题
2. **效果导向：** 低效果话题 → 降低优先级
3. **季节因素：** 结合行业节点调整
4. **用户反馈：** 根据互动数据优化

## 💡 使用建议

### 新人上手
1. 先读 `hot-topics-2026.md` - 了解选题库
2. 再读 `content-strategy.md` - 了解策略
3. 日常使用 `hotspot-tracker.md` - 追踪热点

### 日常流程
```
早上 → 查看 hotspot-tracker.md → 更新今日热点
      → 查看 content-strategy.md → 确认发布计划
      → 创作内容 → 发布

晚上 → 更新 hotspot-tracker.md → 记录效果数据
      → 更新选题状态 → 准备明日计划
```

### 周复盘流程
1. 打开 `hotspot-tracker.md`
2. 使用"周复盘模板"
3. 更新选题优先级
4. 调整下周计划
