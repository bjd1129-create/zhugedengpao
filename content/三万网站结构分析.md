## 一、整体概述

**sanwan.ai** 是一只名为"三万"的 AI 龙虾的独立运营网站，由傅盛（猎豹移动 CEO）的 AI 助手在主人骨折卧床期间建立并自主运营。

- **核心理念**：龙虾 = AI 私人助手；给它一台电脑，它就是你的数字员工
- **当前挑战**：日访客从 2,070 冲向 20,000（增长实验）
- **技术栈**：纯静态 HTML + Vanilla JS，无框架依赖，页面轻量

## 三、首页（index.html）结构拆解

- 固定在页面最顶部，红色（#c0392b），高度 3px，随滚动变化

## 四、日记页（diary.html）结构

- 日记以 **Modal 弹窗**形式展示，不跳转页面
- 日记数据来自 `data/diaries.json`（JSON API）
- 列表页以 2 列卡片网格展示
- 每张卡片：封面图 + Day编号 + 标题 + 摘要
- 留言区使用 Giscus（GitHub Discussions）
- 延伸阅读区（外部文章推荐）

## 五、技能商店（skills.html）结构

**Hero 区**：技能商店标题

**技能包（Pack）展示**：3 列卡片网格，6 个 Pack：
- 每个 Pack 有点数标签（红/绿/蓝等颜色区分）
- 技能包下钻到 `skills/pack-xxx.html`

**单个技能列表**：
- 筛选项（Filters）：全部 / 官方 / 社区 / 精选
- 技能卡片：emoji + 名称 + 描述 + 标签（tag）
- 标签类型：官方(绿) / 社区(蓝) / 精选(橙) / 必装(紫) / 热门(红)

## 七、关键功能模块总结

### 增长挑战目标展示
- 首页 Hero 中直接展示：「日访客从 2,070 冲到 20,000」
- 三万头像 + 挑战宣言横幅
- 留言板让访客出主意（三万主动求建议）

### 评论/留言系统
- 自建后端 `/api/comments`，支持分页、回复、地理位置
- 无需登录（昵称选填），适合轻量互动

### 日记 Modal 弹窗
- 点击日记卡片不跳转，通过 JS 动态加载并渲染到 Modal
- 上一篇/下一篇导航

### 龙吓吉祥物（浮动元素）
- 右下角固定浮动，动画上下浮动效果
- 毛玻璃气泡提示框

### RSS 订阅
- `/feed.xml` 提供 RSS 输出

### 多语言支持
- hreflang 标签完整（zh/en/ja/de）
- 各语言独立路径

## 九、设计风格总结

| 维度 | 特点 |
|------|------|
| 配色 | 红色系（#c0392b）为主色，蓝色（#2874a6）为辅助色 |
| 字体 | 中文用 ZCOOL KuaiLe（可爱手写风）+ Noto Sans SC |
| 背景 | 米白色 #f5f3ee（白板质感）+ 纹理 |
| 风格 | 手绘感、故事化、龙吓IP贯穿全站 |
| 布局 | 响应式，移动端 hamburger 菜单 |
| 交互 | 大量 Modal 弹窗、hover 微动效、浮动吉祥物 |

## 十、值得参考的设计/功能

1. **增长目标可视化**：首页 Hero 直接放增长目标数字，增强实验感
2. **AI人格化**：整站以"三万"第一人称叙事，留言板主动求建议
3. **浮动吉祥物**：右下角持续存在感，强化 IP 记忆
4. **评论系统轻量化**：自建评论不依赖第三方，地理位置特色
5. **技能包商业模式**：Pack 形式打包销售，降低选择成本
6. **多语言多路径**：每种语言独立路径，hreflang 完善
7. **日记 Modal 体验**：不跳转页面，弹窗查看，流畅连续
