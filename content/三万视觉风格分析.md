## 4. 布局风格

### 整体结构
```
导航栏（固定顶部，高度56px）
  ↓
Hero区域（padding: 66px 32px，最大宽度1000px，CSS Grid 1.1fr/0.9fr 分栏）
  ↓
白板纹理区（whiteboard类，带顶部/底部灰线装饰）
  ↓
内容区块（padding: 32px，最大宽度1000px）
  ↓
页脚（footer，分隔线）
```

### 布局参数
- **内容最大宽度**：1000px（主体），860px（Hero v2）
- **栅格**：CSS Grid，`grid-template-columns: 1.1fr 0.9fr`（桌面端Hero）
- **卡片网格**：3列（`repeat(3, 1fr)`，移动端 1列或2列）
- **响应式断点**：`768px`（移动端重新排版）

### 布局特点
- 导航栏固定 + 毛玻璃效果（`backdrop-filter: blur(12px)`）
- Hero 区用 CSS Grid 实现图文分栏，与传统单栏不同，有杂志感
- 白板纹理（whiteboard）穿插使用，模拟真实场景切换

## 5. 卡片设计

### 常规卡片
```css
background: #faf8f4
border: 1.5px solid #ddd
border-radius: 12px / 14px
padding: 18px / 14px
box-shadow: none（默认）→ hover时 0 4px 15px rgba(0,0,0,0.06)
```

### Hover效果
- `transform: translateY(-2px)` / `translateY(-3px)` 轻微上浮
- 边框颜色变为 `#c0392b`（红色）或 `#2874a6`（蓝色）
- 出现柔和阴影 `0 6px 20px rgba(0,0,0,0.08)`

### 特殊卡片
- **日记卡片**：带图片（16:9），顶部有 `day` 标签（红色/蓝色胶囊）
- **科学卡片**：无图片，顶部大emoji图标，3列网格
- **技能卡片**：横向滚动（`overflow-x: auto`），最小宽度 180px
- **连载日记卡片**：虚线边框（`border-style: dashed`），区别于普通日记

### 卡片风格总结
- 统一米白底色 + 浅灰描边，干净不单调
- hover交互统一用"上浮+边框变色"，体验一致
- 没有使用深色卡片，始终保持明亮温暖基调

## 8. 动画效果

| 动画名 | 描述 | 参数 |
|--------|------|------|
| `bounce` | 龙虾吉祥物上下弹跳 | 2s ease-in-out infinite |
| `floatup` | 心形图标上浮 | 3s ease-in-out infinite |
| `mascot-float` | 悬浮吉祥物上下飘动 | 3s ease-in-out infinite |
| `popIn` | 气泡弹入 | 0.5s ease，0.3s delay |
| `btn-glow` | 按钮光晕 | 2s ease-in-out infinite（红光） |
| `reveal` | 滚动渐入 | opacity 0→1，translateY 30px→0，0.6s |
| 轮播滑动 | 卡片轨道滑动 | `0.44s cubic-bezier(.25,.46,.45,.94)` |

### 动画特点
- **吉祥物专属动画**：龙虾 `bounce`、心形 `floatup`，让品牌吉祥物"活起来"
- **微交互动画**：hover时 `translateY(-2px)` 上浮、`scale(1.08)` 放大，有触感
- **滚动渐入** `reveal`：内容从下方淡入，增加层次感
- **轮播弹性曲线**：`cubic-bezier(.25,.46,.45,.94)`，接近 Material Design 标准
- **右下角悬浮CTA**：龙虾 + 气泡，始终可见，动画持续循环

## 11. 整体风格总结

**关键词**：温暖日记感 · 手写书法混搭 · 纸质感 · 活泼吉祥物

| 维度 | 评价 |
|------|------|
| 配色 | 暖米色 + 砖红 + 蓝，温暖有品牌个性，红色足够醒目 |
| 字体 | 中文手写体（站酷/龙藏/楷体）大量使用，Noto Sans SC 保证可读性 |
| 布局 | CSS Grid 分栏，白板纹理穿插，有杂志感但不失亲切 |
| 卡片 | 统一米白底 + 浅灰边框，hover 上浮 + 边框变色，体验一致 |
| 按钮 | 扁平 + 轻微上浮hover，圆角适中，无花哨效果 |
| 间距 | 充足留白，32px 区块间距，1000px 最大宽度，不拥挤 |
| 动画 | 吉祥物专属 bounce/float，滚动 reveal，轻盈不喧宾夺主 |
| 差异化 | 同类AI个人网站多走科技感/深色路线，三万走温暖日记风，反套路 |

**设计定位**：这不是一个"展示AI技术实力"的网站，而是一个"讲AI成长故事"的个人博客。手写体、暖米色、龙虾吉祥物、日记卡片——都在强化"养成"和"陪伴"的叙事感。红色品牌色的使用面积克制（只用于标签/按钮/强调），但足够让人记住。
