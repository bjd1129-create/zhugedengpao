## 三、本轮实际改进（直接执行）

**① 删除 lxgw CDN（render-blocking 杀手）**
- 原因：cdn.jsdelivr.net 被墙，阻塞整个页面渲染
- 行动：删除 `<link>` 引用，字体降级到 Noto Sans SC
- 效果：消除最大 render-blocking 源

**② Google Fonts 非阻塞化**
- 行动：`media="print" onload="this.media='all'"` + `preconnect`
- 效果：页面立即渲染，不等待字体下载

**③ CSS 压缩（esbuild）**
- css/style.css：46KB → 33.6KB（-28%）
- 注：该文件未被 index.html 引用（CSS 全内联），实际无效果。真正的 CSS 在 HTML 内联 85KB 中。
