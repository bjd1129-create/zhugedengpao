# 技能设计方案002：代码审查助手

## 概述

设计一个AI技能，能够自动审查代码、发现bug、提出改进建议和生成文档。

## 核心功能

1. **Bug检测**：静态代码分析，发现潜在bug
2. **代码质量评估**：评分和详细报告
3. **安全漏洞扫描**：OWASP Top 10检测
4. **性能优化建议**：识别性能瓶颈
5. **代码解释**：解释复杂代码逻辑
6. **文档生成**：自动生成代码注释和文档

## 技术实现

```
代码审查助手架构

┌─────────────────────────────────────────┐
│          代码输入                        │
│   Git仓库│PR│本地文件│IDE插件          │
├─────────────────────────────────────────┤
│          解析层                          │
│   AST分析│依赖分析│代码结构            │
├─────────────────────────────────────────┤
│          审查引擎                         │
│   规则引擎│ML模型│知识库               │
├─────────────────────────────────────────┤
│          报告生成                         │
│   问题分类│修复建议│安全评级           │
└─────────────────────────────────────────┘
```

## 交互设计

- 触发方式：命令/注释/PR评论
- 报告格式：Markdown/HTML/JSON
- 修复建议：提供代码diff
- 集成支持：GitHub/GitLab/Jira

## 支持语言

- 前端：JavaScript, TypeScript, Python
- 后端：Java, Go, Rust, Python
- 移动端：Swift, Kotlin

## 使用场景

- Pull Request审查
- 代码合并前检查
- 定时代码质量扫描
- 新员工代码指导
