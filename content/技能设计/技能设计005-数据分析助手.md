# 技能设计方案005：数据分析助手

## 概述

设计一个AI技能，能够理解数据分析需求、编写代码、生成可视化和提供洞察。

## 核心功能

1. **需求理解**：将自然语言转换为分析需求
2. **代码生成**：Python/SQL/R代码
3. **数据处理**：清洗、转换、聚合
4. **可视化生成**：自动选择图表类型
5. **洞察提炼**：关键发现和趋势
6. **报告生成**：自动生成分析报告

## 技术实现

```
数据分析助手架构

┌─────────────────────────────────────────┐
│          需求层                          │
│   自然语言│数据源│指标定义              │
├─────────────────────────────────────────┤
│          理解层                          │
│   意图识别│实体抽取│上下文理解          │
├─────────────────────────────────────────┤
│          执行层                          │
│   代码生成│数据处理│模型计算            │
├─────────────────────────────────────────┤
│          输出层                          │
│   图表│数据│报告│建议                  │
└─────────────────────────────────────────┘
```

## 支持数据源

- 数据库：MySQL, PostgreSQL, BigQuery
- 数据仓库：Snowflake, Redshift
- 文件：CSV, Excel, JSON
- BI工具：Tableau, PowerBI

## 可视化类型

- 趋势图：折线图、面积图
- 对比图：柱状图、条形图
- 构成图：饼图、环形图
- 关系图：散点图、热力图
- 地理图：地图、轨迹图
