# 技能设计方案007：设计助手

## 概述

设计一个AI技能，能够根据描述生成UI设计、设计建议和设计系统规范。

## 核心功能

1. **设计生成**：描述转UI设计稿
2. **设计建议**：配色、布局、字体
3. **组件生成**：生成可复用组件
4. **设计系统**：生成设计规范文档
5. **响应式适配**：多屏幕适配建议
6. **设计评审**：检查设计问题

## 技术实现

```
设计助手架构

┌─────────────────────────────────────────┐
│          输入层                          │
│   描述│参考图│草图│现有设计            │
├─────────────────────────────────────────┤
│          理解层                          │
│   需求分析│风格识别│约束条件           │
├─────────────────────────────────────────┤
│          生成层                          │
│   布局生成│配色生成│组件生成            │
├─────────────────────────────────────────┤
│          输出层                          │
│   设计稿│代码│规范│建议                │
└─────────────────────────────────────────┘
```

## 输出格式

- Figma可编辑文件
- React/Vue组件代码
- 设计规范文档
- 设计说明注释

## 支持场景

- 落地页设计
- App界面设计
- 组件设计
- 设计系统构建
