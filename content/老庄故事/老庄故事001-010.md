# 老庄故事001：逍遥游的起点

庄子在濮水边钓鱼，楚威王派两位大夫来请他去做官。庄子手持钓竿，头也不回地说：“我听说楚国有只神龟，死了三千年了，被供奉在庙堂之上。请问，这只龟是愿意死后留下骨头被人供奉，还是愿意活着在泥水里摇尾巴？”

两位大夫回答：“当然是活着在泥水里摇尾巴。”

庄子说：“那就请回吧。我愿意在泥水里摇尾巴。”

这是庄子著名的“濮水垂钓”故事。他拒绝了高官厚禄，选择了精神上的逍遥自在。在他看来，那些功名利禄不过是束缚人性的枷锁。楚国的相位再高，也不过是供人驱使的牺牲。

庄子的朋友惠施在梁国做相国，听说庄子要来，以为他要来抢相位，大惊失色地在城中找了三天三夜。庄子见到惠施时说：“南方有一种鸟叫凤凰，从南海飞往北海，一路上不是梧桐树不停，不是竹子不吃，不是甘泉不饮。有只猫头鹰捡到一只腐烂的老鼠，抬头看见凤凰飞过，赶紧护住自己的食物。你现在是在担心我来抢你的梁国相位吗？”

这个故事告诉我们：不同境界的人，看到的世界完全不同。惠施把相位看作珍宝，而庄子在凤凰眼中，那不过是一只腐烂的老鼠。

庄子的逍遥游，从濮水边开始。他用钓鱼的方式，钓起了整个精神的自由。

# 老庄故事002：鼓盆而歌

庄子的妻子死了，惠施前来吊唁。进门却看到庄子蹲在地上，敲着盆子唱歌。

惠施气愤地说：“你的妻子和你生活了一辈子，为你生儿育女，现在老死了，你不哭也就罢了，竟然还敲着盆子唱歌，这不是太过分了吗？”

庄子平静地说：“不是这样的。她刚死的时候，我怎么可能不悲伤？但是仔细想想，她原本就不曾出生。不只是不曾出生，而且本来就没有形体。不只是没有形体，而且本来就无气息。杂乎芒芴之间，变而有气，气变而有形，形变而有生，今又变而之死，是相与为春秋冬夏四时行也。”

他的意思是：人的生死就像四季更替，是自然的变化。妻子现在回归自然，在天地之间逍遥，我为什么要悲伤呢？

惠施无法理解这种生死观。在常人眼中，死亡是可怕的、悲伤的。但在庄子看来，死亡只是另一种形式的回归，是从有形回归无形，从有限回归无限。

这个故事展现了庄子独特的生死哲学：生不足喜，死不足悲。一切都是道的变化，我们只是暂时寄寓在这个形体之中。

# 老庄故事003：无用之用的智慧

庄子和弟子们在一座深山里行走，看到一棵大树，枝叶茂盛，但木匠路过看都不看一眼就走。弟子问木匠为什么，大树说这是棵不材之木，做船会沉，做棺材会烂，做器具会毁，做门窗会开裂，总之什么用都没有。

晚上住在朋友家，朋友杀雁来招待他们。朋友有两只雁，一只会叫，一只不会叫。杀哪只呢？朋友说杀那只不会叫的。

弟子们迷惑了：一棵树因为没用而长寿，一只雁因为没用而被杀。到底是有用好还是无用好？

庄子笑着说：“周将处乎材与不材之间。”

这是庄子著名的“无用之用”思想。在复杂的社会中，太有用了会被人利用，太没用了会被人抛弃。最好的是处于材与不材之间，看似无用，实则蕴含大用。

这棵树最后成为山间一道风景，供人乘凉，让人心旷神怡。这难道不是最大的用吗？而那只会叫的雁，能报晓、能预警，它的存在本身就是价值。

庄子的智慧告诉我们：价值不是单一的，标准不同，结果就不同。有时候，看似无用的东西，恰恰是世间最珍贵的东西。

# 老庄故事004：混沌开窍的悲剧

南海之帝叫倏，北海之帝叫忽，中央之帝叫混沌。倏和忽经常在混沌的地方聚会，混沌对他们很好。

倏和忽商量着要报答混沌的恩情，说：“人人都有七窍，用来看、听、吃、呼吸，唯独混沌没有，我们来帮他打开吧。”

于是他们每天给混沌凿一窍，凿了七天，混沌就死了。

这是《庄子·应帝王》中最发人深省的故事。倏和忽出于好心，却害死了混沌。因为对混沌来说，它原本的样子就是最好的，七窍反而成了负担。

这个故事告诉我们：不要用自己的标准去衡量别人，不要把自己的想法强加于人。庄子认为，最高的治国方式是“无为而治”，不要过度干预，让事物按其自然方式发展。

后世的儒家的“仁政”往往出发点很好，但执行起来却变成了对百姓的强制。这就像倏和忽好心办坏事一样。真正的智慧是尊重差异，允许不同，而不是用一把尺子去衡量所有人。

混沌的故事，是庄子对强制同化的深刻批判。

# 老庄故事005：庖丁解牛

梁惠王看到一个厨师在解剖牛，手起刀落，牛身分离，声音和谐，像音乐一样优美。

梁惠王问：“你的技术怎么能达到这种境界？”

厨师放下刀说：“我研究的是道，已经超越了技术的层面。最初我看到的是整头牛，三年后，我看到的就不是整头牛了。现在，我用精神去接触牛，不再用眼睛去看。”

他继续解释：牛的关节之间有缝隙，刀刃薄得几乎没有厚度。好的厨师一年换一把刀，因为刀刃会断；普通的厨师一个月换一把，因为刀刃会弯。而他的刀用了十九年，解剖了几千头牛，刀刃还像新磨的一样。

因为他的刀总是贴着骨的缝隙走，从来不碰经络相连的地方和骨头附结的地方。

梁惠王听后说：“好啊，我听到了养生的道理。”

这个故事告诉我们：做任何事都要顺应规律，不要硬碰硬。庖丁之所以能一刀下去完美分解牛，是因为他对牛的身体结构了如指掌。做事要研究规律，顺应规律，这样才能事半功倍。

同时，这个故事也告诉我们“养生”的道理：人的精神要像刀刃一样，保持锐利但不用力过猛，顺其自然，才能持久。

# 老庄故事006：望洋兴叹

秋天发大水的时候，无数小溪的水都流进黄河。黄河水涨得特别大，两岸之间都看不清对岸的牛马。

河伯沾沾自喜，认为自己是天下最壮美的了。等他顺着水流向东走去，到了北海，往东边一看，一片汪洋，看不到边际。

河伯这才改变了自己的脸色，望着无边无际的北海感叹说：“俗语说，'闻道百，以为莫己若'，我就是这样的人啊。我曾听说有一点点知识的人就自以为了不起，原来就是说我这样的人。而且我曾经认为孔子的见闻少，还认为伯夷的高义是不足信的，现在我亲眼看到了您这样广阔的大海，若不是到了您的门口，我还不知道自己的渺小，那我就危险了，我会永远被有学问的人嘲笑。”

北海若说：“井蛙不可以语于海者，拘于虚也；夏虫不可以语于冰者，笃于时也。”

这个故事告诉我们：人的见识受限于自己的环境和经历。河伯在黄河边觉得自己很了不起，但到了北海才知道什么叫真正的广阔。

真正的智者，永远保持谦虚的态度，因为知道世界之大、个人之小。而那些自以为是的人，往往是见识不够的人。

# 老庄故事007：鲁侯养鸟

有一只海鸟落在鲁国的城郊，鲁侯派人抓住它，用太牢（猪牛羊三牲）供奉它，奏《九韶》之乐让它欣赏。

结果呢？鸟头晕眼花、悲哀忧愁，不敢吃一块肉，不敢喝一杯酒，三天就死了。

这是用自己的方式来养鸟，而不是用养鸟的方式来养鸟。

庄子的结论是：此已言养鸟也，未知养己也。这则寓言告诉我们做任何事都要从对象的实际需求出发，而不是从自己的主观愿望出发。

父母教育孩子常常犯同样的错误：把自己未实现的梦想强加给孩子，用自己认为好的方式去爱孩子，却从不问问孩子真正需要什么。

治国亦然：当官的认为自己给百姓的都是好的，却从不问问百姓真正想要什么。结果往往是好心办坏事。

真正的智慧是因材施教、因地制宜，而不是一刀切。

# 老庄故事008：东施效颦

西施是越国的美女，她心口痛，皱着眉头捂着胸口。同村的丑女看见了，觉得这样很好看，也学着皱眉捂胸。

结果村里有个富人家看见了，觉得这个丑女更丑了；有个穷人家看见了，带着妻儿子女跑得远远的，再也不敢看她。

这则寓言告诉我们：模仿别人要根据自己的实际情况，不能盲目照抄。每个人都有自己独特的气质和特点，适合别人的不一定适合自己。

东施的错误不是她想变美，而是她选择了不适合自己的方式。她只看到了西施皱眉的样子很好看，却没有意识到西施本来就很美，皱眉只是因为心口痛，而不是刻意为之。

这个故事也告诉我们：真诚比技巧更重要。东施的皱眉是刻意模仿，所以看起来做作；西施的皱眉是自然流露，所以有一种病态的美。

做真实的自己，比模仿别人更有价值。

# 老庄故事009：呆若木鸡

纪渚子为宣王养斗鸡。过了十天，宣王问：“鸡可以斗了吗？”

回答说：“还不行，这鸡凭着一股意气而骄傲自负。”

过了十天再问，回答说：“还不行，它看到别的鸡的影子还有反应。”

又过了十天再问，回答说：“还不行，它还会怒目而视，气势还很强。”

再过十天再问，回答说：“差不多了。即使别的鸡叫，它也已经没有反应了，看起来像木头鸡一样。它的德行完备了，其他鸡见到它没有敢应战的，都回头就跑。”

这则寓言告诉我们：真正的高手往往看起来像木头一样，没有锐气，但实际上是“内敛”的最高境界。

真正有实力的人不需要虚张声势，因为他们不需要证明什么。而那些整天吹嘘自己的人，往往是内心不自信的人。

“呆若木鸡”这个成语后来变成了贬义词，形容人傻愣的样子，但庄子的原意恰恰相反——呆若木鸡是形容真正的高手，那种沉稳、淡定、不动声色却有绝对实力的人。

# 老庄故事010：轮扁斫轮

齐桓公在堂上读书，轮扁在堂下砍车轮。他放下工具，走上来问桓公：“您读的是什么书？”

桓公说：“是圣人的书。”

轮扁问：“圣人还在吗？”

桓公说：“已经死了。”

轮扁说：“那您读的是古人的糟粕而已。”

桓公大怒，说：“我读书，你一个做车轮的怎么能议论？你说清楚，说不清楚就处死你。”

轮扁说：“我是从我做的事中明白这个道理的。砍车轮，凿孔如果太松，车轮就不牢固；如果太紧，就装不进去。必须不松不紧，恰到好处。这只能用心去体会，不能言传。我没办法把这种技术传给儿子，我的儿子也没办法学会，所以我七十岁了还在砍车轮。”

他的结论是：“古人与其所言者，亦已去矣。”

这则寓言说明：真正精微的道理是言不尽意的，无法用语言完全表达。读书只能得到表面的知识，真正的智慧需要亲身实践才能体会。

这也是庄子“得意忘言”思想的体现：语言只是工具，意义才是目的。不要执着于语言文字本身，而要透过语言去领悟背后的深意。
