# 老庄故事011：庄周梦蝶

庄子在草地上睡觉，梦见自己变成了一只蝴蝶，翩翩起舞，非常自在。醒来之后，庄子犯糊涂了：不知道是庄子梦见自己变成了蝴蝶，还是蝴蝶梦见自己变成了庄子？

这就是著名的“庄周梦蝶”故事。庄子认为，醒着和做梦没有什么区别，人生如梦，梦如人生。有一天你可能醒来发现，自己不过是另一个人梦中的人物。

这个故事触及了哲学中最深奥的问题：什么是真实？我们的感知是否可靠？如果连自己的身份都无法确定，那我们追求的一切又有什么意义？

从现代科学的角度看，我们的大脑接收外界信号，但这些信号是否真实地反映了外界？每个人都有自己的主观世界，我们永远无法完全确认自己的认知是否正确。

庄子的智慧在于：不要执着于区分真实与虚幻，因为在某种意义上，一切都是“道”的显现。蝴蝶是道的化现，庄子也是道的化现，本质上并无不同。

这种物我两忘的境界，是庄子追求的最高精神状态。

# 老庄故事012：材与不材

庄子带着弟子进山，看到一棵大树，木匠说这是不材之木，做什么都不行，所以能长这么大。弟子问：“那它是因为没用才得以长寿的吗？”

晚上住在朋友家，朋友杀雁招待他们，有一只雁不会叫，朋友说杀这只。为什么？因为不会叫就没用。

弟子困惑了：一棵树因为没用而活，一只雁因为没用而死。

庄子笑着说：“周将处乎材与不材之间。”

这是庄子对人生的深刻思考。在现实社会中，太有用会被消耗，太没用会被抛弃。最好的是处于材与不材之间，看似无用，实则大用。

但更深的理解是：价值不是固定的，而是相对的。一棵树在木匠眼中是无用的，但在鸟眼中是家园，在行人眼中是阴凉。角度不同，价值就不同。

真正智慧的人不执着于有用或无用，而是顺其自然，根据环境调整自己的姿态。水善利万物而不争，所以能长久。

# 老庄故事013：知鱼之乐

庄子和惠施在濠水桥上游玩。庄子说：“你看，这些鱼游来游去，多么快乐啊。”

惠施说：“你不是鱼，你怎么知道鱼快乐？”

庄子说：“你不是我，你怎么知道我不知道鱼快乐？”

惠施说：“对啊，我不是你，所以我不知道你；你也不是鱼，所以你也不知道鱼。这是完全一样的道理。”

庄子说：“请回到话题开始。你说'你怎么知道鱼快乐'，说明你已经知道我知道鱼快乐才来问我。我是在濠水桥上知道的。”

这就是著名的“知鱼之乐”的故事。它讨论的是认识论的问题：我们如何知道别人或别的事物的感受？

庄子认为，有些东西不需要逻辑推理，可以直接感知。人在桥上，看到鱼儿在水中自由游动，自然而然地感受到一种快乐，这种感受不需要证明，就像我们看到孩子的笑脸，自然知道他们开心。

惠施的逻辑主义有它的价值，但庄子的直觉主义同样重要。人类认识世界的方式不仅是理性，还有感性和直觉。

# 老庄故事014：狙公赋芋

楚国有个老人养了一群猴子，他能理解猴子的心意，猴子也喜欢他。

后来家里穷了，老人想减少猴子的食物，但又怕猴子们不高兴。他先骗猴子说：“以后早上给你们三个橡子，晚上给四个，够吗？”

猴子们都生气了。

老人马上改口说：“那就早上给四个，晚上给三个，够吗？”

猴子们都高兴了。

这则寓言说明：同一个结果，可以有不同的表达方式。早上三晚上四，与早上四晚上三，总量一样，但猴子的反应完全不同。

人类的语言艺术也是如此。同样的话，换一个说法，效果完全不同。同样一个缺点，换一个角度描述，就变成了优点。

这也告诉我们：不要被表面的变化所迷惑，要看透事物的本质。总量没有变，只是安排的方式变了。如果只看表面，就会被耍得团团转。

真正的智慧是看到变化背后的不变，不要被形式所迷惑。

# 老庄故事015：大樗之树

匠人去齐国，路过一个曲辕地方，看到一棵巨大的栎树。它的树荫可以遮蔽几千头牛，树干粗到可以做成酒杯。观赏的人多得像集市一样。

匠人看都不看就走了。他的徒弟看完之后说：“我从来没有见过这么壮美的木材，为什么您不肯看一眼呢？”

匠人说：“这是一棵毫无用处的树。做什么都不行，正因为如此，它才能活得这么久、长得这么大。”

晚上这棵树托梦给匠人说：“你要把我比作像那些有用的文木吗？山楂、橘子、柚子这些果树，果实熟了就被打落，打落之后枝干也跟着毁坏。大树枝被折，小树枝被拖。它们的才能害了它们自己，所以不能终其天年而中途夭折。”

这棵树之所以能活这么久，就是因为它没有用。

无用之用，方为大用。这棵树的“无用”恰恰是它最大的价值——它不贡献木材、不结果实、不能做器具，所以人类不砍它，它得以自然生长。

人也一样，有些人的价值不在于他们做了什么，而在于他们没做什么。但在这个功利的社会，有多少人愿意做一棵“无用”的树呢？

# 老庄故事016：舀水灌顶

子舆得了严重的疾病，身体弯曲，肩膀比头还高，阴阳之气凌乱不堪。

朋友子祀来看他，问：“谁能把这当作是正常的呢？”

子舆说：“伟大啊，造物主！把我变成这个样子。”

他走到井边照镜子，说：“造物主把我的左臂变成了一只鸡，让我用它来报晓；把我的右臂变成了一张弓，射猎取乐；把我的臀部变成了车轮，让我坐在上面驰骋天下。”

他的结论是：“事物的变化有得必有失，得失不过是自然的一部分。”

这则寓言展现了庄子对待逆境的态度：无论遭遇什么，都看作是道的安排，不必悲伤，不必抱怨。身体弯曲是造物主的安排，与其抗争，不如接受。

更重要的是，子舆在困境中还能自嘲，还能发现新的价值：左臂变成鸡可以报晓，右臂变成弓可以射猎，这种乐观的态度本身就是一种智慧。

真正的逍遥不在于身体的状态，而在于心灵的自在。

# 老庄故事017：支离疏的故事

有个叫支离疏的人，他的下巴缩到肚脐以下，两个肩膀高过头顶，头发长到可以梳成髻，五官都长在背上。

他给人缝洗衣服，可以养活十口人；国家征兵的时候，他大摇大摆地走在街上，官员根本不管他。

到了赈济灾民的时候，他可以领到三钟米和十捆柴。

这则寓言告诉我们：有时候，看起来最没用的人，恰恰是活得最好的人。支离疏因为身体残疾，不能当兵，不能劳役，反而得以保全自身。

庄子的结论是：“夫支离其形者，犹足以养其身，终其天年，又况支离其德者乎？”

这句话的意思是：连形体残缺的人都能安享天年，更何况是那些德行看起来“残缺”的人呢？那些不被世俗认可、不随波逐流的人，反而可能活得更好。

这是一个深刻的道理：世俗的成功标准不一定是正确的，最适合自己的生活方式，只有自己知道。

# 老庄故事018：enton门论剑

赵文王喜欢剑术，门下有三千多剑客，他们日夜在宫殿里击剑，死伤的人每年有几百。

文王因此荒废了朝政，百姓都很担心。

太子找庄子去劝谏文王。庄子说自己是剑客，愿意用剑来说服文王。

文王见到庄子，说：“你用什么剑法？”

庄子说：“我的剑法，三寸之剑，可以在上万人的剑客中取敌首级。”

文王非常高兴，安排了七千人跟他比试，七天七夜，死伤无数，终于选出了最好的一个。

那个人出场的时候，庄子蓄势待发，文王问他：“你的剑有什么特点？”

庄子说：“我的剑是天子剑，剑长三尺，用阴阳来磨砺，用春秋冬夏来锻治。”

文王一听，大悦，立刻明白了庄子的意思。

所谓“天子剑”，是以天下为剑，治理国家，而不是在宫殿里击剑取乐。

文王从此不再沉迷于剑术，开始认真治理国家。

这则故事说明：最高明的说服，不是直接指出对方的错误，而是用对方能接受的方式，让他自己明白道理。

# 老庄故事019：臧丈人的钓鱼之道

文王到渭水边打猎，看到一个老人在钓鱼，动作安闲，姿势优雅。

文王想把这个老人请来做官，但担心群臣反对和国内舆论。于是派官员去试探，先给老人家里送了很多东西，表示文王的善意。

老人不接受，说：“我钓鱼，不是为了鱼。拿香饵钓鱼，不如用鱼饵钓鱼。我用渔具钓鱼，心里却不想鱼，所以我的鱼钩是简单的，我的鱼饵是普通的，但我得到的是多余的。”

文王最后还是把老人请来做了官。后来国家大治。

这个故事告诉我们：做事要无心而为。臧丈人钓鱼不是为了鱼，所以能钓到很多鱼；出仕不是为了官位，所以能治理好国家。

真正的高手做事，看起来漫不经心，实际上却能达到最好的效果。这就是“无为而无不为”的道理。

用心计较，反而做不好；顺其自然，反而能成功。

# 老庄故事020：匠石见栎社树

匠人经过曲辕，看到一棵栎树，被当作社神来祭祀。树荫可以遮蔽几千头牛，木工匠石看都不看一眼。

他的徒弟看完之后问：“自从我拿着斧头跟随您，从来没有见过这么壮美的木材。您为什么不肯看一眼呢？”

匠人说：“算了，不要说它了。这是一棵没有用的树。用它做船会沉，做棺材会烂，做器具会毁，做门窗会开裂，做柱子会被虫蛀。这棵树什么都不能做，正因为如此，它才能活这么久、长得这么大。”

晚上，这棵树托梦给匠人说：“你用什么标准来衡量我？那些有用的文木呢？山楂、梨、橘、柚等果树，果实熟了就被打落，枝干也跟着毁坏，大枝被折，小枝被拖，它们的才能害了它们自己。”

这则故事的核心思想是：无用之用，方为大用。一棵树因为不能做家具，反而成了神树，被人供奉。

人也一样，有时候看起来没用，实际上是大用。那些不随波逐流、不被世俗标准定义的人，往往有更大的价值。

庄子的思想是反对功利主义的，他认为人的价值不应该用功名利禄来衡量。
