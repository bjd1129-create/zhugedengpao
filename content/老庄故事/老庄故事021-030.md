# 老庄故事021：井底之蛙

一只青蛙在井里，对来自东海的大鳖炫耀说：“我在这里多么快乐啊！跳到井里，水就托住我的两腋和下巴；在软软的泥巴里跳来跳去，那些癞蛤蟆、螃蟹、蝌蚪都比不上我。我独占一坑水，想停就停，想跳就跳，这就是我的快乐。您为什么不常来看看呢？”

大鳖左脚还没伸进去，右膝已经被井口绊住了。它犹豫地退了出来，对青蛙描述东海的样子：“东海很大很深，用千里那么远形容不了，用千仞那么高量不完。禹的时代十年有九年发大水，但海水不因此增加；商汤的时代八年有七年干旱，但海水不因此减少。”

青蛙听了，惊恐地瞪大了眼睛，脸上的表情都变了。

井蛙不可以语于海者，拘于墟也。夏天的虫子不可以和它谈论冰，因为受限于生存的时间。

这则寓言告诉我们：人的见识受限于自己的环境和经历。青蛙在井里觉得自己很快乐，因为它不知道大海的广阔。

真正的智慧是认识到自己的局限性，不要用自己的标准去衡量别人。大鳖没有嘲笑青蛙，因为它知道对井里的青蛙来说，那一口井已经是它的整个世界。

# 老庄故事022：儒生盗墓

有一天晚上，两个儒生戴着月亮一样的帽子，互相帮助着爬上了高高的坟墓。

其中一个儒生先爬上去，摘掉帽子，放开头发，开始挖坟墓。另一个儒生在下面接土。

坟墓快要空了，上面那个儒生对下面那个说：“太阳出来了，你赶紧看看我的帽子。”

下面那个儒生说：“这里没有帽子，要帽子干什么？”

上面那个儒生说：“你真是不知轻重！帽子有什么用？帽子是用来保护脑袋的。现在我们的脑袋都快要没了，还管什么帽子！”

这则故事是庄子对儒家虚伪的讽刺。儒家的礼仪规矩繁琐，表面上一套，背地里一套。嘴上说着仁义道德，做的却是苟且之事。

真正的道家认为，道德不在于形式，而在于内心。与其花时间打扮帽子，不如注意真正重要的事情。

# 老庄故事023：伯乐治马

管理马的人，用烙铁烧红后在马身上烙印，用剪刀剪马毛，用铲子铲马蹄，给马带上笼头，钉上蹄铁，用鞭子抽打它们。

这样一来，马就死了一半了。

真正的养马之道是什么呢？让它们在广阔的原野上奔跑，吃草喝水，高兴的时候就互相摩擦脖子。馬的天性就是这样，但伯乐说“我善于管理马”，结果把马管理得死了一半。

这则是对人类干预自然的批判。庄子的观点是，最好的管理是顺其自然，而不是过度干预。

教育孩子也一样，与其用各种规矩束缚他们，不如让他们自由发展。强迫性的教育往往适得其反，真正的才能是在自由中自然成长的。

# 老庄故事024：触氏蛮丛

有两个国家，一个在蜗牛的左角，叫触氏；一个在蜗牛的右角，叫蛮氏。两国经常打仗，争地盘，死了好几万人。

他们追击败兵，十五天才能回来。

这就是庄子眼中的“国家”观念。在更高的视角下，国家之间的争斗不过是蜗牛角上的争斗。

从宇宙的角度看，地球不过是一粒尘埃，中国不过是一点尘埃上的一点点，而人类不过是尘埃上的微生物。那人与人之间的争斗、国与国之间的战争，又算得了什么呢？

这个故事告诉我们：要有宏观的视角，不要被眼前的小事所困扰。一切争斗在无限的时间和空间面前，都是微不足道的。

# 老庄故事025：真人之心

什么是真人？古时候的真人，不追求成功，不害怕失败，不欣喜得到，不忧虑失去。来了就接受，走了就忘记，遵循自然而不求改变。这就是真人。

真人的呼吸很深，用脚跟呼吸；普通人的呼吸浅，用喉咙呼吸。违逆之处越多，欲望越多，离道就越远。

真人不知道悦生，不知道恶死。出生不欣喜，入死不拒绝。自由地来，自由地去。不忘记自己的来源，不寻求自己的去向。接受一切，这就是不用心智损害大道，不用人为辅助天然。

这就是真人的境界：无心而为，顺其自然，不以物喜，不以己悲。

# 老庄故事026：支离其德

州支离益是个残疾人，但他的嘴唇向外翻着，肩膀比头还高，颈椎朝天，脏腑都挤在一处，大腿和胸可以并排走路。

但卫国的君主说他心地忠诚，让他做臣子。

齐国的丞相每次出门，车马都跟随着他，馈赠他东西的人不断。支离益却说：“你们听说过哀它这个人吗？它没有尾巴，却想把尾巴的事推掉；它没有脚，却想把脚的事推掉。我这个人的身体，残缺不全，还想把道德的事推掉。残缺不全的身体，犹足以养其身，终其天年，又况支离其德者乎？”

这说明：在这个世界上，有时候看起来最没用的人，恰恰是最有用的人。

# 老庄故事027：渔夫与屈原

屈原被放逐之后，在江边吟诗，一副憔悴的样子。

一个渔夫问他：“您不是三闾大夫吗？怎么落到这个地步？”

屈原说：“世上的人都浊，就我清白；众人都醉，就我清醒。所以被放逐了。”

渔夫说：“真正聪明的人，不固执己见，能顺应时势。世人皆浊，为什么不跟他们一起搅浑水？众人都醉，为什么不吃他们的酒糟喝他们的酒？为什么要想得那么高，表现得那么清高，让自己被放逐呢？”

屈原说：“我听说，刚洗过头的人一定要弹掉帽子上的灰尘，刚洗过澡的人一定要抖掉衣服上的尘土。谁能让自己的清白之身，受到外物的污染呢？我宁愿跳入湘江，葬身鱼腹，也不愿让我的洁白沾染上世俗的尘埃。”

渔夫摇着船桨走了，唱着歌：“沧浪之水清兮，可以濯我缨；沧浪之水浊兮，可以濯我足。”

这就是庄子和屈原的区别。屈原执着于清白，不愿与世俗同流合污，最终投江而死。庄子则主张与世浮沉，不固执己见，顺其自然。

# 老庄故事028：阳子居见老子

阳子居去见老子，说：“假如有一个人，办事敏捷，思考周密，行动迅速，理解力强，精通技艺，这样的人可以跟得上大道吗？”

老子说：“这样的人在老虎面前就危险了。因为老虎有它自己的方法来感知人，猎人有它自己的方法来杀虎。所以我说，那种有才能的人，能保存自己就很不错了。”

阳子居变了脸色说：“那么大道是什么呢？”

老子说：“最完美的人，行事谨慎但不劳累，言语精当但不炫耀，行为端方但不害人。他清静虚无，顺其自然。”

阳子居听了，说：“从前我听说您是得道之人，现在我真正见到了道。您这样的人，清静虚无，顺其自然，这才是真正的道啊。”

# 老庄故事029：秦失吊老聃

老聃死了，他的朋友秦失去吊丧，哭了三声就出来了。

弟子问：“他不是您的朋友吗？”

回答说：“是的。”

弟子问：“这样吊丧，可以吗？”

回答说：“可以。我以前认为他是个人，现在知道他不是。刚才我进去吊丧，看到有老年人哭他，像哭自己的孩子；有年轻人哭他，像哭自己的父母。一定有什么话是他们不应该说而说的，是迫不得已才说的，是自然规律在催赶他们。”

“他们在聚合的时候一起悲伤，违反了自然的规律，忘记了我们所禀受的是自然给的。古代的真人叫做'悬解'，那些不能自解的人，是被外物束缚的人。”

这就是庄子对死亡的态度：死亡是自然的一部分，不应该悲伤。那些悲伤的人，是因为没有看透这一点。

# 老庄故事030：浑沌开窍（再述）

南海之帝为倏，北海之帝为忽，中央之帝为浑沌。倏与忽时相与遇于浑沌之地，浑沌待之甚善。

倏与忽谋报浑沌之德，说：“人皆有七窍，以视听食息，此独无有，尝试凿之。”

日凿一窍，七日而浑沌死。

这个故事的核心是：不要用自己认为好的方式去对待别人。倏和忽认为七窍是好的，所以要给浑沌凿七窍，但浑沌并不需要七窍，它原本的状态就是最好的。

真正的善意是尊重对方的需要，而不是把自己认为好的东西强加于人。
