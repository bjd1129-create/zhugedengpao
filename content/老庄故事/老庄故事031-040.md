# 老庄故事031：任公子钓鱼

任公子做了一个巨大的鱼钩，用很粗的黑绳子做鱼线，用五十头阉割过的牛做鱼饵。他蹲在会稽山上，把鱼竿投向东海。

一年过去了，什么也没钓到。

忽然间，大鱼游过来，掀起巨大的波浪，声音像鬼叫，海水被搅动，千里之内都能听到。

大鱼上钩了。任公子把这条鱼剖开做成肉干，从浙江以东到苍梧以北，所有人都吃饱了。

那些用小渔竿在小水沟钓鱼的人，议论纷纷，想学他的方法。但那些人永远也钓不到这样的大鱼。

这则故事告诉我们：要做大事，就要有大的准备。任公子用五十头牛做饵，蹲在会稽山上，等了一年，终于钓到了大鱼。

那些用小渔竿在小水沟钓鱼的人，永远也钓不到大鱼。不是因为他们不努力，而是因为他们的格局太小。

做事要有耐心，要舍得投入，要能看到长远的结果，而不是只盯着眼前的小利。

# 老庄故事032：臧丈人垂钓

文王到渭水之阳打猎，看到一个老人在池塘边钓鱼，动作安闲，姿势优雅。

文王问他：“您钓鱼的技术很高明啊。”

老人说：“我听说，钓鱼这件事，用鱼饵钓鱼不如用鱼钓鱼。用有形的鱼饵钓鱼，只能得到鱼；用无形的心钓鱼，才能得到大道。”

文王听了，说：“您说的道理很深啊。”

老人说：“我听说伊尹曾经背着锅碗瓢盆去求见商汤，用烹调技术来说服汤王。姜太公拿着钓竿在渭水边钓鱼，遇到文王。但他们一开始都不是为了钓鱼，而是为了求见贤明的君主。”

文王听了，若有所思。

# 老庄故事033：匠石运斤成风

惠子和庄子在濠梁之上辩论完之后，各自散去。

后来，惠子死了，庄子路过惠子的坟墓，对跟随的人说：“从前郢地有一个人，用白灰涂在自己的鼻尖上，像苍蝇的翅膀一样薄，让一个匠石用斧子来砍。匠石挥动斧子，风声呼呼，顺着鼻尖砍下去，白灰被削掉了，但鼻子一点也没有受伤。”

“郢地的人站在那里，面不改色，宋元君听说后，派人找匠石说：'请你也给我表演一下。'匠石说：'我确实能砍，但我的对手已经死了。'”

庄子感叹说：“现在惠子死了，我再也没有对手了，没有人能和我辩论了。”

这则故事说明：真正的友谊是在精神层面上的。匠石能表演他的绝技，是因为有郢地那个不怕死的人。庄子能和惠子辩论，是因为惠子是他真正的对手。

朋友不在多，而在精。有这样一个懂你的人，一个能和你辩论的人，一个能激发你思考的人，人生足矣。

# 老庄故事034：南伯子綦的樱桃树

南伯子綦在商丘一带游历，看到一棵巨大的樱桃树。但这棵树大到可以遮蔽几千头牛，树干粗到可以做成酒杯。

奇怪的是，树上结满了樱桃，但没有人来摘。

子綦说：“这是什么树？这树一定有不同于一般树的用处。”

抬头看树的枝条，弯弯曲曲，不能做栋梁；低头看树干，松散腐败，不能做棺材；舔一下树叶，嘴里就烂了；闻一下气味，醉三天都醒不过来。

子綦说：“这真是一棵不成材的树，所以能活这么久。这就是'无所可用'的道理。”

《人间世》中的这则寓言告诉我们：在这个世界上，有时候看起来无用的人，反而能活得更好。那些被社会需要的人，往往被消耗得很快；而那些看起来无用的人，反而能安享天年。

# 老庄故事035：支离疏的智慧

支离疏这个人，下巴缩到肚脐以下，两个肩膀比头还高，颈椎指向天空，五脏都长在背上。

他替人缝洗衣物，可以养活十口人。国家征兵的时候，他大摇大摆地走在街上，官员根本不管他。

赈济灾民的时候，他领到三钟米和十捆柴。

支离疏说：“我利用我的残疾，享受天年；而那些身体健全的人，反而被征兵打仗，被劳役累死。”

这就是庄子的智慧：福祸相依，祸兮福所倚。有时候看起来的不幸，恰恰是最大的幸运。

# 老庄故事036：孔子的困惑

孔子被围困在陈国和蔡国之间，七天没有升火做饭。

他若无其事地弹琴唱歌，但脸色没有变。

颜回在一边择菜，子路和子贡在一边说：“先生两次被赶出鲁国，在卫国被人铲掉车迹，在宋国被人砍掉讲学的树，在商、周的故地被人围困，现在又在陈国和蔡国之间被人围困。要杀先生的人，没有罪行却没有停止过。而先生还弹琴唱歌，没有丧事的样子，君子这样羞辱自己，不是太过分了吗？”

颜回说不出话来，只能进去告诉孔子。

孔子推开琴，笑着说：“那他们是很粗鄙的人。”

然后叫颜回进来，说：“《易经》上说：'君子以恐惧修身。'我现在被围困，是天命的考验吗？我应该继续求道而不顺利，这难道不是天意吗？”

# 老庄故事037：渔夫与神龟

有人用一张有九个眼的网在江边捕鱼，很快捕到了很多鱼。

神龟用爪子弄破了网，逃走了。

有个渔夫捡到了这只龟，就把它献给江潭的龙王。龙王很高兴，把龟放在宫殿里，用珠和贝壳装饰它，给它起名叫“珠龟”。

它享受了六千年的寿命，最后还是死了。

这则故事告诉我们：被人用网捕到的龟，虽然能享受荣华富贵，但最终还是会死；而在江里用爪子弄破网逃走的龟，虽然要自己谋生，但能活得更久。

有时候，自由比富贵更重要。被人赏识看起来是好事，但代价是失去自由；默默无闻看起来是坏事，但能保持独立。

# 老庄故事038：老莱子的教子

老莱子的妻子从外面回来，说：“你为什么坐在这里？”

老莱子说：“我听说，楚王想让我去做官，我不接受，现在不知道该怎么办。”

妻子说：“我给你酿酒，酒酿好了就摆宴席。我去山上采菜，放在席上。我对使者说，这不是我做的，是我自己劳动所得。”

老莱子高兴地说：“好。”

于是他们搬家了，再也没有人找到他们。

这则故事说明：真正的隐士不需要刻意躲藏，只要不贪图富贵，自然就不会被权贵所累。

# 老庄故事039：子贡的菜园

子贡路过菜园，看到一个老丈人在浇菜园，挖了一条地洞到井里，用瓮浇水，费力多而功效少。

子贡说：“有一种机械，一天可以浇一百畦，用力很少而功效多，先生不想用吗？”

老人变了脸色，笑着说：“我听我的老师说，使用机械的人，一定会用机巧之心。心中有了机巧之心，就会失去纯朴的精神。失去纯朴的精神，就会心神不定，心神不定的人，就不能承载大道。”

子贡感到羞愧，脸色发红，一句话都说不出来。

这则是对技术文明的批判。庄子认为，技术越进步，人心越复杂，离道越远。

# 老庄故事040：庄子的贫困

庄子家里很穷，去找监河侯借米。

监河侯说：“好的，等我收了租子，借给你三百金，可以吗？”

庄子变了脸色，说：“我昨天来的时候，在半路上听到有人叫我的名字。我回头一看，是车辙印里的一条鲋鱼。”

“我问它：'鲋鱼，你在这里干什么？'它说：'我是东海的波臣，你能不能给我一升半斗的水，救救我的命？'”

“我说：'好的，我去南方游说吴国和越国的国王，引西江的水来救你，可以吗？'”

“鲋鱼变了脸色，说：'我失去了我习惯的水，我现在的处境是只要有一点水就能活。你说这种话，不如早点到卖干鱼的铺子里去找我。'”

这就是庄子的智慧：说要引西江的水来救一条困在车辙印里的鲋鱼，说得好听，但不切实际。真正有用的是立刻给一升半斗的水，而不是空许诺言。
