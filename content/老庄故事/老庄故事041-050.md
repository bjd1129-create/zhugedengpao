# 老庄故事041：盗亦有道

盗聖问盗跖：“做强盗也有道吗？”

盗跖说：“当然有道。进到屋子里，能猜到里面有什么宝贝，这叫圣明；第一个进去的，叫英豪；最后出来的，叫义气；知道能不能动手的，叫智慧；分赃均匀的，叫仁义。”

“不具备这五德，而能成为大盗的，天下从来没有过。”

这则故事是庄子对儒家的讽刺。儒家讲仁义礼智信，但盗聖说，强盗也有这些品德。所以儒家的道德标准并不是最高的，也不是独有的。

庄子通过这个故事说明：所谓的道德规范，只是人为的设定，并不是天然的真理。强盗可以讲仁义，君子也可能不仁不义。形式上的道德，没有实际的价值。

真正的道德是发自内心的，而不是装出来的。

# 老庄故事042：河伯与海若再论

河伯在秋天发大水的时候，顺流东行，到了北海，往东一看，看不到边际。

河伯望洋兴叹，对海若说：“俗语说，'闻道百，以为莫己若'，我就是这样的人。我听说有一点点知识的人就自以为了不起，原来就是说我这样的人。”

海若说：“井蛙不可以语于海者，拘于墟也；夏虫不可以语于冰者，笃于时也；曲士不可以语于道者，束于教也。”

“现在的你，终于看到了大海，知道自己的渺小了，我可以和你谈论大道了。”

“海有五类：东西北三面有边际，南面没有边际。计算一下，一昼夜之间，天下的水注入大海里，大海也不见增减；一年四季，天下的水流进大海里，大海也不满不空。”

这就是大海的境界：无论注入多少，都不见增减；无论流出多少，都不满不空。

真正有德行的人，就像大海一样，包容一切，不以物喜，不以己悲。

# 老庄故事043：伯昏无人射箭

伯昏无人邀请后羿和羿去登山，站在悬崖边上，把脚分了一半在外面，邀请羿来射箭。

羿紧张地往后退了一步。

伯昏无人说：“唯有至人，才能上窥青天，下潜黄泉，挥斥八极，神气不变。现在你眯着眼睛，心里想着射箭的技巧，注意力分散，怎么能射中呢？”

这就是庄子的道理：真正的技艺，不在于技巧，而在于心神。只有心神合一、物我两忘的人，才能达到最高的境界。

那些过分强调技巧的人，永远达不到最高点，因为他们的心神不宁。

# 老庄故事044：丑人娶美人

阳子的母亲很丑，但她被选入后宫，生了阳子。后来她又生了两个孩子，都被宠幸了。

这则故事说明：美丑不是固定不变的。在这个环境下是丑的，在另一个环境下可能就是美的。

人的价值也是如此。别人眼中的废物，可能是另一个人眼中的宝贝。不要被别人的评价所左右，要相信自己的价值。

# 老庄故事045：支离其形

有人名叫疏这个人，住在菜市场里，大家都看不起他，但卫国国君说他德行好，让他做臣子。

这说明：世俗的评判标准不一定是正确的。那些被社会边缘化的人，可能恰恰是有大德的人；而那些社会地位高的人，可能恰恰是道德败坏的人。

不要用外在的标准来衡量人的价值。

# 老庄故事046：商人的秘密

宋国有个商人，到了越国，卖出一种防晒的膏药，结果发了财。

有人问他：“你用什么方法赚这么多钱？”

商人说：“我只是卖防晒膏药而已。但越国的人习惯纹身，他们认为皮肤黑是美的，所以防晒膏药对他们来说是必需品。”

这个故事说明：商机在于发现别人的需求。同样的产品，在这个地方可能无人问津，在另一个地方可能供不应求。

创业的智慧在于：不要做你认为好的产品，而是做市场需要的产品。

# 老庄故事047：臧丈人的故事

文王想请一个隐士来做官，但担心群臣反对。于是派人在夜晚送礼物给这个隐士。

隐士拒绝了，说：“我听说，君子有道则现，无道则隐。现在我在这里隐居，就是为了保持我的志向。你送礼物来，不是想让我违背我的志向吗？”

这则故事说明：真正的高人，不会为了利益而改变自己的志向。他们宁可贫穷，也不愿失去精神上的自由。

# 老庄故事048：庚桑楚的弟子

庚桑楚是老子的弟子，他说：“我整治百姓的方法，已经三年了，百姓都不敢有非分的想法。”

他的弟子南荣畴趔着脚跟迷惑不解。

庚桑楚说：“不要有这种表情。我说的是，让百姓不要有什么想法，不要有什么智巧，不要有什么欲望，让他们自然地生活。”

这就是庄子的“无为而治”的道理：不是什么都不做，而是不强制干预，让百姓自然发展。

# 老庄故事049：颜回的坐忘

颜回告诉孔子：“我进步了。”

孔子问：“什么意思？”

颜回说：“我忘记了仁义。”

孔子说：“还不够。”

过了几天，颜回又说：“我进步了。”

孔子问：“什么意思？”

颜回说：“我忘记了礼乐。”

孔子说：“还不够。”

过了几天，颜回又说：“我进步了。”

孔子问：“什么意思？”

颜回说：“我坐忘了。”

孔子惊讶地说：“什么叫坐忘？”

颜回说：“不用耳去听，不用心去记，不用形去感知，耳目心三者并用，与大道融为一体，这就叫坐忘。”

孔子说：“与大道融为一体，则好恶不能干扰你。如果你能做到这样，我请求做你的学生。”

这就是“坐忘”的境界：忘记一切外在的形式，达到与道合一的境界。

# 老庄故事050：庄子的最后时光

庄子的生命快要结束了，他的弟子想给他一个隆重的葬礼。

庄子说：“我用天地为棺椁，日月为连璧，星辰为珠玑，万物为送葬。我的葬礼还不够隆重吗？还有什么比这更隆重的？”

弟子说：“我怕乌鸦和老鹰吃掉你。”

庄子说：“在地上被乌鸦和老鹰吃掉，在地下被蝼蛄和蚂蚁吃掉。非要夺走乌鸦的食物给蚂蚁，不是偏心吗？”

这就是庄子的最后境界：生死如一，视死如归。他把自己完全交给自然，不执着于任何形式。
