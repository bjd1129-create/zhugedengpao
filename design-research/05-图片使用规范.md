# 图片使用规范

## 概述

图片是品牌视觉传达的核心载体之一。在数字产品和品牌传播中，图片承担着传达情感、传递信息、建立信任和强化品牌识别的多重职责。本规范旨在建立一套系统化的图片选择标准、管理流程和使用准则，确保所有品牌接触点上的图片视觉表现一致、专业、有效。

本规范覆盖图片的来源管理、质量标准、风格规范、处理流程、存储格式、版权合规和实际应用场景，适用于网站、App、社交媒体、广告投放、印刷物料等所有媒介形式。

---

## 第一章：图片来源管理

### 1.1 图片来源分类

品牌图片来源分为四大类，每类有不同的管理规范和使用权限：

**A类：品牌原创图片**
- 定义：由品牌内部团队（设计师、摄影师）原创拍摄或制作的图片
- 所有权：品牌拥有完整版权
- 使用权限：全平台、全形式、无限制使用
- 管理要求：原始文件归档，建立命名规范，标注拍摄/制作日期

**B类：签约摄影师/机构图片**
- 定义：与品牌签约的外部摄影师或图库机构提供的定制图片
- 所有权：依据合同约定（买断版权或授权使用）
- 使用权限：依据合同约定范围
- 管理要求：合同归档，记录授权范围和到期时间，定期核查

**C类：商业图库图片**
- 定义：从Shutterstock、Getty Images、Unsplash等商业图库授权获取的图片
- 所有权：获取使用授权（非版权买断）
- 使用权限：依据具体授权协议，注意扩展授权（Extended License）和标准授权的区别
- 管理要求：保留授权凭证（截图、下载记录、发票），记录授权到期时间

**D类：用户生成内容（UGC）**
- 定义：由用户创作并经用户授权使用的图片
- 所有权：用户保留版权，品牌获得在特定场景的使用授权
- 使用权限：仅限授权时明确的场景
- 管理要求：获取书面使用授权（App内弹窗授权、邮件确认等），记录授权人信息

### 1.2 图片采购流程

**标准采购流程**：
1. **需求发起**：使用部门提交图片需求，包含用途、尺寸、数量、风格描述、预算范围
2. **方案评估**：品牌/设计团队评估是否可使用现有图片库或需要采购新图
3. **来源选择**：优先使用品牌原创图片 → 次选签约资源 → 最后考虑商业图库
4. **合规审核**：法务或合规团队确认版权状态
5. **采购执行**：通过财务审批流程进行采购
6. **归档入库**：图片入库品牌资产库，标注来源和使用权限

**禁止行为**：
- 直接从搜索引擎下载图片（未经验证版权状态）
- 使用他人已发布内容截图（即使注明来源也不可取）
- 使用带有其他品牌LOGO或版权信息的图片（，除非构成合理使用）

### 1.3 品牌图片库管理

**数字资产管理（DAM）系统**：品牌应建立或使用数字资产管理平台集中管理所有图片资产：

**图片库结构建议**：
```
/品牌图片库
├── /原创图片
│   ├── /2026年第一季度
│   ├── /2025年第四季度
│   └── /历史存档
├── /签约资源
│   ├── /签约摄影师A
│   └── /签约机构图库
├── /商业授权
│   ├── /已购买授权
│   └── /授权待确认
├── /UGC授权
│   └── /用户授权存档
└── /参考收藏
    ├── /竞品分析
    └── /趋势参考
```

**元数据规范**：每张入库图片必须标注：
- 文件名（遵循命名规范）
- 版权状态（A/B/C/D类）
- 授权范围和到期时间
- 使用记录（何时何地使用过）
- 内容标签（便于搜索）

## 第二章：图片质量标准

### 2.1 分辨率与尺寸规范

**数字屏幕应用**：
| 应用场景 | 最小分辨率 | 推荐分辨率 | 文件格式 |
|----------|------------|------------|----------|
| 社交媒体头像 | 400×400px | 800×800px | PNG/JPG |
| 网站Hero图 | 1920×1080px | 2560×1440px | JPG/WebP |
| 社交媒体信息流 | 1200×630px | 2400×1260px | JPG/WebP |
| App启动图 | 1536×2048px | 2048×2732px | PNG |
| 电子邮件Banner | 600×200px | 1200×400px | JPG/GIF |
| 广告投放-大图 | 1200×628px | 2400×1256px | JPG/WebP |

**打印应用**：
| 应用场景 | 分辨率要求 | 色彩模式 |
|----------|------------|----------|
| 杂志/画册 | ≥300dpi | CMYK |
| 海报（大幅） | ≥150dpi | CMYK |
| 名片 | ≥300dpi | CMYK |
| 包装盒 | ≥300dpi | CMYK |

**关键原则**：图片不得通过放大来满足尺寸要求——只能从大图缩小使用，不能从小图放大使用。

### 2.2 图片质量评估维度

**清晰度（Sharpness）**
- 图片主体边缘清晰，无明显噪点或压缩伪影
- 使用双三次插值（Bicubic）算法进行缩放
- 避免使用过度锐化的图片

**构图（Composition）**
- 主体位置符合视觉重心原则
- 留白区域足够放置文字或其他元素
- 无分散注意力的杂乱背景

**光线（Lighting）**
- 光线均匀，无局部过曝或欠曝
- 色温一致（除非刻意营造特殊氛围）
- 光影方向统一

**色彩（Color）**
- 色彩饱和度适中，符合品牌调性
- 无色偏（除非刻意风格化处理）
- 与品牌色彩系统兼容

### 2.3 禁止使用的图片类型

以下类型的图片**严格禁止**在品牌传播中使用：

- **库存照片感**：过度修图、人物表情不自然、场景不真实
- **分辨率不足**：任何低于应用场景最低要求的图片
- **水印残留**：带有其他品牌水印或版权水印
- **过度处理**：严重失真、夸张滤镜、脱离真实感
- **争议性内容**：涉及政治、宗教、暴力、歧视等敏感内容的图片
- **隐私问题**：未获授权的私人场景、未成年人特写、医疗机构内部

## 第三章：图片风格规范

### 3.1 品牌图片风格定义

品牌图片应统一遵循以下风格特征，确保视觉一致性：

**人物图片风格**
- **真实性优先**：真实场景中的人物，避免过度摆拍
- **自然表情**：展现真实情感，不做作
- **光线自然**：柔和的自然光或受控的人造光，无生硬阴影
- **多元包容**：展示不同年龄、性别、种族、体型、职业的人群
- **穿着得体**：符合品牌调性的着装规范，不过度暴露
- **场景真实**：工作/生活真实场景，非布景摆拍

**产品图片风格**
- **背景简洁**：白色或浅灰色纯色背景（标准产品图）
- **角度规范**：正面、侧面、45度角、细节特写标准化
- **光线一致**：统一的光线方向和强度，便于产品对比
- **无透视变形**：产品主体无倾斜或扭曲
- **真实性呈现**：不刻意隐藏产品缺陷，不过度美化

**场景/环境图片风格**
- **空间层次**：前景、中景、背景层次分明
- **光影叙事**：通过光影传达情绪和空间感
- **品牌元素植入**：场景中自然植入品牌元素（非强制露出）
- **真实感**：真实的城市/自然/室内场景，非CG生成

### 3.2 人物图片的禁用场景

- 不得使用人物图片暗示其具有某种医疗状况
- 不得使用图片扭曲或嘲笑特定群体
- 不得使用图片暗示某种产品能改变人的外貌/身体
- 不得不恰当地使用已故名人或公众人物图片
- 未经本人书面授权，不得使用真实员工/用户照片用于非约定的场景

### 3.3 图片与文字叠加规范

当图片需要叠加文字时，必须遵循以下规范以确保可读性：

**文字可读性保障**：
- 文字与背景的对比度不低于4.5:1（WCAG AA标准）
- 在图片上放置文字时，优先使用以下方法之一：
  - 在文字下方添加半透明色块（品牌主色或黑色，50-70%透明度）
  - 使用文字阴影（Text Shadow）增强可读性
  - 使用品牌渐变蒙版（从边缘向中心的渐变透明）

**文字放置区域**：
- 优先放置在图片的"安全区"（低信息密度区域）
- 避免放置在图片主体人物面部或关键信息上
- 文字方向应与图片主体的视线方向保持一致（视线引导原则）

## 第四章：图片处理规范

### 4.1 品牌色彩校准

所有品牌图片在正式使用前需经过品牌色彩校准：

**校准标准**：
- **色温**：统一偏暖（10-15%暖色调），避免过冷或过暖
- **饱和度**：适度提升饱和度（10-20%），使色彩更生动但不失真
- **对比度**：适度增加对比度（5-10%），增强视觉冲击力
- **肤色统一**：人像图片需进行肤色调统一处理，避免色温不一致

**校准工具**：推荐使用Adobe Lightroom、Camera RAW或Capture One进行批量校准，并保存预设（Preset）以确保一致性。

### 4.2 图片裁剪规范

**构图裁剪原则**：
- 遵循三分法或黄金比例构图
- 保留足够的安全边距（不低于画面5%）
- 人物图片裁剪时避免在关节处切断（手腕、脚踝、膝盖）

**尺寸适配裁剪**：
- 不同尺寸需求应从原图裁剪，而非拉伸变形
- 纵向和横向图片分开管理，避免强制裁剪横图变竖图
- 建立标准裁剪比例：1:1（社交头像）、16:9（网页横图）、4:5（Instagram纵向）、9:16（Stories/Snapchat）

### 4.3 图片压缩与优化

**Web性能优化**：
| 格式 | 适用场景 | 质量参数 |
|------|----------|----------|
| WebP | 现代浏览器图片 | 质量75-85%，无损可选 |
| JPEG | 照片类大图 | 质量80-85% |
| PNG | 需要透明背景 | 压缩级别6-9 |
| AVIF | 新一代格式（兼容性允许时） | 质量75-80% |
| SVG | 图标、LOGO、简单图形 | 矢量无限清晰 |

**响应式图片策略**：
- 使用`<picture>`元素或srcset属性提供多分辨率图片
- 图片CDN自动根据设备屏幕尺寸和分辨率适配最佳版本
- 实现懒加载（Lazy Loading）以优化首屏加载性能

### 4.4 图片安全处理

**敏感信息遮盖**：
- 银行卡号、身份证号、密码等敏感信息必须打码处理
- 遮盖应使用品牌元素（而非马赛克）以保持品牌一致性
- 遮盖范围应大于信息实际范围，确保无泄露风险

**品牌LOGO处理**：
- 图片中不可避免出现的第三方品牌LOGO，应尽量避免特写
- 若需保留，应评估是否构成合理使用或品牌合作背书

## 第五章：图片版权与合规

### 5.1 版权基础原则

**版权归属判定**：
- 拍摄照片：拍摄者为版权人（受雇情况下为雇主）
- 设计作品：设计者为版权人
- AI生成图片：2026年法律实践中，AI生成图片的版权归属仍存在争议，建议在法律明确前将其视为"无明确版权保护"处理，优先使用有明确授权的图片

**版权期限**：
- 中国：《著作权法》规定的自然人作品版权保护期为作者终生及死后50年
- 美国：作者死后70年，公司作品为发表后95年或创作后120年
- 欧盟：作者死后70年

### 5.2 授权类型识别

**商业图库常见授权类型**：

**标准授权（Standard License）**：
- 可用于商业和非商业项目
- 不可单独转售图片本身
- 不可用作转售商品的设计元素
- 发行量有限制（通常≤50万份）

**扩展授权（Extended License）**：
- 可用于所有标准授权场景
- 可用于转售商品（如印有图片的T恤）
- 无发行量限制
- 可作为Template（模板）出售

**Editorial Use Only（编辑用途）**：
- 仅限新闻、评论、教育用途
- 不得用于商业广告
- 通常无需Model Release（模特授权）

**免责声明（RF - Royalty Free）**：
- 一次性付费，永久使用（≠免费）
- 不可转售图片本身
- 不可单独注册版权

### 5.3 模特授权（Model Release）

**需要Model Release的场景**：
- 图片用于商业广告
- 图片用于产品包装
- 图片使用方式可能对模特造成困扰或负面影响
- 模特为公众人物

**Model Release内容要素**：
- 模特全名和签名
- 身份证件信息
- 授权使用范围描述
- 授权期限（如有）
- 模特获得的报酬（如有）
- 日期和地点

### 5.4 版权合规检查清单

**每次使用图片前的强制检查**：
- [ ] 图片来源明确，有完整的授权链
- [ ] 使用场景在授权范围内
- [ ] 授权未过期
- [ ] 已获取必要的Model Release（如涉及人物）
- [ ] 图片中无侵犯第三方权益的内容（其他品牌LOGO、艺术品、建筑版权等）
- [ ] 已保存授权凭证（截图、发票、合同）
- [ ] 已记录使用日期和用途（便于到期续约或下架）

## 第六章：图片应用场景规范

### 6.1 网站图片规范

**页面类型图片要求**：

**首页Hero区域**：
- 全宽背景图或焦点图，尺寸≥1920×1080px
- 图片主体应位于画面中央1/3区域（避免文字遮挡）
- 整体氛围与品牌调性一致
- 移动端适配：提供纵向裁剪版本

**关于我们页面**：
- 团队照片：真实工作场景，避免"合影墙"式摆拍
- 公司环境：展示真实办公环境或生产基地
- 历史沿革：使用档案照片或黑白处理增强历史感

**产品详情页**：
- 产品主图：白底标准产品图，角度统一
- 细节图：展示材质、工艺、接口等细节
- 场景图：产品在真实使用场景中的照片
- 对比图：与竞品或前代产品的对比（若有）

### 6.2 社交媒体图片规范

**各平台规格**：

| 平台 | 头像尺寸 | 封面图尺寸 | 信息流图尺寸 |
|------|----------|------------|--------------|
| 微信 | 400×400px | 900×300px | 1200×800px |
| 微博 | 200×200px | 800×200px | 1080×1080px |
| 小红书 | 1:1方图 | 动态横幅 | 1080×1350px |
| 抖音 | 1:1方图 | 动态竖版 | 1080×1920px |

**内容图片风格**：
- 统一的滤镜风格或调色方案
- 水印规范（LOGO位置、尺寸、透明度）
- 品牌色边框或装饰元素
- 图片与文案的比例关系规范

### 6.3 广告投放图片规范

**点击率优化原则**：
- 主体突出：产品或人物占画面≥60%
- 留白区域：画面两侧或顶部留出文字空间
- 高对比度：主体与背景的颜色/明度对比强烈
- 情感触发：人物图片使用正脸和自然表情
- 文字精简：广告文案不超过图片面积的20%

**平台差异化**：
- 朋友圈广告：原生感强，避免明显的"广告感"
- 信息流广告：融入内容流，有"价值感"
- 开屏广告：冲击力强，信息简洁
- 搜索广告（配图）：与搜索意图高度相关

### 6.4 印刷物料图片规范

**色彩管理**：
- RGB转CMYK：所有图片在交付印刷前必须转换为CMYK色彩模式
- 色彩校准：屏幕显示色彩与印刷成品色彩可能存在差异，需预留10-15%的色彩容差
- 出血设置：印刷图片四周需延伸≥3mm出血

**分辨率保障**：
- 大幅印刷（≥1平方米）：150dpi可接受
- 标准印刷（画册、名片）：300dpi必须保证
- 精细印刷（杂志内页）：350dpi最佳

## 第七章：AI生成图片使用规范

### 7.1 AI图片的使用场景

**适用场景**：
- 概念图和情绪板（Mood Board）
- 不存在真实场景的插画类内容
- 高度风格化的艺术表达
- 快速原型和设计验证
- 背景纹理和装饰图形

**不适用场景**：
- 需要展示真实产品外观的产品图
- 新闻报道和真实性要求高的编辑内容
- 需要特定人物形象的商业广告（除非获得真实授权）
- 任何可能误导消费者相信其为真实场景的用途

### 7.2 AI图片的标识规范

根据2026年行业自律准则，AI生成图片必须进行明确标识：

**视觉标识**：
- 在图片右下角添加"AI Generated"水印
- 水印样式：品牌灰色，透明度30%，字号8pt
- 静态图片和动态图片均需标识

**页面级披露**：
- 页面使用AI图片时，应在页面底部或图片附近区域统一声明
- 声明文字："本页面包含AI生成图片，仅供说明和演示用途"

### 7.3 AI图片的质量控制

**必须检查的要素**：
- 人物的手部（AI常见翻车点）和面部是否自然
- 文字是否可读（AI常生成乱码文字）
- 物体比例是否正确
- 是否有明显的AI生成痕迹（过于完美、不自然的光影）
- 是否存在版权元素（模仿知名艺术家风格、著名建筑等）
