# 2048 游戏规则

## 概述

**2048** 是一款流行的滑块益智游戏，由意大利开发者 Gabriele Cirulli 于 2014 年首次创建。玩家通过滑动带数字的方块，使相同数字相互合并，最终目标是得到一个 **2048** 的方块。

---

## 游戏规则

### 基础操作

- 使用键盘 **方向键**（↑ ↓ ← →）或 **WASD** 控制所有方块的滑动方向。
- 所有方块同时向指定方向滑动，直到碰到边界或另一个方块。

### 合并规则

1. **相同合并**：当两个数字相同的方块相撞时，它们会合并成一个方块，数字翻倍。
   - 例：`2` + `2` → `4`，`4` + `4` → `8`，以此类推。
2. **一次一格**：每次滑动中，每个方块只能合并一次。
   - 例：向右滑动 `[2, 2, 2, 2]` → `[4, 4, 0, 0]`，而不是 `[8, 0, 0, 0]`。
3. **合并方向**：方块总是滑到该方向的最远处再合并。

### 新方块生成

- 每次有效滑动后，在空白位置随机生成一个新方块：
  - **90%** 概率生成 `2`
  - **10%** 概率生成 `4`

### 计分系统

- 每次成功合并，加分 = 合并后的方块数值。
  - 例：合并出 `8` → 加 **8 分**
- 历史最高分保存在本地存档中。

### 游戏结束

- 当 **4×4 网格填满** 且 **没有任何相邻方块可以合并** 时，游戏结束。

### 胜利条件

- 成功合并出一个 **2048** 的方块即获胜（可选择继续游戏）。

## 技巧与策略

### 1. 角落法则
将最大的数字保持在角落（如右下角或左下角），避免被挤到中间。

### 2. 不要急于向上滑动
优先水平方向移动，垂直移动时要注意不要把最大数从角落挤走。

### 3. 保持第二大的数字在最大数旁边
这样可以确保在合并时不会浪费空间。

### 4. 避免随机按方向键
每一步都应该有目的，无意义的移动会导致空格迅速填满。

### 5. 提前规划
尽量提前想好几步之后的布局，而不只是看眼前的合并机会。
