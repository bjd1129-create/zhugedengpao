# 俄罗斯方块 - 游戏规则说明

> 小花出品 · 进化基准测试版

## 游戏概述

本游戏基于经典俄罗斯方块规则，使用 Python + pygame 实现，包含以下特色功能：
- 幽灵方块（Ghost Piece）
- 硬降落（Hard Drop）
- 加速升级系统

## 方块种类

共 7 种经典方块（Tetromino），用不同颜色区分：

| 名称 | 形状 | 颜色 |
|------|------|------|
| I | 横条4格 | 青色 |
| O | 2×2方块 | 黄色 |
| T | T形 | 紫色 |
| J | J形 | 蓝色 |
| L | L形 | 橙色 |
| S | S形 | 绿色 |
| Z | Z形 | 红色 |

## 计分规则

消除行数与得分对应关系：

| 消除行数 | 得分 |
|----------|------|
| 1 行 | 100 分 |
| 2 行 | 300 分 |
| 3 行 | 600 分 |
| 4 行（Quad） | 1000 分 |

额外得分：
- 软降（↓ 每下落1格）：+1 分
- 硬降落（空格）：+2 分/格

## 升级系统

- 每消除 10 行后升级
- 每次升级下落速度提升 15%
- 最高速度限制为 0.05 秒/格

## 操作说明

| 按键 | 功能 |
|------|------|
| ← / → | 左右移动 |
| ↑ | 旋转（顺时针） |
| ↓ | 软降（加速下落） |
| 空格 | 硬降（直接落底） |
| P | 暂停/继续 |
| R | 重新开始 |
| Enter | 开始游戏（初始界面）|

## 特色功能

### 幽灵方块
- 显示当前方块落点的虚影位置
- 以半透明边框呈现
- 帮助玩家预判最佳落点

### 墙踢（Wall Kick）
- 旋转时若碰撞墙壁，自动尝试左右偏移 1-2 格
- 避免因贴近墙壁无法旋转的尴尬

### 硬降落
- 按空格键直接将方块落到底部
- 快速消除行的有效手段

## 游戏结束

当新生方块在生成位置就与已有方块碰撞时，游戏结束。

