# 超级玛丽 - 游戏规则说明

> A Python + Pygame 实现的经典横版闯关游戏

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| `←` `→` 或 `A` `D` | 左右移动 |
| `空格` / `W` / `↑` | 跳跃 |
| `↓` / `S` | 下蹲（大玛丽时） |
| `R` | 重置当前关卡 |
| `ESC` | 退出游戏 |

## 🏆 游戏目标

1. **向右奔跑**，穿越整个关卡
2. **躲避或踩死敌人**
3. **收集金币和道具**获得高分
4. **到达终点旗子**过关

## 📦 道具说明

| 道具 | 效果 |
|------|------|
| 🪙 **金币** | +100分，收集计入金币总数 |
| 🍄 **超级蘑菇** | 马里奥变大（从普通变超级） |
| ?方块（带问号） | 顶开可获得金币或蘑菇 |

## 👾 敌人说明

| 敌人 | 行为 | 应对方式 |
|------|------|----------|
| 🟤 **栗宝（Goomba）** | 直线行走，会掉落悬崖 | 踩头击杀 +100分 |
| 🟢 **乌龟（Koopa）** | 正常行走，被踩变龟壳 | 踩头变龟壳，龟壳可踢飞 |
| 🐢 **龟壳** | 静止或移动中 | 踢飞后撞击其他敌人 +400分 |

## 💀 伤害机制

- **普通马里奥**：碰到任何敌人直接死亡，损失1条命
- **超级马里奥**：碰到敌人变小（不会立即死亡），有2秒无敌时间（闪烁效果）

## 🎯 计分规则

| 动作 | 得分 |
|------|------|
| 踩死栗宝 | +100 |
| 踩变龟壳 + 再踢 | +400 |
| 吃超级蘑菇 | +1000 |
| 收集金币 | +200 |
| 过关奖励 | +金币数 × 200 |

## ❤️ 生命系统

- 初始生命：**3条命**
- 生命归零 → 游戏结束（GAME OVER）
- 马里奥掉出屏幕底部 → 立即死亡，损失1条命

## 🚩 过关机制

1. 到达关卡尽头的**绿色旗子**
2. 马里奥自动向下滑落
3. 显示过关画面，进入**下一关**

## 🎨 游戏特色

- **粒子效果**：吃道具、踩敌人时有彩色粒子
- **行走/跳跃动画**：马里奥和敌人有简单帧动画
- **相机平滑跟随**：镜头平滑跟随马里奥移动
- **关卡随机元素**：云朵、敌人位置固定但丰富
- **程序生成音效**：无需外部音频文件

## 🗺️ 关卡结构

```
起点 → 砖块/问号区 → 管道障碍 → 
中段平台 → 更多敌人 → 旗子终点
```

## 🔧 技术实现

- **语言**：Python 3 + Pygame
- **架构**：面向对象（Entity基类 + 具体实体）
- **渲染**：纯代码绘制（无需外部图片资源）
- **帧率**：60 FPS 稳定运行

## 🐛 已知限制

1. 不支持双人游戏
2. 只有一个预设关卡（可循环）
3. 无存档功能
4. 龟壳碰撞检测有待优化
