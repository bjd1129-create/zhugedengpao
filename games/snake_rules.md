# 🐍 贪吃蛇游戏规则说明

## 快速开始

```bash
# 安装 pygame（如果还没有）
pip install pygame

# 运行游戏
python snake_game.py
```

---

## 操作方式

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` | 控制蛇的移动方向 |
| `W A S D` | 控制蛇的移动方向（替代方向键） |
| `ESC` | 暂停 / 继续 |
| `ENTER` 或 `SPACE` | 开始游戏 / 重新开始 |

---

## 游戏目标

控制蛇吃食物，每吃一个食物得分，同时蛇身变长。尽量存活更长时间，挑战最高分！

---

## 得分系统

| 食物类型 | 图标 | 分数 | 说明 |
|----------|------|------|------|
| 普通食物 | 🍎 | 10分 | 最常见的红色食物 |
| 金星食物 | ⭐ | 25分 | 出现概率低，高分奖励 |
| 加速食物 | ⚡ | 5分 | 吃完触发加速模式 |
| 生长食物 | 🌱 | 15分 | 额外增加3节蛇身 |

**注意：** 吃到 ⭐ 金星食物后，会激活 **双倍得分buff**，持续约8秒，期间所有得分翻倍！

---

## 加速模式

- 吃到 **⚡ 蓝** 色食物后激活
- 速度从 10 FPS 提升到 18 FPS（后期最高 25 FPS）
- 持续约 **5 秒**
- 加速期间屏幕左上角显示 ⚡ 图标

---

## 障碍物系统

- 游戏开始后，随机生成灰色方块作为障碍物
- 撞到障碍物 = 游戏结束
- **每吃满5个食物**，自动升级：
  - 蛇的速度略微提升
  - 障碍物数量增加
  - 额外生成一个新食物

---

## 关卡（LV）

每吃 5 个食物升一级。等级越高：
- 障碍物越多（最高15个）
- 速度越快（最高 25 FPS）
- 挑战越大

---

## 最高分

- 游戏会自动保存历史最高分到 `snake_highscore.txt`
- 关闭游戏后再次打开，最高分仍会保留

---

## 死亡条件

满足以下任一条件，游戏结束：
1. 蛇头撞到墙壁
2. 蛇头撞到自己身体
3. 蛇头撞到障碍物

---

## 面向对象设计（代码结构）

```
SnakeGame        # 主游戏类，控制游戏循环 & 状态机
├── Snake         # 蛇：移动、碰撞检测、绘制
├── GameMap       # 地图：管理食物 & 障碍物
│   ├── Food      # 食物：类型、得分、绘制
│   └── Obstacle  # 障碍物：位置、绘制
├── ParticleSystem # 粒子特效：吃食物时的爆炸效果
│   └── Particle   # 单个粒子
├── Dir           # 方向常量
└── GameState     # 状态枚举：MENU / PLAYING / PAUSED / GAMEOVER
```

| 蛇身 | 浅绿 `(80, 200, 80)` |
