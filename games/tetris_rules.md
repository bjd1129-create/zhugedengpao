# 俄罗斯方块 — 游戏规则说明

## 概述

本游戏基于经典俄罗斯方块（Tetris）规则，使用 Python + Pygame 开发。支持 7 种标准方块、旋转、消行得分、等级加速、下一块预览以及最高分记录。

---

## 方块种类（7 种）

| 名称 | 形状 | 颜色 |
|------|------|------|
| I 型 | 横条/竖条 | 青色 |
| O 型 | 正方形 | 黄色 |
| T 型 | T 字形 | 紫色 |
| S 型 | S 形 | 绿色 |
| Z 型 | Z 形 | 红色 |
| J 型 | J 形 | 蓝色 |
| L 型 | L 形 | 橙色 |

---

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ / Z | 旋转方块 |
| ← | 左移一格 |
| → | 右移一格 |
| ↓ | 加速下落一格 |
| 空格 (Space) | 硬降（直接落到底） |
| P | 暂停 / 继续 |
| R | 重新开始 |

---

## 计分规则

消行数量与得分对应关系（乘以当前等级）：

| 消行数 | 得分 |
|--------|------|
| 1 行   | 100 × 等级 |
| 2 行   | 300 × 等级 |
| 3 行   | 700 × 等级 |
| 4 行（四重消） | 1500 × 等级 |

---

## 等级加速

- **初始速度**：下落间隔 800 ms
- **每升 1 级**：下落间隔减少 50 ms
- **最快速度**：100 ms（等级 15+）
- **升级条件**：每消除 10 行升一级

---

## 下一块预览

屏幕右侧面板显示即将出现的下一个方块，帮助玩家提前规划布局。

---

## 最高分记录

- 最高分保存在 `tetris_highscore.json` 文件中
- 游戏结束时会自动更新最高分记录
- 记录持久化，关闭游戏后保留

---

## 游戏结束条件

当新生成的方块在出生点就已经与已有方块产生碰撞时，游戏结束。此时按 **R** 键可重新开始。

---

## 安装与运行

```bash
# 安装 pygame（若未安装）
pip install pygame

# 运行游戏
python tetris_game.py
```
