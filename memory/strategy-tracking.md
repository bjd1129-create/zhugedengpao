## 战略建议格式规则（4月1日起强制执行）

每份战略建议必须包含以下字段，缺一视为"不完整"：

```
## [建议名称]
- **问题**：描述要解决的问题
- **建议内容**：具体建议
- **执行者**：谁来做（Agent名称或负责人）
- **最晚检查**：何时检查执行状态（时间戳）
- **降级方案**：如果无法执行，备选是什么
- **资源依赖**：涉及哪些API/成本/额度
- **状态**：✅已执行 / 🔄执行中 / ❌未执行（+原因）
```
