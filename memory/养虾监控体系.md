# 养虾监控体系 V1

## 老庄提供（2026-03-30）

### 层级一：可视化监控（Grafana）

配置文件 ~/.openclaw/openclaw.json：
```json
{
  "monitoring": {
    "enabled": true,
    "port": 18790,
    "metrics": {
      "token_usage": true,
      "task_duration": true,
      "error_rates": true
    }
  }
}
```

关注指标：
- Token消耗热力图
- 任务耗时分布（P90应下降）
- 错误率曲线（应下降）

### 层级二：进化日志

查看 ~/.openclaw/.learnings/：
- LEARNINGS.md - 学到的新规矩
- ERRORS.md - 犯过的错（不应重复）
- FEEDBACK.md - 纠正记录

监控SOUL.md和USER.md变化。

### 层级三：全链路追踪

使用腾讯云可观测平台或OpenTelemetry追踪：
- Span节点：enter_openclaw_system → invoke_agent → chat → execute tool
- 提示词完整内容
- 工具调用参数

### 层级四：主动考试

1. 每日进化报告Cron
2. openclaw doctor 健康自检
3. 回归测试（以前做错的题重做）

### 避坑指南
- 不要只看Token总量，要结合错误率和成功率
- 定期备份 ~/.openclaw/ 目录
