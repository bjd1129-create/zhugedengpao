# 养虾计划 V2 - 深度进化指南

## 老庄提供方案V2（2026-03-30）

### 第一阶段：开启"自我进化"引擎

1. 安装进化技能：
```
clawhub install self-evolve
# 或
clawhub install self-improving-agent
```

2. 配置进化开关：
```
openclaw config set skills.self-evolve.enable true
openclaw config set skills.self-evolve.dailyReport 08:30
openclaw config set skills.self-evolve.weeklyLearn 08:30
openclaw config set skills.self-evolve.monthlyUpgrade 09:00
```

### 第二阶段：高频率燃烧Token技巧

| 配置项 | 省钱模式 | 进化模式 |
|--------|---------|---------|
| 记忆检索 | QMD(片段) | 禁用QMD(全量) |
| 心跳频率 | 30-120分钟 | 5-10分钟 |
| 上下文 | /compact压缩 | 无限累积 |
| 技能更新 | 手动安装 | self-evolve自动 |
| 模型选择 | 小模型 | 旗舰模型 |

### 第三阶段：五步迭代流程（进化宪法）

触发词：「自我迭代」「复盘」「反思」

执行逻辑：
1. 日志复盘：完整读取工作日志，标记❌和✅
2. 模式识别：同类错误≥2次→固定问题模式
3. 根因定位：能力/态度/流程问题
4. 措施固化：写入MEMORY.md永久生效
5. 强制自检：下次执行前先读【固化规则库】
