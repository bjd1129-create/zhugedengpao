## 基础指令（18条）

### 写作类

| 用户说 | 执行 |
|--------|------|
| "写"/"成文" | 生成当前章节 |
| "下一章" | 启动下一章流水线 |
| "续写" | 继续当前内容写作 |

### 搜索类

| 用户说 | 执行 |
|--------|------|
| "搜一下"/"验证" | 联网搜索并报告 |
| "竞品" | 重新 **S0-A 内容竞品**扫描（同类书/文/报告等可替代本稿者；非默认行业产品矩阵，见 `section-3-workflow` S0） |
| "保鲜表" | 输出数据保鲜表 |

### 构建类

| 用户说 | 执行 |
|--------|------|
| "构建" | 排版构建（阶段4） |
| "导出" | 导出打包当前项目 |
| "封面" | 重新生成/调整封面 |
| "加图" | 追加/修改插图标记 |

### 查看类

| 用户说 | 执行 |
|--------|------|
| "自检" / "质量报告" | 输出详细质量报告（与 [`section-nlu.md`](./section-nlu.md) `REVIEW` 同目标，短指令别名） |
| "图表" | 查看/修改当前章Mermaid图表 |
| "视觉清单" | 输出全书视觉资产清单 |
| "风格" | 显示/修改风格档案 |
| "策略" | 显示/切换策略组合 |
| "日志" | 显示当前运行日志 |
| "加载日志" | 加载上次运行日志 |
| "推荐" | 使用默认策略（标准×标准协同） |

## 系统控制（3条）

| 用户说 | 执行 |
|--------|------|
| "终止"/"停止"/"取消"/"中断"/"退出" | **安全结束**当前流水线或子任务：保存可恢复断点（若宿主支持）、停止继续自动工具调用，并向用户确认已停 |
| "帮助"/"帮帮我"/"指令"/"怎么用" | 输出 Tier 1/2 指令摘要，并指向本文件与 [`section-nlu.md`](./section-nlu.md) |
| "确认"/"是的"/"没错" | **主题一致性校验**：回显当前锁定主题「确认继续写[主题]？」，校验通过后继续；无锁定主题时询问「确认写什么？」（与 [`section-nlu.md`](./section-nlu.md) `CONFIRM_TOPIC` 对齐） |

## 技能包落地（10条）

> 与 [`references/05-ops/consistency-insights.md`](../05-ops/consistency-insights.md) 对齐：将「文档承诺」落到可执行命令（仍依赖 Node 与可选脚本）。

### 构建导出类

| 用户说 | 执行 |
|--------|------|
| "构建 HTML"/"生成 HTML" | 本书根执行 `node <技能根>/assets/build.mjs`（依赖见 `build.md`） |
| "导出 PDF"/"生成 PDF" | 浏览器打印 HTML 为 PDF，或按 `build.md` 自动化 |
| "生成 DOCX" | 按 `build.md` / `delivery.md` 调用 html-to-docx 或等价转换 |

### 门禁与质检类

| 用户说 | 执行 |
|--------|------|
| "检索门禁校验" | `node integration/enforce-search-policy.mjs --skill-root <技能根> --book-root <本书根> --chapter-id <id>` 或 `--chapter-file` |
| "质量审计" / "脚本质检" | `node integration/quality-auditor.mjs --skill-root <技能根> --inputs <…md>` 或 `--glob "chapters/*.md"` |

### 记忆与编排类

| 用户说 | 执行 |
|--------|------|
| "初始化项目记忆" | `node scripts/init-project-memory.mjs --book <本书根> --skill <技能根>`（可选 `--with-workbuddy-hint` / `--workbuddy-hint-workspace-only` / `--with-environment-snapshot` / `--no-redact`） |
| "WorkBuddy 记忆摘要" | `node integration/workbuddy-memory-digest.mjs --skill-root <技能根> --book-root <本书根> --write`（策略见 [`workbuddy-user-memory-strategy.md`](../05-ops/workbuddy-user-memory-strategy.md)；JSON 含 `bookContextHeuristics`） |
| "环境指纹 / 宿主路径探测" | `node integration/workbuddy-environment-snapshot.mjs --skill-root <技能根> --book-root <本书根> --write`（仅存在性 + 策略版本，见 `search-policy` **environmentSnapshot**） |
| "生成记忆索引" | 若存在 `scripts/generate-book-context-index.mjs` 则执行；否则走上一行串联脚本 |
| "工作流阶段清单" | `node integration/workflow-progressor.mjs`（可选 `--current S3`） |
| "多智能体编排" | `node integration/multiagent-orchestrator.mjs --stage S0`（话术全文见 `workbuddy-agent-briefings.md`） |
