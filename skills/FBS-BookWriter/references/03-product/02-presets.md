## §2 三方映射表（config↔CSS↔预设）

> 唯一配置映射源。`books.config.js` 和 `build.mjs` 均参照此表。

### 配色映射

| 预设 | config `color` | config `lightBg` | config `accentBg` | config `accentGold` | CSS `--c-main` | CSS `--c-lightbg` | CSS `--c-accentbg` | CSS `--c-gold` |
|------|---------------|-----------------|------------------|--------------------|-|-|-|-|
| ①琥珀橙 | `#E67E22` | `#FDF2E9` | `#FDEBD0` | `#D4A843` | 同 | 同 | 同 | 同 |
| ②深青蓝 | `#1A5276` | `#EBF5FB` | `#D6EAF8` | `#D4A843` | 同 | 同 | 同 | 同 |
| ③橄榄绿 | `#27AE60` | `#EAFAF1` | `#D5F5E3` | `#D4A843` | 同 | 同 | 同 | 同 |
| ④薰衣草紫 | `#8E44AD` | `#F5EEF8` | `#EBDEF0` | `#D4A843` | 同 | 同 | 同 | 同 |
| ⑤钢青蓝 | `#2C5F7C` | `#F4F7FA` | `#E3EDF4` | `#D4A843` | 同 | 同 | 同 | 同 |
| ⑥自定义 | 用户指定 | 自动推导 | 自动推导 | `#D4A843`（默认） | 同 | 同 | 同 | 同 |

CSS注入方式：
```css
:root {
  --c-main: ${book.color};
  --c-gold: ${book.accentGold || '#D4A843'};
  --c-lightbg: ${book.lightBg};
  --c-accentbg: ${book.accentBg};
}
```

### 视觉预设映射

| 预设 | config `visualPreset` | config `illustrationStyle` | config `mermaidTheme` |
|------|----------------------|--------------------------|---------------------|
| 🅰 实战手册 | `geometric` | `flat` | `base` |
| 🅱 创业指南 | `wave` | `watercolor` | `base` |
| 🅲 行业白皮书 | `grid` | `minimal` | `neutral` |
| 🅳 咨询手册 | `bubble` | `line-art` | `base` |
| 🅴 技能教程 | `ladder` | `flat` | `base` |

> 🅰🅴共用 `flat` 插图风格。`mermaidTheme` 可选：`base`（默认）|`neutral`|`dark`|`forest`。

### 排版密度（density）

| 选项 | config `density` | 行距 | 段间距 | 页边距 | 默认 |
|------|-----------------|------|--------|--------|------|
| 标准 | `standard` | 1.8 | 8px | 25mm/20mm | ✅ |
| 紧凑 | `compact` | 1.5 | 4px | 20mm/15mm | |
| 宽松 | `loose` | 2.2 | 16px | 30mm/25mm | |

> **density** = 排版密度（行距/段距），与 **V1 视觉密度**（内容层：每N字≥1视觉元素）是两个不同概念。V1 定义见 `templates.md` §八。

### 其他config字段

| config 字段 | 默认值 | 说明 |
|------------|--------|------|
| `enableMermaidCDN` | `true` | HTML中引入Mermaid.js CDN；PDF由Puppeteer渲染 |
| `chartDensity` | `auto` | 图表密度：auto按V1规则|high|low|none |
| `coverImage` | `''` | 封面图路径；空→按降级路径（见 `visual.md` §1） |

## §3 微调入口（可选，用户说"自定义"时展示）
