# FBS-BookWriter 产品框架

> 更新日期：2026-03-21 | 状态：已确认

---

## 核心定位

> **"FBS-BookWriter：专业书籍的质量门禁，学术写作的合规护航"**

### 定位说明

| 维度 | 说明 |
|------|------|
| **核心用户** | 需要出版专业书籍、白皮书、手册的知识工作者 |
| **核心场景** | 学术写作、专业出版、内部知识沉淀 |
| **非核心场景** | 营销文案、社交媒体内容、新闻稿件 |
| **核心价值** | 质量门禁（≥7.5分通过）+ 学术合规（强制警告） |

---

## 赛道选择

**蓝海策略：专业书籍 × 白皮书 × 手册**

| 赛道 | 特征 | FBS-BookWriter |
|------|------|----------------|
| **蓝海（我们）** | 专业书籍、白皮书、手册 | ✅ 质量门禁 × 学术合规 × 多Agent协作 |
| **红海（回避）** | 营销文案、SEO文章、社交媒体 | ❌ 同质化严重、Jasper等头部已占据 |

### 蓝海市场机会

- **Jasper等头部调整期**：需要3-6个月重新聚焦专业内容
- **市场需求**：2026年专业书籍AI写作工具窗口期
- **差异化空间**：学术合规警告 + 质量门禁7.5分阈值

## 质量门禁体系

### 评分维度

| 层级 | 名称 | 检查项 | 满分 | 通过阈值 |
|------|------|--------|------|----------|
| **P层** | 段级 | P1问题驱动、P2对话、P3禁止对称排比、P4拒绝注水 | 4×10 | ≥30分 |
| **C层** | 章级 | C1承认局限、C2结尾行动、C3打破均匀、C4数据具体化 | 4×10 | ≥30分 |
| **B层** | 篇级 | B1标题去公式化、B2-A段落节奏、B2-B标点多样性、B2-C结构雷同、B3全局节奏 | 10分 | ≥7.5分 |

### B层篇级规则（Critic-L3执行）

| 规则 | 要求 |
|------|------|
| **B1 标题去公式化** | 公式化标题 = 0；只能是提问式/结论式/悬念式 |
| **B2-A 段落节奏** | 变异系数 ≥ 0.3；短段落15-25%、长段落10-20% |
| **B2-B 标点多样性** | 每千字 ≥ 5种标点类型 |
| **B2-C 结构雷同** | 连续3章节相同开头 → 警告；连续2段落相同结尾 → 警告 |
| **B3 全局节奏** | 章节长度变异系数 > 0.4 → 警告；风格偏离度 > 0.3 → 警告 |

## 学术合规体系

### 学术用途识别关键词库

| 类别 | 关键词 |
|------|--------|
| **论文类** | 论文、毕业论文、学位论文、毕业设计 |
| **期刊类** | 期刊、投稿、发表、SCI、SSCI、CSSCI、核心期刊 |
| **学术类** | 学术、研究、科研、课题、基金项目、学术诚信 |
| **署名类** | 第一作者、通讯作者、作者、署名 |
| **数据类** | 实验数据、调查数据、原始数据、伪造数据、数据造假 |

### 合规检查流程（G层）

1. 检测学术关键词触发
2. 弹出强制警告（不可忽略）
3. 用户必须显式确认
4. 记录确认状态

## 交付体系

### 四种交付选项

| 选项 | 格式 | 产出物 | 适用场景 |
|------|------|--------|----------|
| **A轻量** | MD单文件 | `书名.md` | 快速分享、笔记存档 |
| **B标准** | MD + HTML | `书名.zip` | 最常用交付格式 |
| **C完整** | MD + HTML + 资产 | `书名_完整.zip` | 深度编辑/二次创作 |
| **D网站** | 完整网站包 | `书名_site.zip` | 直接托管/分享 |

### 核心策略

> **MD+HTML为主，PDF/DOCX用户自助转换**

| 格式 | 定位 | 可靠性 | 用户操作 |
|------|------|--------|----------|
| MD | 主力文字输出 | 100% | 直接下载 |
| HTML | 主力可视化输出 | 100% | 直接下载 |
| PDF | 用户自助 | 依赖浏览器 | 浏览器打印为PDF |
| DOCX | 用户自助 | 依赖在线工具 | 在线工具转换 |
