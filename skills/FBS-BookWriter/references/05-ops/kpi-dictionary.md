# KPI 指标字典（定义 + 公式）

## 1. 说明

本字典统一产品化与运营指标口径，避免跨团队定义不一致。

统计原则：

- 默认按自然周/自然月统计。
- 明确分母、去重规则、统计窗口。
- 指标异常波动需记录原因与修复动作。

## 2. Route-E（企业高附加值）指标

| 指标 | 定义 | 公式 |
|---|---|---|
| 企业线索数 | 周期内新增有效企业机会数 | `有效线索总数` |
| 立项率 | 从线索到正式立项比例 | `立项项目数 / 有效线索数` |
| 里程碑按时率 | 已承诺里程碑按时交付比例 | `按时里程碑数 / 承诺里程碑总数` |
| 终稿一次通过率 | 首次终审即通过比例 | `首次终审通过项目数 / 进入终审项目数` |
| 企业复购率 | 周期内发生二次及以上采购比例 | `复购客户数 / 活跃企业客户数` |

## 3. Route-G（增长规模）指标

| 指标 | 定义 | 公式 |
|---|---|---|
| 新增试用数 | 周期内新开通试用账号数 | `新增试用账号总数` |
| 7日激活率 | 新增用户 7 日内完成首个有效流程比例 | `7日内激活用户数 / 新增用户数` |
| 首单转化率 | 试用后完成首单购买比例 | `首单购买用户数 / 试用用户数` |
| 月留存率 | 上月活跃用户本月仍活跃比例 | `连续两月活跃用户数 / 上月活跃用户数` |
| ARPU | 人均收入 | `周期收入 / 付费用户数` |

## 4. 质量与交付公共指标

| 指标 | 定义 | 公式 |
|---|---|---|
| 章节质量通过率 | 章节质量门禁一次通过比例 | `一次通过章节数 / 评估章节总数` |
| 阻断率 | 触发门禁阻断比例 | `阻断次数 / 流程触发总次数` |
| 平均修订轮次 | 单章节平均修订次数 | `修订总轮次 / 章节总数` |
| 检索覆盖率 | 章节中带引用事实段占比 | `带引用事实段数 / 事实段总数` |

## 5. 采集建议

- 流程质量指标：来自质量账本与审计脚本输出。
- 运营转化指标：来自宿主侧埋点与订单系统。
- 周报中至少包含：本周值、上周值、环比、异常说明。
