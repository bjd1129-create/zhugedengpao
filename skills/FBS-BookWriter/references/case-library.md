# FBS-BookWriter 师生案例集

> 建立竞品无法复制的护城河——真实写作素材库

## 1. 案例集定位

**核心价值：** 为AI写作提供经过质量验证的真实素材参考，让每一类写作都有可对标的优秀案例。

**战略意义：**
- 竞品无法复制的护城河（需长期积累）
- 用户成功的证明材料
- 质量评分的参照基准
- 变现路径的展示窗口

---

## 2. 案例分类体系

### 2.1 四大类型

| 类型 | 定义 | 评分基准 | 积累目标(第一年) |
|------|------|----------|------------------|
| 学术论文 | 毕业论文/期刊论文/学术报告 | ≥8.0分 | ≥30个 |
| 白皮书 | 行业白皮书/技术白皮书/政策白皮书 | ≥8.5分 | ≥25个 |
| 手册/指南 | 操作手册/用户指南/培训手册 | ≥7.5分 | ≥25个 |
| 行业研究报告 | 市场报告/投资报告/调研报告 | ≥8.0分 | ≥20个 |

### 2.2 成功/失败双轨案例

每个类型下分设两个子库：

```
├── 成功案例库（评分≥8.5）
│   ├── 学术论文/优秀/
│   ├── 白皮书/优秀/
│   ├── 手册指南/优秀/
│   └── 研究报告/优秀/
│
└── 失败案例库（评分<7.0）
    ├── 学术论文/待改进/
    ├── 白皮书/待改进/
    ├── 手册指南/待改进/
    └── 研究报告/待改进/
```

## 4. 多维度标签系统

### 4.1 标签维度

```
┌─────────────────────────────────────────────────────────┐
│                    案例标签体系                           │
├─────────────────────────────────────────────────────────┤
│  [行业] 金融 | 教育 | 医疗 | 科技 | 制造 | 消费 | 法律 | 咨询 │
│  [类型] 专业书籍 | 白皮书 | 手册/指南 | 研究报告 | 工具书      │
│  [读者] B端 | C端 | G端 | 学生 | 教师 | 企业管理者          │
│  [评分] 优秀(≥8.5) | 良好(7.5-8.4) | 合格(7.0-7.4) | 待改进(<7.0) │
│  [篇幅] 微薄(万字内) | 标准(3-5万) | 专著(10万+)           │
│  [来源] 用户授权 | 公开案例 | 定制开发                      │
│  [语言] 中文 | 英文 | 中英双语                            │
└─────────────────────────────────────────────────────────┘
```

### 4.2 标签组合查询示例

- "金融 + 白皮书 + B端 + 优秀" → 金融行业白皮书优秀案例
- "教育 + 手册指南 + C端 + 良好" → 教育类用户手册良好案例
- "科技 + 研究报告 + B端 + 优秀 + 10万+" → 科技行业长篇报告优秀案例

## 5. 案例积累策略

### 5.1 入库机制

| 来源 | 触发条件 | 入库流程 | 奖励 |
|------|----------|----------|------|
| 用户授权 | 用户同意+评分≥7.5 | 自动推送+审核确认 | Pro延期/优惠券 |
| AI自建示范 | 新功能上线 | 专家审核+标注"AI示范" | 无 |
| 公开案例 | 学术期刊/获奖作品 | 标注来源+获授权确认 | 无 |

### 5.2 用户激励政策

**入库条件：**
- 写作成果评分≥7.5分
- 用户签署授权协议（电子确认）
- 完成脱敏处理

**激励措施：**
| 评分区间 | 奖励内容 |
|----------|----------|
| 9.0+ | 6个月Pro会员 + 平台首页推荐 |
| 8.5-8.9 | 3个月Pro会员 |
| 8.0-8.4 | 1个月Pro会员 |
| 7.5-7.9 | 平台积分200点 |

### 5.3 质量门槛

- **最低入库标准：** 评分≥7.5分
- **优秀案例标准：** 评分≥8.5分 + 读者反馈≥4星
- **下架标准：** 被投诉侵权 / 存在虚假信息

## 6. 读者反馈数据

### 6.1 反馈收集机制

```
用户完成写作 → 推送反馈邀请 → 5维度评分 → 入库归档
```

### 6.2 5维度评分体系

| 维度 | 评估问题 | 权重 |
|------|----------|------|
| 实用性 | "这本书对您的实际工作有帮助吗？" | 30% |
| 专业性 | "内容专业度是否达到预期？" | 25% |
| 可读性 | "阅读体验是否流畅清晰？" | 20% |
| 时效性 | "数据和案例是否足够新？" | 15% |
| 性价比 | "相较于其他工具，是否值得？" | 10% |

## 7. 变现路径记录

### 7.1 变现类型矩阵

| 变现路径 | 适用类型 | 案例特征 | 记录字段 |
|----------|----------|----------|----------|
| 自用归档 | 全类型 | 个人/企业内部 | 用途/范围 |
| 正式出版 | 论文/专著 | 评分≥8.5 | 出版社/时间/版税 |
| 课程转化 | 手册/指南 | 实操性强 | 课程名称/平台 |
| 企业培训 | 手册/指南 | B端导向 | 企业名称/套数 |
| 内容授权 | 研究报告 | 数据独特 | 授权对象/金额 |
| 会员订阅 | 全类型 | 持续更新 | 订阅人数/收入 |

### 7.2 变现数据脱敏标准

- 收入金额 → 范围区间（如"5-10万"）
- 企业名称 → 行业+规模（如"某金融机构"）
- 个人姓名 → 角色（如"某高校教授"）

## 8. 案例展示与应用

### 8.1 案例展示页

每个案例展示：
- 书名/标题（脱敏后）
- 评分雷达图
- 核心亮点（3条）
- 读者反馈摘要
- 适用的写作场景
- 相关推荐案例

### 8.2 AI写作参考机制

当用户进入特定写作场景时，AI自动推荐相关案例：

```
用户选择："我要写一本金融行业白皮书"
       ↓
AI检索：金融+白皮书+B端+优秀
       ↓
推荐TOP3案例 + 亮点摘要 + 阅读链接
       ↓
用户可选择"参考此案例风格"或"跳过"
```

## 9. 隐私保护与合规

### 9.1 脱敏标准

| 信息类型 | 处理方式 |
|----------|----------|
| 人名 | 删除或改为"某先生/女士" |
| 企业名 | 改为"某金融机构/某科技公司"等泛化描述 |
| 联系方式 | 完全删除 |
| 特有数据 | 改为同类典型数据 |
| 商业敏感信息 | 完全删除 |

### 9.2 授权协议要点

```
[授权确认函]
我确认授权FBS-BookWriter使用我的写作成果作为案例参考，
并同意进行必要的脱敏处理。
授权范围：□公开案例展示 □AI训练参考 □商业推广
授权期限：永久
撤回权利：可随时联系客服撤回
```
