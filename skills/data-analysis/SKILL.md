## 常见陷阱

### 统计陷阱

| 陷阱 | 说明 | 规避方法 |
|------|------|----------|
| **辛普森悖论** | 分组和整体趋势相反 | 分层分析 |
| **幸存者偏差** | 只看成功案例 | 考虑完整样本 |
| **选择性偏差** | 样本不具代表性 | 随机抽样 |
| **相关非因果** | 相关不等于因果 | 验证因果机制 |

### 可视化陷阱

| 陷阱 | 说明 | 规避方法 |
|------|------|----------|
| **纵轴截断** | 夸大变化幅度 | 从0开始 |
| **3D图表** | 视觉扭曲 | 使用2D图表 |
| **饼图过多** | 难以比较 | 使用柱状图 |
| **颜色过多** | 视觉混乱 | 限制颜色数量 |

### 业务陷阱

| 陷阱 | 说明 | 规避方法 |
|------|------|----------|
| **虚荣指标** | 看起来好但无意义 | 关注可行动指标 |
| **指标孤岛** | 单一指标决策 | 多维度评估 |
| **数据滞后** | 决策时机错过 | 建立实时监控 |

## 分析流程

```
1. 明确问题
   └── 业务目标、分析目的

2. 数据收集
   └── 数据来源、质量检查

3. 数据清洗
   └── 缺失值、异常值处理

4. 探索分析
   └── 描述统计、可视化

5. 深度分析
   └── 统计建模、假设检验

6. 结论建议
   └── 洞察提炼、行动建议

7. 报告输出
   └── 决策简报、图表展示
```
