## 技能描述

高级长期记忆系统，基于ChromaDB向量数据库实现语义检索，支持记忆存储、相关记忆召回、记忆更新等高级记忆管理功能。

| 组件 | 说明 |
|------|------|

| **语义检索** | 相关记忆召回 |
| **记忆更新** | 动态更新记忆内容 |

## 使用方式

### 记忆检索（memory_search）

```typescript
memory_search({
  query: "搜索关键词",
  maxResults: 10,
  minScore: 0.5
})
```

**参数说明：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| query | string | 必需 | 搜索查询 |
| maxResults | int | 10 | 最大返回数 |
| minScore | float | 0.0 | 最小相关度（0-1） |

---

### 记忆读取（memory_get）

```typescript
memory_get({
  path: "memory/YYYY-MM-DD.md",
  from: 10,
  lines: 50
})
```

**参数说明：**

| 参数 | 类型 | 说明 |
|------|------|------|
| path | string | 记忆文件路径 |
| from | int | 起始行号 |
| lines | int | 读取行数 |

### 相关性评分

| 分数 | 相关度 | 说明 |
|------|--------|------|
| 0.9+ | 极高 | 直接相关 |
| 0.7-0.9 | 高 | 高度相关 |
| 0.5-0.7 | 中 | 部分相关 |
| <0.5 | 低 | 弱相关 |
