# 机票查询技能

> **技能获取时间:** 2026-03-18 19:47
> **技能来源:** 老庄指示
> **适用场景:** 机票价格查询、航班信息、价格日历

---

## 技能描述

查询机票价格和航班信息，支持单程/往返查询、多日期价格日历、自动识别中文城市名、国内外航线。

---

## 安装方式

```bash
# 安装机票查询技能
clawhub install flight-price-cn

# 查看完整使用方法
clawhub docs flight-price-cn
```

## 使用方式

### 自然语言触发

直接用自然语言查询：

```
"北京飞上海下周五机票多少钱"
"香港到东京往返，找最便宜的日期"
"下个月深圳飞成都的价格日历"
"广州到杭州明天的航班"
```

---

## 支持功能

| 功能 | 说明 | 示例 |
|------|------|------|
| **单程查询** | 查询单程机票 | "北京飞上海明天" |
| **往返查询** | 查询往返机票 | "北京到上海往返" |
| **价格日历** | 多日期价格对比 | "下个月北京飞上海价格日历" |
| **最便宜日期** | 自动找最低价 | "找北京飞上海最便宜的日期" |
| **城市识别** | 自动转机场代码 | "北京" → PEK/PKX |
| **国内+国际** | 支持国内外航线 | "香港到东京" |

```
用户：北京飞上海下周五机票多少钱

AI：正在查询...
【北京 → 上海 机票价格】2026-03-21

| CA123 | 07:00-09:20 | ¥580 | 国航 |
| MU456 | 08:30-10:50 | ¥520 | 东航 |

| HU123 | 14:00-16:20 | ¥450 | 海航 ✨最低 |

建议：选择HU123，价格最优

```

```
| 03-27 ↔ 04-03 | ¥2,520 | +¥340 |

建议：周三出发价格最优
```

## 注意事项

| 注意项 | 说明 |
|--------|------|
| **实时性** | 价格实时变化 |
| **提前预订** | 建议提前2-4周预订 |
| **淡旺季** | 节假日价格较高 |
| **税费** | 价格通常不含税费 |
| **签证** | 国际航班需确认签证 |
