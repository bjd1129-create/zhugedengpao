# Hacker News技能

> **技能获取时间:** 2026-03-18 19:39
> **技能来源:** 老庄指示
> **适用场景:** 获取Hacker News热帖、技术资讯追踪

---

## 技能描述

抓取Hacker News首页热帖，支持按主题过滤，适合追踪技术领域最新动态。

---

## 安装方式

```bash
# 搜索技能
clawhub search hacker-news

# 安装技能
clawhub install hacker-news
```

---

## 使用方式

### 触发词

| 触发词 | 功能 |
|--------|------|
| `hn` | 获取HN热帖 |
| `pull HN` | 获取HN热帖 |
| `hn 10` | 获取前10条热帖 |
| `hn AI` | 获取AI相关热帖 |
| `hn tech` | 获取技术相关热帖 |
| `hn health` | 获取健康相关热帖 |

---

## 支持主题过滤

| 主题 | 触发词 | 说明 |
|------|--------|------|
| **技术** | `hn tech` | 编程、架构、工具 |
| **AI** | `hn AI` | 人工智能、机器学习 |
| **健康** | `hn health` | 健康、医疗、生物 |
