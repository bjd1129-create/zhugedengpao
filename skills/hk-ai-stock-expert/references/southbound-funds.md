# 南向资金分析框架

## 什么是南向资金

南向资金是指内地投资者通过港股通投资香港股市的资金，包括：
- **港股通 (沪)**: 上海证券交易所 → 港交所
- **港股通 (深)**: 深圳证券交易所 → 港交所

## 数据获取

### 官方渠道
- **港交所官网**: www.hkex.com.hk → 市场数据 → 港股通
- **上交所官网**: www.sse.com.cn → 披露 → 港股通持股
- **深交所官网**: www.szse.cn → 市场数据 → 港股通

### 第三方工具
- **东方财富 Choice**: 资金流向 → 港股通
- **Wind**: 港股通持股数据库
- **同花顺 iFinD**: 南向资金监控

## 核心指标

### 1. 净流入/流出
```
日净流入 = 当日买入金额 - 当日卖出金额
周净流入 = 近 5 个交易日净流入之和
月净流入 = 近 20 个交易日净流入之和
```

**信号解读：**
| 指标 | 数值 | 信号 |
|------|------|------|
| 日净流入 | > 10 亿港元 | 强烈看好 |
| 日净流入 | 5-10 亿港元 | 看好 |
| 日净流入 | -5~5 亿港元 | 中性 |
| 日净流入 | < -5 亿港元 | 看空 |
| 日净流入 | < -10 亿港元 | 强烈看空 |

### 2. 持仓变化
```
持仓变动 = 当前持仓 - 上期持仓
持仓变动率 = 持仓变动 / 上期持仓 × 100%
```

**信号解读：**
| 指标 | 数值 | 信号 |
|------|------|------|
| 持仓变动率 | > 20% | 大幅增持 |
| 持仓变动率 | 10-20% | 增持 |
| 持仓变动率 | -10~10% | 持仓稳定 |
| 持仓变动率 | -20~-10% | 减持 |
| 持仓变动率 | < -20% | 大幅减持 |

### 3. 持仓集中度
```
前十大持仓占比 = 前十大持仓市值 / 总持仓市值 × 100%
```

**信号解读：**
| 指标 | 数值 | 信号 |
|------|------|------|
| 前十大占比 | > 60% | 高度集中 (龙头效应) |
| 前十大占比 | 40-60% | 适度集中 |
| 前十大占比 | < 40% | 分散布局 |

### 4. 资金流向持续性
```
连续流入天数 = 净流入 > 0 的连续交易日数
连续流出天数 = 净流入 < 0 的连续交易日数
```

**信号解读：**
| 指标 | 数值 | 信号 |
|------|------|------|
| 连续流入 | ≥ 5 天 | 强势看好 |
| 连续流入 | 3-4 天 | 温和看好 |
| 连续流出 | ≥ 5 天 | 持续看空 |
| 连续流出 | 3-4 天 | 温和看空 |

## 分析框架

### 第一步：整体趋势判断
1. 统计近 20 日净流入总额
2. 计算净流入均线 (5 日/10 日/20 日)
3. 判断趋势方向 (向上/向下/震荡)

### 第二步：板块分布分析
1. 按行业分类统计净流入
2. 识别资金偏好板块
3. 对比板块估值水平

### 第三步：个股持仓变化
1. 列出持仓变化前 10 大个股
2. 分析增持/减持原因
3. 关联股价表现

### 第四步：资金流向与股价相关性
1. 计算相关系数
2. 识别领先/滞后关系
3. 判断资金驱动还是基本面驱动

## 实战应用

### 场景 1: 发现资金流入标的
```
筛选条件：
1. 近 5 日南向资金净流入 > 5 亿港元
2. 持仓变动率 > 10%
3. 当前股价处于 52 周低位 50% 分位以下
4. 基本面未恶化

→ 可能是左侧布局机会
```

### 场景 2: 识别资金流出风险
```
预警条件：
1. 近 5 日南向资金净流出 > 5 亿港元
2. 持仓变动率 < -10%
3. 连续流出天数 ≥ 5 天
4. 股价跌破关键支撑位

→ 注意减仓或止损
```

### 场景 3: 跟踪机构动向
```
重点关注：
1. 南向资金前十大持仓股
2. 持仓变化幅度 > 20% 的个股
3. 新进前十大持仓的个股
4. 退出前十大持仓的个股
```

## 注意事项

1. **数据滞后**: 南向资金持仓数据 T+1 披露，注意时效性
2. **被动配置**: 部分资金为指数被动配置，不代表主动看好
3. **短期波动**: 单日数据噪音大，建议看周度/月度趋势
4. **结合基本面**: 资金流向仅作参考，需结合基本面判断
5. **警惕反转**: 大幅流入后可能获利回吐，注意节奏

## 常用查询命令

```sql
-- 查询南向资金日度净流入
SELECT date, net_inflow, buy_amount, sell_amount
FROM southbound_funds
WHERE date >= '2025-01-01'
ORDER BY date DESC;

-- 查询个股持仓变化
SELECT stock_code, stock_name, holding_shares, holding_value, change_pct
FROM southbound_holdings
WHERE date = '2025-03-26'
ORDER BY change_pct DESC
LIMIT 20;

-- 查询板块分布
SELECT sector, sum(holding_value) as total_value, count(*) as stock_count
FROM southbound_holdings
WHERE date = '2025-03-26'
GROUP BY sector
ORDER BY total_value DESC;
```
