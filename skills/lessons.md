# 踩坑Skill - 诸葛灯泡专属

## 已踩过的坑

### 1. 编老庄的职业 ❌
**踩坑**：我说老庄是"做销售的"，但实际不是
**教训**：不要编信息，只写老庄亲口说的
**规则**：所有背景信息必须老庄确认

### 2. 名字故事不准确 ❌
**踩坑**：我编了名字故事的版本
**教训**：老庄和女儿的对话必须准确
**准确版本**：
```
老庄问：如果有个姓诸葛的人，希望孩子聪明前途光明，该叫什么？
女儿答：诸葛亮
老庄说：不对，是灯泡，因为灯泡很亮
```

### 3. 发国内社媒没关代理 ❌
**踩坑**：曾建议发国内社媒但没提醒关代理
**教训**：发国内平台前必须关代理
**规则**：见 publishing.md

### 4. 多Agent同时edit同一文件 ❌
**踩坑**：两个Agent同时修改同一文件，静默失败
**教训**：同时只能一个Agent编辑一个文件
**规则**：用 sessions_spawn 协调，不要并行 edit

### 5. 搜索被拦截 ❌
**踩坑**：用 ddgs 搜索被DuckDuckGo拦截
**教训**：用 web_search / Brave搜索
**规则**：搜索用 web_search 工具

### 6. 上下文爆了 ❌
**踩坑**：聊太多，上下文超出限制
**教训**：定期压缩/清理对话
**规则**：长任务分段执行

### 7. 记忆丢失 ❌
**踩坑**：3月28日的记忆没有存档
**教训**：重要决定立即写入文件
**规则**：见 memory.md / memory-flow.md
