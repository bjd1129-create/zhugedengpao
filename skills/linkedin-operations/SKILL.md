## 技能描述

系统化LinkedIn运营能力，包括内容体系搭建、自动生成帖子、收件箱管理、内容策略规划等，助力建立思想领导力。


## 使用场景

### 场景一：思想领导力定位

```
用户：帮我定位LinkedIn思想领导力

AI：正在分析...
🎯 【LinkedIn思想领导力定位】

## 专业领域
- AI Agent技术研究
- 企业AI转型实践

## 目标受众
- 企业技术决策者
- AI技术爱好者
- 创业者/产品经理

## 内容支柱
1. AI技术深度解读
2. 行业趋势分析
3. 实战经验分享
4. 工具评测推荐

## 发布节奏
- 每周3-5篇帖子
- 周二/周四/周六发布
- 最佳时间：上午9-10点
```

### 场景二：生成LinkedIn帖子

```bash
# 生成帖子
linkedin-generate --topic "AI Agent趋势" --style "thought-leadership"
```

**输出示例：**
```markdown
AI Agent正在改变我们的工作方式，但很多人忽略了关键的一点：

不是AI替代人类，而是会用AI的人替代不会用的人。

我在过去6个月深度研究了100+AI Agent案例，发现3个关键趋势：

1️⃣ 单一任务Agent → 多Agent协作
2️⃣ 被动执行 → 主动决策
3️⃣ 工具使用者 → 工具创造者

最成功的企业不是简单部署AI，而是重新设计工作流程。

你观察到哪些AI Agent应用趋势？评论区分享你的见解。

#AIAgent #FutureOfWork #ArtificialIntelligence
```

### 场景三：收件箱管理

```bash
# 查看消息分类
linkedin-inbox --categorize

# 生成回复建议
linkedin-inbox --reply --message-id xxx
```

```markdown

[洞察：你的独特观点]

[证据：数据、案例或经验]

[CTA：引导讨论]

#Hashtags
```

### 行业洞察帖

```markdown

[3个关键点]
1️⃣ [点1]
2️⃣ [点2]
3️⃣ [点3]

[你的看法]

```

