## SEO写作核心要点

### 1. 标题优化

| 要点 | 标准 |
|------|------|
| **关键词位置** | 核心关键词尽量靠前 |
| **长度限制** | 60字符内（中文30字） |
| **吸引点击** | 包含数字、问号、情感词 |
| **避免堆砌** | 关键词出现1次即可 |

**示例：**
```
❌ 错误：AI人工智能技术发展人工智能应用
✅ 正确：2026年AI技术10大突破：发展趋势全解析
```

---

### 2. Meta描述

| 要点 | 标准 |
|------|------|
| **长度** | 155字符内（中文75字） |
| **关键词** | 必须包含核心关键词 |
| **吸引点击** | 引导用户点击的行动词 |
| **内容概括** | 准确描述文章内容 |

**示例：**
```
❌ 错误：这是一篇关于AI的文章
✅ 正确：深入解析2026年AI技术最新突破，涵盖大模型、Agent、多模态等10大趋势，助你把握AI发展脉络。
```

---

### 3. 标题层级结构

```
H1 - 文章标题（唯一）
├── H2 - 第一章标题
│   ├── H3 - 1.1小节
│   └── H3 - 1.2小节
├── H2 - 第二章标题
│   ├── H3 - 2.1小节
│   └── H3 - 2.2小节
└── H2 - 总结
```

**规则：**
- H1只出现1次
- H2/H3层级清晰
- 关键词分散在各级标题

---

### 4. 关键词密度

| 指标 | 标准 |
|------|------|
| **密度范围** | 2-3% |
| **分布方式** | 标题、首段、正文、结尾 |
| **避免堆砌** | 自然融入，不影响阅读 |
| **同义词** | 使用相关词、长尾词 |

**计算公式：**
```
关键词密度 = (关键词出现次数 / 文章总字数) × 100%

例：1000字文章，关键词出现20-30次
```

---

### 5. 链接优化

#### 内链

| 要点 | 说明 |
|------|------|
| **相关性** | 链接到相关文章 |
| **锚文本** | 使用关键词作为锚文本 |
| **数量** | 每篇3-5个内链 |

#### 外链

| 要点 | 说明 |
|------|------|
| **权威性** | 链接到权威网站 |
| **相关性** | 链接与主题相关 |
| **新窗口** | 外链用target="_blank" |

---

### 6. 图片优化

| 要点 | 标准 |
|------|------|
| **alt标签** | 必须包含关键词 |
| **文件名** | 使用关键词命名 |
| **压缩** | 图片小于100KB |
| **格式** | WebP优先，兼容JPG/PNG |

**示例：**
```html
❌ 错误：<img src="IMG_001.jpg">
✅ 正确：<img src="ai-technology-trends-2026.webp" alt="2026年AI技术发展趋势图">
```

## 关键词研究方法

### 1. 搜索研究

```python
# 使用web_search分析关键词
web_search(query="AI技术 + 相关搜索", count=10)

# 分析要点：
# 1. 搜索结果标题中的关键词
# 2. 底部"相关搜索"词
# 3. 高搜索量低竞争词
```
