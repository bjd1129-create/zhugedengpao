## 工具箱简介

腾讯技术公益数字工具箱由腾讯公益慈善基金会发起，以腾讯技术公益平台（techforgood.qq.com）作为统一申请入口，整合近30款通用型数字化工具，助力公益机构低成本快速接入数字化服务。

**统一入口**：https://techforgood.qq.com/tools

## 申领资格要求

### 机构资质

申请机构需满足以下全部条件：

1. **机构类型**：在中国内地依法注册或登记的社会团体、社会服务机构、基金会等社会组织，或普通高等学校、科研院所等事业单位
2. **项目要求**：已有发起相关公益项目，且项目发展较为成熟，运营流程成型、规范
3. **需求匹配**：现阶段有数字化需求且尚未解决，技术需求能用工具箱相关产品解决
4. **人员配置**：认同平台服务方式，且有数字化工作的对接专员

**不支持申请的主体**：个人、个体工商户、股份有限公司、有限责任公司和其他营利法人

### 额度规则（数字化服务类）

- 每个公益组织拥有 **5000元/年** 的采购额度
- 额度可用于申领第三方数字化服务产品
- 福利申领后，需满**一个自然年**方可再次申领同类福利
- 腾讯自有产品权益不占用此额度，单独申领

**小型机构（10人以下）**：
- 优先申领：腾讯会议 + 腾讯文档 + 企业微信（免费，解决基础协同）
- 按需补充：腾讯问卷（调研）、ima（AI辅助写作）
- 额度建议：益通云自助建站（2998元）用于搭建官网

**中型机构（10-50人）**：
- 基础套件同上
- 额度建议：益帮帮项目管理（4500元）或灵析联系人管理（4999元）

**大型机构（50人以上）**：
- 全套腾讯自有产品
- 额度建议：根据最迫切需求选择（建站/项目管理/筹款管理选一）
- 可考虑：腾讯元器搭建机构专属AI助手

### 按业务需求推荐

| 核心需求 | 推荐组合 |
|---------|---------|
| 刚起步，什么都缺 | 企业微信 + 腾讯文档 + 腾讯会议 + 益通云建站 |
| 项目多，管理乱 | 益帮帮项目管理 + 腾讯文档 |
| 想做好捐赠人维护 | 灵析联系人管理 + 腾讯问卷 |
| 志愿者队伍大 | 芳邻时间银行 + 企业微信 |
| 想用AI提效 | ima + 腾讯元器 + 腾讯文档 |
| 经常办线下/线上活动 | 腾讯会议 + 腾讯问卷 + 金数据 |
| 服务听障人群 | 音书手语翻译 + 腾讯会议 |
