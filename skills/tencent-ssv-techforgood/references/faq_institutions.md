# 公益机构用户高频问题与解答

> 本文档收录公益机构常见的咨询问题和标准化解答，供公益虾技能在回答时参考。

## Q1：如何成立一家公益基金会？

**回答要点**：

成立基金会需要经过以下主要步骤：

1. **确定基金会类型**：
   - 全国性基金会：在民政部登记，原始基金不低于800万元（人民币）
   - 地方性基金会：在省级民政部门登记，原始基金不低于400万元

2. **寻找业务主管单位**：
   - 需要有相关领域的政府部门或授权机构同意作为业务主管单位
   - 部分地区已试点直接登记（无需业务主管单位）

3. **准备申请材料**：
   - 设立申请书
   - 章程草案
   - 验资证明和住所证明
   - 理事名单、身份证明及简历
   - 业务主管单位同意设立的文件

4. **向登记管理机关提交申请**：
   - 民政部门审查后作出准予登记或不予登记的决定

5. **同步申请慈善组织认定**：
   - 新设立时可同步申请认定为慈善组织

**特别提醒**：各地具体流程可能略有差异，建议先向拟注册地的民政部门咨询了解当地具体要求。

## Q2：已有社会组织如何申请慈善组织认定？

**回答要点**：

依据《慈善组织认定办法》（2024年9月5日施行）：

1. **申请条件**：
   - 已经设立的基金会、社会团体、社会服务机构
   - 以开展慈善活动为宗旨
   - 不以营利为目的
   - 有符合规定的章程
   - 有必要的财产
   - 有符合条件的组织机构和负责人
   - 近两年未受到与慈善活动相关的行政处罚

2. **申请流程**：
   - 向登记管理机关提交认定申请及相关材料
   - 登记管理机关自受理之日起20日内作出决定
   - 准予认定的发给认定证书，在统一信息平台公告

3. **认定后的权利**：
   - 可以依法开展慈善活动
   - 两年后可申请公开募捐资格
   - 享受税收优惠

## Q3：怎样取得公开募捐资格？

**回答要点**：

1. **基本条件**：
   - 已认定为慈善组织
   - 依法登记满两年
   - 近两年内未因违反慈善法受到行政处罚

2. **申请材料**：
   - 公开募捐资格申请书
   - 注册会计师出具的财务审计报告（近两年）
   - 募捐方案
   - 其他登记管理机关要求的材料

3. **审批流程**：
   - 向登记管理机关提交申请
   - 登记管理机关20日内作出决定
   - 准予的发放公开募捐资格证书

4. **注意事项**：
   - 取得资格后，每次公开募捐前需将募捐方案备案
   - 不具备资格的组织不得自行开展公开募捐，但可与有资格的慈善组织合作

## Q4：如何进行公开募捐方案备案？

**回答要点**：

依据《慈善组织公开募捐方案备案指引（试行）》（2025年11月印发）：

1. **备案内容**：
   - 募捐目的和用途
   - 募捐起止时间
   - 募捐方式
   - 受益人范围
   - 资金使用计划
   - 募捐成本预算

2. **备案方式**：
   - 通过"慈善中国"信息平台在线提交
   - 开展公开募捐活动前完成备案

3. **备案时限**：
   - 登记管理机关在收到备案材料后及时确认
   - 备案完成后方可开展募捐活动

4. **变更和补正**：
   - 募捐方案需要变更的，应重新备案
   - 信息有误需要补正的，可在线提交补正申请

## Q5：机构开展网络募捐需要注意什么？

**回答要点**：

1. **平台选择**：
   - 必须在民政部指定的互联网公开募捐信息平台发布募捐信息
   - 可以同时在自有网站、微信公众号等自有渠道发布
   - 自有渠道发布的，应注明在指定平台的链接

2. **信息要求**：
   - 募捐信息应真实、完整
   - 包含慈善组织名称、公开募捐资格证书、募捐方案备案等信息
   - 不得进行虚假宣传

3. **资金管理**：
   - 网络募捐资金应进入慈善组织账户
   - 不得截留或挪用
   - 定期向社会公开募捐情况和资金使用情况

4. **合作募捐**：
   - 与其他组织合作开展网络募捐时，应明确双方权责
   - 募捐信息中应标明合作各方的基本信息

## Q6：慈善组织每年需要做哪些合规工作？

**回答要点**：

1. **年度报告**：
   - 每年向登记管理机关提交年度工作报告和财务会计报告
   - 年报应在规定时间内（通常为次年3-6月）完成

2. **信息公开**：
   - 在"慈善中国"平台公开年度工作报告
   - 公开募捐的慈善组织定期公开募捐情况和项目实施情况
   - 重大关联交易提前公开

3. **财务审计**：
   - 年度财务报告应经注册会计师审计
   - 具有公开募捐资格的慈善组织应编制审计报告

4. **支出比例**：
   - 慈善活动支出和管理费用需符合法定比例（具体比例依据慈善法及配套规定）

5. **换届和变更**：
   - 理事会换届需按章程规定进行
   - 法定代表人、住所等变更需向登记管理机关申请

## Q7：如何申请公益性捐赠税前扣除资格？

**回答要点**：

1. **申请条件**：
   - 已认定为慈善组织
   - 符合财政、税务部门规定的其他条件
   - 申请前连续3年的年度检查合格

2. **申请流程**：
   - 由财政、税务、民政部门联合确认
   - 通常每年集中确认一批

3. **注意事项**：
   - 资格有效期通常为3年
   - 到期需重新确认
   - 获得资格后需向捐赠人开具公益事业捐赠统一票据

4. **捐赠人权益**：
   - 企业捐赠：年度利润总额12%以内部分准予扣除，超出部分可结转3年
   - 个人捐赠：不超过应纳税所得额30%的部分准予扣除

## Q8：机构如何规范管理志愿者？

**回答要点**：

1. **招募管理**：
   - 发布招募信息时明确志愿服务项目内容、时间、条件等
   - 根据项目需要对志愿者进行筛选和培训

2. **权益保障**：
   - 为志愿者购买人身意外伤害保险
   - 提供必要的工作条件和安全保障
   - 不得要求提供超出约定范围的服务

3. **记录管理**：
   - 如实记录志愿服务信息（服务时间、内容等）
   - 通过全国志愿服务信息系统进行记录
   - 为志愿者出具服务记录证明

4. **激励机制**：
   - 配合开展星级志愿者认定
   - 提供志愿服务证书
   - 建立志愿者反馈和评估机制

## Q9：突发灾害时机构如何参与应急慈善？

**回答要点**：

1. **法律依据**：
   - 《慈善法》第十一章"应急慈善"（2023年修订新增）

2. **参与方式**：
   - 配合应急管理部门和民政部门的统一协调
   - 开展应急募捐、物资捐赠和运送
   - 提供灾后救助和恢复重建服务

3. **注意事项**：
   - 应急慈善活动应服从当地统一调配
   - 基层政府和基层组织应提供便利
   - 捐赠人可指定受益人，慈善组织应尊重捐赠人意愿
   - 确保物资质量安全
   - 及时公开捐赠和使用信息

## Q10：慈善组织如何防范法律风险？

**回答要点**：

1. **合规管理**：
   - 建立完善的内部治理结构
   - 制定并严格执行财务管理制度
   - 定期开展合规自查

2. **常见风险**：
   - 信息公开不及时不充分
   - 资金使用不规范
   - 关联交易未披露
   - 年报迟报或漏报
   - 募捐方案未备案

3. **法律后果**：
   - 行政处罚（警告、限期改正、罚款）
   - 吊销登记证书
   - 取消公开募捐资格
   - 情节严重的可能追究刑事责任

4. **建议措施**：
   - 聘请专业法律顾问
   - 定期进行内部审计
   - 及时关注法规更新
   - 参加行业培训和交流
