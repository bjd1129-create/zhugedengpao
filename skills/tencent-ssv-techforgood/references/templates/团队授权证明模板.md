# 腾讯技术公益 — 团队授权证明模板

> **📅 收录日期：2026年3月**
> **原始文件**：`腾讯技术公益-团队授权证明模板.doc`
> **用途**：公益机构在腾讯技术公益平台注册时，需由法人签署团队授权证明，指定团队管理员。本文档为官方模板，机构按需填写后盖章提交。

---

## 模板正文

---

<div align="center">

### 团队授权证明

</div>

**腾讯技术公益平台：**

本机构指派 **______**（身份证号：**____________**，职务：**______**，联系方式：**____________**）（下称"被指派人"）作为本机构团队管理员，被指派人具体权限为：

> 代表本机构上传机构信息、资料，注册平台机构账号，且机构法人授权"被指派人"上传法人信息、资料；签署并同意平台一切协议及相关规则；设置平台内相关人员（如管理员、机构成员等）的角色、权限、信息；申请使用本平台相关服务，如工具箱、创投计划、志愿者平台等；通过本平台发布并组织机构服务活动；其他与本平台服务相关的一切事务。

本机构已充分理解并同意平台服务协议及平台相关的所有规则。被指派人在平台所签署的一切文件和处理与之有关的一切事务，本机构均予以承认，且被指派人在平台的一切行为均代表本机构，与本机构的行为具有同等法律效力（无论被指派人是否为本机构的正式员工）。本机构保证被指派人严格遵守各项法律法规，在平台的一切行为不会侵犯任何第三方的合法权益，包括但不限于不违反被指派人与任何第三方的（劳动）合同义务等内容。本机构同意承担因被指派人的行为造成的全部法律责任与后果。若机构团队管理人员发生变更，本机构同意并认可该被指派人指派的人员自动变更为本机构团队管理员，享有机构团队管理员在平台的一切权利和义务。

本机构充分知悉并同意上述事项。

## 填写说明

| 填写项 | 说明 |
|-------|------|
| 被指派人姓名 | 填写将担任团队管理员的人员全名 |
| 身份证号 | 被指派人的身份证号码 |
| 职务 | 被指派人在机构中的职务 |
| 联系方式 | 被指派人的手机号码 |
| 机构盖章 | 加盖机构公章 |
| 被指派人签字 | 被指派人亲笔签名 |
| 法人签字 | 机构法定代表人亲笔签名 |
| 日期 | 填写签署日期 |

## 使用场景

- **平台注册**：公益机构首次在腾讯技术公益平台注册机构账号时提交
- **团队管理员变更**：机构团队管理员发生变更时提交新的授权证明
- **权限管理**：被指派人将代表机构行使平台上的所有管理权限

## 注意事项

1. 本模板需要**机构公章 + 被指派人签字 + 法人签字**三方确认
2. 被指派人不要求必须是机构正式员工，但机构需承担其行为的全部法律责任
3. 管理员变更后，原管理员指派的人员自动成为新管理员
4. 建议机构保留一份签署后的扫描件备档
