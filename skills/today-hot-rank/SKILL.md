# 今日热榜技能

> **技能获取时间:** 2026-03-18 19:39
> **技能来源:** 老庄指示
> **适用场景:** 获取各平台热搜榜单、热点追踪

---

## 技能描述

从 tophub.today 抓取各平台热搜榜单，包括微博、知乎、百度、抖音、头条、B站等主流平台。

---

## 安装方式

```bash
# 搜索技能
clawhub search 今日热榜

# 安装技能
clawhub install 今日热榜
```

---

## 支持平台

| 平台 | 类型 | 特点 |
|------|------|------|
| **微博** | 社交媒体 | 娱乐八卦、社会热点 |
| **知乎** | 知识社区 | 深度讨论、专业话题 |
| **百度** | 搜索引擎 | 热门搜索词 |
| **抖音** | 短视频 | 流行趋势、网红热点 |
| **头条** | 新闻资讯 | 时事新闻、社会事件 |
| **B站** | 视频社区 | 游戏、动漫、科技 |

## 使用场景

### 场景一：热点追踪

```
用户：今天有什么热搜？

AI：为您查询今日热榜...
【微博热搜 Top 5】
1. XXX事件
2. XXX明星
3. ...

【知乎热榜 Top 5】
1. XXX问题
2. ...
```

### 场景二：选题策划

```
用户：帮我看今天的热点，给我选题建议

AI：根据今日热搜分析...
【热点1】XXX事件 - 建议角度：技术解读
【热点2】XXX话题 - 建议角度：行业分析
...
```

### 场景三：舆情监控

```
用户：今天有没有AI相关的热搜？

AI：正在筛选AI相关热搜...
【知乎】AI行业最新动态
【B站】AI工具测评
...
```
