# 火车票查询技能

> **技能获取时间:** 2026-03-18 19:46
> **技能来源:** 老庄指示
> **适用场景:** 火车票余票查询、车次时刻表、票价查询

---

## 技能描述

通过12306 API查询火车票信息，支持自然语言查询，包括余票查询、车次时刻表、站点信息、票价查询等。

---

## 安装方式

```bash
# 安装12306技能
clawhub install 12306

# 查看完整API使用方法
clawhub docs 12306
```

---

## 使用方式

### 自然语言触发

直接用自然语言查询即可：

```
"北京到上海明天的高铁"
"查一下后天从广州到深圳的火车票"
"国庆节北京到成都还有票吗"
"下周三上海到杭州的动车"
```

---

## 查询参数

| 参数 | 说明 | 示例 |
|------|------|------|
| **出发地** | 出发城市/站点 | 北京、上海虹桥 |
| **目的地** | 到达城市/站点 | 广州、深圳北 |
| **日期** | 出发日期 | 明天、后天、2026-03-20 |
| **车型** | 列车类型 | 高铁、动车、普速 |

---

## 注意事项

| 注意项 | 说明 |
|--------|------|
| **实时性** | 余票数据实时变化 |
| **节假日** | 节假日票源紧张 |
| **预售期** | 通常预售15天 |
| **候补功能** | 无票可选择候补 |
