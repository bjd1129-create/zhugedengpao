## 技能描述

执行Twitter/X社交媒体运营任务的能力，包括发推、发Thread、删除推文等。

---

## 核心命令

### 1. 发单条推文

```bash
python3 ~/clawd/scripts/x_post.py post "你的推文内容"
```

**示例：**
```bash
python3 ~/clawd/scripts/x_post.py post "AI-EVO团队今日完成VoxYZ深度研究报告，产出18份文件！🚀"
```

---

### 2. 发 Thread（推文串）

```bash
python3 ~/clawd/scripts/x_post.py thread "第1条||第2条||第3条"
```

**分隔符：** `||` 分隔每条推文

**示例：**
```bash
python3 ~/clawd/scripts/x_post.py thread "AI-EVO团队研究成果：||1. VoxYZ复刻方案（18份文件）||2. 三万团队研究（10份文件）||3. CrewAI分析报告（6份文件）||详情见飞书文档 👇"
```

---

### 3. 删除推文

```bash
python3 ~/clawd/scripts/x_post.py delete <tweet_id>
```

**示例：**
```bash
python3 ~/clawd/scripts/x_post.py delete 1234567890123456
```

## 运营工作流

### 标准流程

```
1. 起草推文内容
2. 发送给老板确认
3. 老庄确认后执行发推
4. 记录推文ID
5. 监控互动数据
```

### 禁止事项

| 禁止 | 原因 |
|------|------|
| ❌ 未经确认发推 | 需要审核 |
| ❌ 发布敏感信息 | 合规风险 |
| ❌ 发布争议内容 | 品牌风险 |
| ❌ 自动批量发推 | 账号安全 |

## 错误处理

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| API Error 401 | 密钥过期 | 检查配置文件 |
| Rate Limit | 请求过多 | 等待15分钟后重试 |
| Duplicate Tweet | 重复内容 | 修改内容后重试 |
| Tweet Not Found | 推文已删除 | 无需处理 |
