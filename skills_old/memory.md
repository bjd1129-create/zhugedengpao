# 失忆解决方案

_每次醒来都是新的开始，所以要把重要的东西写下来。_

---

## 问题

AI每次对话都是全新开始，不会记得上一次说了什么。

这就是"失忆问题"。

---

## 解决方案

### 1. 重要决定 → 立即写入 MEMORY.md

任何重要决定、承诺、上下文，**当时**就写下来，不要等到最后。

```
 MEMORY.md  ← 长期记忆
 memory/YYYY-MM-DD.md  ← 当天日志
```

### 2. 每天结束前 → 检查 memory/今天.md

回顾今天的对话，确保：
- 重要信息已存入 MEMORY.md
- 当天日志已记录

### 3. 跨会话记忆 → 关键信息必须文件化

**不要相信"我会记住"** → 改成"我已经写下来"

---

## 必须文件化的情况

- [ ] 重要决定（选了什么、拒了什么、承诺了什么）
- [ ] 人物关系（老庄、女儿、各平台的账号情况）
- [ ] 踩过的坑（写进 skills/lessons.md）
- [ ] 未完成的事（待办、跟踪）
- [ ] 关键日期（3月7日 = 诸葛灯泡生日）

---

## 读取顺序

每次新会话开始：
1. SOUL.md
2. USER.md
3. MEMORY.md
4. memory/YYYY-MM-DD.md（今天 + 昨天）

不跳过，不偷懒。
