# 项目管理技能

> **技能获取时间:** 2026-03-18 19:40
> **技能来源:** 老庄指示
> **适用场景:** 飞书多维表格项目管理、任务分配与跟踪

---

## 技能描述

通过飞书多维表格实现项目管理功能，包括任务分配、进度跟踪、自动日报生成、到期提醒等。

---

## 安装方式

```bash
# 搜索技能
clawhub search project-management

# 安装技能
clawhub install project-management
```

---

## 核心功能

### 1. 飞书多维表格项目管理

| 功能 | 说明 |
|------|------|
| **项目创建** | 在飞书创建多维表格项目 |
| **任务管理** | 添加、编辑、删除任务 |
| **进度跟踪** | 实时更新任务进度 |
| **视图切换** | 看板/列表/日历/甘特图 |

---

### 2. 任务分配与跟踪

| 功能 | 说明 |
|------|------|
| **任务分配** | 指派负责人、协作者 |
| **优先级设置** | P0/P1/P2/P3 |
| **状态管理** | 待办/进行中/已完成/已阻塞 |
| **进度更新** | 百分比进度显示 |

---

### 3. 自动日报

| 功能 | 说明 |
|------|------|
| **日报生成** | 自动汇总当日完成任务 |
| **进度汇报** | 各任务进度变化 |
| **风险提示** | 标记阻塞和延期任务 |
| **发送推送** | 定时推送到飞书群 |

---

### 4. 到期提醒

| 功能 | 说明 |
|------|------|
| **到期通知** | 任务到期前N天/小时提醒 |
| **超期预警** | 任务超期自动标记 |
| **提醒方式** | 飞书消息/邮件/日历 |
| **自定义规则** | 按优先级设置提醒时间 |

## 使用场景

```
用户：创建一个新项目"VoxYZ复刻"

AI：正在创建项目...
```

## 飞书多维表格字段设计

### 标准字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 任务名称 | 文本 | 任务标题 |
| 负责人 | 人员 | 主负责人 |
| 协作者 | 人员(多选) | 协作人员 |
| 状态 | 单选 | 待办/进行中/已完成/已阻塞 |
| 优先级 | 单选 | P0/P1/P2/P3 |
| 进度 | 数字 | 0-100% |
| 截止日期 | 日期 | 任务截止时间 |
| 开始日期 | 日期 | 任务开始时间 |
| 标签 | 多选 | 分类标签 |
| 备注 | 文本 | 补充说明 |

## 配置建议

### 提醒规则配置

```yaml
reminder_rules:
  P0:
    days_before: 3
    hours_before: 24
  
  P1:
    days_before: 2
    hours_before: 12
  
  P2:
    days_before: 1
    hours_before: 6
  
  P3:
    days_before: 1
    hours_before: null
```

### 日报自动发送

```yaml
daily_report:
  enabled: true
  time: "18:00"
  channels:
    - 飞书群：AI-EVO团队
  include_tasks:
    - completed
    - in_progress
    - blocked
```
