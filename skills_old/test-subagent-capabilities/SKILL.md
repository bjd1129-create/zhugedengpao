# 测试子Agent实际技能与能力

> **技能获取时间:** 2026-03-18 18:41
> **技能来源:** 老庄指示
> **适用场景:** 验证子Agent是否具备某项工具或能力

---

## 技能描述

当需要验证子Agent是否具备某项能力（如飞书文档访问、文件写入、API调用等）时，使用此技能进行测试验证。

---

## 核心方法

### 1. 设计测试任务

测试任务必须：
- 明确测试目标（验证什么能力）
- 包含具体操作步骤
- 有明确的成功/失败判断标准

**测试任务模板：**

```markdown
## 测试任务

请执行以下操作，验证你是否有[能力名称]权限：

1. [具体操作1]
2. [具体操作2]
3. [具体操作3]

请汇报：
- [能力1]是否成功
- [能力2]是否成功
- [能力3]是否成功
```

---

### 2. 派发测试任务

使用 `sessions_spawn` 创建测试子Agent：

```typescript
sessions_spawn({
  label: "test-[capability-name]",
  runtime: "subagent",
  task: "测试任务内容...",
  timeoutSeconds: 120  // 测试任务建议短超时
})
```

---

### 3. 验证结果

根据子Agent汇报判断能力是否具备：

| 结果 | 判断 |
|------|------|
| ✅ 成功 | 能力已具备，可直接使用 |
| ❌ 失败 | 能力受限，需要替代方案 |

## 实战案例

### 案例：测试子Agent飞书文档能力

**测试任务：**

```markdown
## 测试任务

请执行以下操作，验证你是否有飞书文档访问权限：

1. 尝试读取飞书文档: https://feishu.cn/docx/xxx
2. 尝试在该文档末尾追加一行文字："子Agent测试成功 - [当前时间]"
3. 输出一个txt文件到: /Users/bjd/intelligence/research/ai-evo/test/subagent-test.txt

请汇报：
- 飞书文档读取是否成功
- 飞书文档写入是否成功
- txt文件是否成功创建
```

**测试结果：**

| 任务 | 状态 |
|------|------|
| 飞书文档读取 | ✅ 成功 |
| 飞书文档写入 | ✅ 成功 |
| txt文件创建 | ✅ 成功 |

**结论：** 子Agent具备完整飞书文档读写能力，可直接操作。

## 常用能力测试清单

| 能力 | 测试工具 | 测试方法 |
|------|----------|----------|
| 文件写入 | `write` | 创建一个txt文件 |

| 飞书文档写入 | `feishu_doc` | 追加内容到飞书文档 |
| 网页访问 | `web_fetch` | 访问一个公开URL |
| 浏览器操作 | `browser` | 打开一个网页并截图 |
| 命令执行 | `exec` | 执行一个简单命令（如 `echo`） |

## 注意事项

1. **测试任务要具体** - 明确操作步骤和预期结果
2. **使用短超时** - 测试任务建议120秒内完成
3. **记录测试结果** - 将能力验证结果更新到MEMORY.md
4. **定期复测** - 环境变化后需要重新验证能力
